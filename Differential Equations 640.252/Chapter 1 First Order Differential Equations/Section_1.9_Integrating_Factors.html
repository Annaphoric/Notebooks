Section 1.9: Integrating Factors

    Integration Factor
  • let non-homogenous DE {dy/dt + g(t) y = b(t)};
  • let μ(t) = integration factor, a function that satisfies d(μ(t) y(t))/dt = μ(t) dy/dt + μ(t) g(t) y(t)

  • The Product Rule of Derivatives says that:
      • d(μ(t) y(t))/dt = μ(t) dy/dt + dμ/dt y(t)
  • Therefore:
      • μ(t) dy/dt + μ(t) g(t) y(t) = μ(t) dy/dt + dμ/dt y(t)
  • Cancelling out μ(t) dy/dt and dividing by y(t), the equation remains as:
      • dμ/dt = μ(t) g(t)
  • Solving for a solution to the DE via separation & integration gives:
      • μ(t) = e∫(g(t))dt

  • Note: dμ/dt == g(t) is always true
    Example 1:
      Determine the general solution to DE {dy/dt + 2y / t = t - 1}.
    1. let g(t) = 2/t, b(t) = t - 1; Solving for integration factor μ(t):
        2. μ(t) = e∫(2/t)dt // The constant of integration is absorbed by the integration factor
        μ(t) = e2 ln(t)
        μ(t) = eln(t2)
        μ(t) = t2

    2. Multiplying μ(t) to both sides of the DE and simplifying:
        3. (t2) dy/dt + (t2) 2y / t = (t2) (t - 1)
        t2 dy/dt + 2t y = t3 - t2

    3. Applying the product rule and solving for the general solution via integration:
        4. d(t2 y)/dt = t2 dy/dt + 2t y // This is true because dμ/dt = 2t

        ∫(d(t2 y)/dt)dt = ∫(t3 - t2)dt
        t2 y = t4/4 - t3/3 + C
        y(t) = t2/4 - t/3 + C/t2

        Note: {y(t) = t2/4 - t/3} is one solution to the DE (where C = 0)
        Note: The associated homogenous DE is dy/dt = -2y/t

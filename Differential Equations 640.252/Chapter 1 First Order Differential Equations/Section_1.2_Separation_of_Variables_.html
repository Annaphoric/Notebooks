Section 1.2: Separation of Variables

    Differential Equations & their Solutions
  • The standard form of a first-order differential equation is:
      • dy/dt = f(t, y), where y = dependent variable & t = independent variable (time)

  • A solution to a DE is a function of y that satisfies the equation for all t as:
      • dy/dt = f(t, y(t))

  • In the case for DE dy/dt = y, the function y(t) = c et, where c = any constant, easily satisfies the equation; therefore, it is considered a solution to the DE
      • If there are multiple solutions to a DE, it is possible that they all algebraically look different from each other

  • One way to solve for an initial value problem (IVP) is by integrating the DE to solve for a general solution, substituting values, and solving for the integration constant

    Separable Equations
  • A DE is considered separable only if it can be expressed as a product of a function of t & a function of y, having the form:
      • dy/dt = g(t) h(y)

  • A DE is considered both separable and autonomous only if it can be expressed as a function of y, but not a function of t, having the form:
      • dy/dt = h(y)

  • There may be instances where the DE cannot be integrated, especially if it is non-autonomous
  • The DE will first have to be separated via the separation of variables, that is, the dependent variable with its derivative should lie on one side of the equation, whilst the independent variable with its derivative should lie on the other
  • It will now have the form:
      • h(y) dy = g(t) dt
  • Although this is an informal way to algebraically simplify the equation, integration is now possible on both sides: in terms of y on the left side & in terms of t on the right

  • If integration leads to a general solution that contains a function that is not a solution, a conditional equillibrium solution will have to be specified to ensure that the general solution remains a general solution to the DE, such as:
      • dy/dt = y2 --> y(t) = -1 / (t + C) && y(t) = 0

  • If integration leads to an equation that cannot be solved for y, an implicit solution will have to be solved for y, rather than an explicit one

    Example 1:
      Solve for the initial value problem given: DE dy/dt = 12t3 - 2 sin(t), IC y(0) = 3;
    1. Integrating the DE:
        2. ∫(dy/dt)dt = ∫(12t3 - 2 sin(t))dt
        y = 3t4 + 2 cos(t) + C // This is the general solution

    2. Using the IC to solve for integration constant C:
        3. y(0) = 3(0)4 + 2 cos(0) + C == 3
        2 + C = 3
        C = 1

    3. Writing out the solution to the IVP:
        4. y(t) = 3t4 + 2 cos(t) + 1

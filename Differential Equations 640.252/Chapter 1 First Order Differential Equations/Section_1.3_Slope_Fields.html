Section 1.3: Slope Fields

    Slope Fields
  • Given dy/dt = f(t, y), f(t, y) is the slope of the tangent line passing through point (t1, y1), where y1 = y(t1)

  • The diagram showing slope lines to the direction of increasing t is known as a slope field
      • Generally, the more slope lines drawn, the more defined the diagram
  • Many possible solutions can be distinguished by simply examining the graph by its trends
  • There are also other analytical methods to determining solutions which will be explained at a later section
  • The positions of the slope lines differ according to the integration constant after solving for the general solution to a DE

  • The slope field of an autonomous DE, like dy/dt = f(y), is the same when shifted horizontally
  • Solutions to this type of equation can be determined by using values for y in between equillibrium solutions, where dy/dt = 0

  • If analytic methods, such as integration, cannot be used to solve a DE, qualitative analysis is the closest method that can be performed to understand the long-term behaviour of the function rather than an explicit solution
  • This can be done by graphing the DE (forming a slope field) using different values of y
      • Equillibrium solutions of an autonomous D can be visualised by determining values of y where solutions to f(y) = t does not exist

Section 1.1: Modeling via Differential Equations

    Mathematical Models
  • The main purpose of mathematical models is to convert a real-world problem into a math problem that can be solved by hand or, even better, computer
  • Models give a representation of an object by making use of its characteristics; it is then our decision to choose the model that is most convenient to solve
  • Predictions can be made using a model only if the system data agrees with it; otherwise, the model must be improved or another model must be chosen
  • A precise model formulates accurate quantitative predictions, whilst a simplified model formulates reliable qualitative predictions

    Building a Model
      There are 3 steps to building a mathematical model:
    1. State the assumptions on which the model will be based, including descriptions of the relationships between the observed quantities
    2. Describe the quantities (variables & parameters) that will be used
    3. Derive equations that relate the quantities based on the formulated assumptions

  • Assumptions describe the most important aspects of a system

  • Quantities fall into 3 basic categories:
      • Independent variables
    • A variable that is not affected by other variables, like time
      Dependent variables
    • A variable that acts as a function for the independent variable(s), like position relative to time
      Parameters
    • A value or constant that is not affected by the independent variable(s) but can be adjusted by the experimenter, like mass of an object

  • Most convenient models appear in the form of a differential equation, which mathematically describes how some dependent variable changes relative to an independent variable, like velocity (rate of positional change)

  • When describing proportionality such as "A is proportional to B", a proportionality constant k is commonly used to form the equation:
      • A = k B

  • An example of a model would be describing the air resistance of a falling object:
      • Assumption: Air resistance increases as downwards velocity increases
      Quantities: Dependent variable -> force of air resistance (A); Independent variable -> velocity (v); Parameter -> friction coefficient (k);
      Equation: A(v) = k v || A(v) = k v2

    Population Growth
  • Population growth is an popular example that has been modelled very extensively

  • Assumption: The rate of population growth is proportional to the population size (disregarding space or resource limitations; population growth remains small & environment remains large)
  • Quantities: Dependent variable -> population size (P); Independent variable -> time (t); Parameter -> grwoth-rate coefficient (k)
  • Equation: dP/dt = k P || P'(t) = k P

  • This is a form of qualitative analysis because the rate of population growth increases as population size increases; the equation does not give a discrete value, rather a trend

  • The units of every quantity is vital to keep track of variables
      • In this case, population size is number of people, time may vary, and the growth-rate coefficient is the inverse of time (1/time)

  • The equation is a differential equation (DE) because is shows how the dependent variable P changes over the independent variable t
  • It is considered a first-order DE because the highest derivative within the equation is the first derivative
  • Since there are no other independent variables mentioned within the equation, the DE is considered ordinary; otherwise it would be partial

  • A DE represented as its differential form is known as Leibniz notation

  • Assumption: The rate of population growth is zero when population size is zero (population doesn't exist)
  • Quantities: same
  • Equation: dP/dt = 0 if P = 0

  • This equation can be proved by setting the population size equal to zero in the population growth equation:
      • dP/dt = k (0)
      dP/dt = 0
  • Since the DE gives a real value given the initial condition (IC) P(t) = 0, P(t) = 0 is said to be a solution to the DE
      • Specifically, P(t) = 0 is an equillibrium solution because the dP/dt = 0 is a constant, discrete value

  • Assumption: The rate of population growth is never zero if the population size is not zero
  • Quantities: same
  • Equation: dP/dt = k P(t0) != 0, where t0 = any point in time
  • An initial value problem (IVP) contains both the differential equation and an initial condition
      • An IVP can be solved for a solution which satisfies both equations
    • To solve for a solution, the derivative of P(t), (P') must match the DE
    • For example, P(t) = c ekt if dP/dt = k c e == k P(t), where c = any constant (which can also be expressed as c = P0)
    • When an expression is given as the solution to a DE, it is referred to as a general solution

  • In reality, this model is not accurate in real-world applications because there is usually a limit to how much a population can grow due to lack of space or resources
    The Logistic Growth Model
  • To account for carrying capacity due to lack of space & resources, the following assumptions must stay true:
      • The rate of population growth is proportional to the population if the population size is small
      The rate of population growth is negative if the population size reaches a number large enough to be hindered by lack of space & resources

  • In this case, an additional quantity will have to be used. This quantity is a secondary parameter N which is known as the carrying capacity
  • The following must be true:
      • P(t) increases if P(t) < N && P(t) decreases if P(t) > N)

  • The model will also have to be modified to account for the carrying capacity
  • Since the rate of population growth in the original model increases without any limit, a factor must be multiplied alongside it that remains as close to one as possible if the population size reaches its carrying capacity; this is written as:
      • dP/dt = k P (1 - P/N)
  • This is known as the logistic population model
      • This model is non-linear because the dependent variable P does not have the linear exponent of 1

  • To solve for an equillibrium solution for this new DE, the DE must be equal to a discrete value
      • In this case, it can be equal to zero
      The only solutions that satisfy dP/dt == 0 is P(t) = 0 || P(t) = N
      This can be proven by substituting the values of P(t) into the DE

  • When testing out multiple values for P(t) into the DE, it can be observed that:
      • dP/dt increases for P in (0, N)int && dP/dt decreases for P in (-∞, 0)int || P in (N, ∞)int
    Predator-Prey Systems
  • There may be situations where there are multiple dependent variables
  • The predator-prey system is a model where there are 2 dependent variables; the predator variable is denoted as F (for foxes) and the prey variable is denoted as R (for rabbits)

  • The following assumptions must be true:
      • Assumption: Rabbits reproduce indefinitely at a rate proportional to their population size only if there no foxes present
      Assumption: The foxes eat the rabbits, thereby decreasing rabbit population, proportional to the rate at which foxes interact with the rabbits
      Assumption: The fox population decreases if there are no rabbits present
      Assumption: The population growth rate of foxes is proportional to the number of rabbits eaten
  • Parameters: α = growth-rate coefficient of rabbits; β = proportionality constant for rabbit-fox interaction; γ = proportionality death-rate; δ = proportionality constant for fox benefit after eating rabbit
  • Equations: dR/dt = α R - β R F; dF/dt = δ R F - γ F

  • Since this model consists of 2 equations (one for each dependent variable), the pair of equations is referred to as a first-order system of ODEs
      • This system is said to be coupled because the differentials of the dependent variables contain the variables themselves

  • Since there are 2 dependent variables being measured, it is impractical to find a solution to the DEs unless a qualitative or numerical method is utilised
      • A solution can either be found analtyically, qualitatively or numerically
    • Analytic approaches involve the use of explicit formulas to give explicit solutions
    • A qualitative approach uses geometry to give an overview of the model's behaviour, rather than precise values
    • The numerical approach uses an approximation technique to give a reliable solution, usually done by a computer

Section 1.6: Equillibria & the Phase Line

    Phase Lines
  • An autonomous DE is a DE that is a function of solely y
  • It has the form:
      • dy/dt = f(y)

  • Autonomous DEs have the same value for any t-value given a y-value

  • A phase line is a vertical line consisting of all values of y for one t-value (t0)
      • To simplify the phase line, it should consist of the sign between each equillibrium solution/point (where dy/dt = 0) for increasing t
      They can help determine the long-term behaviour of the DE

  • According to the Existence & Uniqueness Theorem, a solution to an equillibrium point can never reach it, rather simply approach it; it is asymptotic at the horizontal where the equillibrium solution lies

  • An equillibrium point is a sink if any solution given an IC is asymptotic to the equillibrium solution as t increases; in other words, the graph flows inwards towards the equillibrium solution
  • An equillibrium point is a source if any solution given an IC is asymptotic to the equillibrium solution as t decreases; in other words, the graph flows outwards away from the equillibrium solution
  • An equillibrium point is a node if any solution given an IC is asymptotic inwards or outwards to the equillibrium solution as t increases or decreases; in other words, the graph flows inwards or outwards above or below the equillibrium solution but never both
    Linearization
  • Linearization Theorem
      • let DE dy/dt = f(y), y0 = equillibrium point;
      The derivative of the function determines whether an equillibrium point is a sink, source, or neither; in other words, the derivative linearly approximates f(y) at y0
      if f'(y0) < 0: y0 is a sink
      else if f'(y0) > 0: y0 is a source
      else if f'(y0) == 0: the nature of y0 cannot be determined

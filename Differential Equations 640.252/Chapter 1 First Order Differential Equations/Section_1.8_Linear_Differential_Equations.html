Section 1.8: Linear Differential Equations

    Linear Differential Equations
  • Linear DEs are first-order DEs that have a primary function of t (a(t)) multiplied by y and a secondary function of t (b(t)) on its own
  • A linear DE has the form:
      • dy/dt = a(t) y + b(t)

  • If b(t) == 0, the DE is said to be homogenous or unforced, else it is non-homogenous or forced
  • If a(t) == λ , where λ = constant, the DE is said to be a constant-coefficient equation
    Linearity Principles
  • There are 2 linearity principles, one for homogenous linear DEs & one for non-homogenous linear DEs
      • Principle 1:
    • If y(t) is a solution to homogenous linear DE dy/dt = a(t) y, then λ y(t), where λ = constant, is also a solution
        • The general solution to any homogenous linear DE can be determined via separation, then integration:
      • dy/dt = a(t) y
      • ∫(dy/(y dt)) dy = ∫(a(t)) dt
      • ln|y| = ∫(a(t)) dt + C
      • y(t) = e∫(a(t)) dt + C

      • let k = eC
      • y(t) = k e∫(a(t)) dt

        It is observed that nonzero solutions of y(t) are all multiples of each other for every k
        y(t) = 0 is a solution to every homogenous equation

      Principle 2:
    • The general solution to any non-homogenous equation is the sum of the general solution to it associated homogenous equation and one solution to the non-homogenous equation
        • let non-homogenous DE dy/dt = a(t) y + b(t):
      • The associated homogenous DE is dy/dt = a(t)
      • let yh(t) be any solution to dy/dt = a(t), yp(t) be any solution to dy/dt = {a(t) y + b(t)}; then {k yh(t) + yp(t)}, where k = constant, is also a solution to dy/dt = {a(t) y + b(t)}
      • let yp(t) & yq(t) be 2 different solutions to dy/dt = {a(t) y + b(t)}; then {yp(t) - yq(t)}

  • A steady-state solution is a general solution whereby every specific solution tends towards a common periodic function, such as an oscillation
    Solving Linear Equations
  • A non-homogenous DE can be solved using the Method of the Undetermined Coefficient
      • First, the DE is separated by putting the t terms & y terms on opposite ends so that {dy/dt + a(t) y = b(t)}
      let α, β = coefficient required to solve for a solution yp(t) to the non-homogenous DE
      Note: if b(t) = et of some multiple, only α et is used; if b(t) = trigonometric function, α sin(t) + β cos(t) is used
      yp(t) = α b(t) would simply be substituted into the DE and solved for α
      Note: if it is not possible to algebraically solve for α as is, multiply t to b(t) and solve for α t b(t)
      Once α is solved, it is plugged into yp(t) to give the solution
      The general solution to the non-homogenous equation is the sum of the general solution to the associated homogenous DE (yh(t)) & yp(t)
    Example 1:
      Determine the general solution to DE dy/dt = (cos(t)) y:
    1. Since the DE is homogenous, it can be separated and integrated as normally
        2. ∫(dy/(y dt)) dy = ∫(cos(t)) dt
        y(t) = e∫(cos(t)) dt + C
        y(t) = k esin(t), where k = eC

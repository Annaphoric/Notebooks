Section 3.7: The Trace-Determinant Plane

    The Trace-Determinant Plane
  • The behaviour of a linear system is represented by its corresponding eigenvalues & eigenvectors

  • Given dY/dt = A Y, where A = [[a b] [c d]], Y = solution vector:
      • The characteristic polynomial is:
    • λ2 - (a + d) λ + a d - b c
    • λ2 - T λ + D, where T = trace(A) = (a + d), D = det(A) = (a d - b c)
      The corresponding eigenvalues are:
    • λ = (T +- sqrt(T2 - 4D)) / 2
        • if (T2 - 4D) > 0, the eigenvalues are real & distinct
        if (T2 - 4D) == 0, the eigenvalues are repeated
        if (T2 - 4D) < 0, the eigenvalues are complex

  • The Trace-Determinant Plane is a plot of D over T, forming a graph on the TD-plane
      • The graph is a curve (upwards parabola with a vertex at the origin) corresponding to D = T2 / 4 from T2 - 4D == 0
    • The region along the curve indicates that D == T2 / 4 and the eigenvalues are repeated
    • The region above the curve indicates that D > T2 / 4 and the eigenvalues are complex
        • In this case, the real part of the eigenvalue is a = T2 / 2 & the imaginary part of the eigenvalue is b = T2 - 4D (which is always less than zero)
        if T > 0, the solution curves of the linear system represent a spiral source
        if T == 0, the solution curves of the linear system represent a center
        if T < 0, the solution curves of the linear system represent a spiral sink
    • The region below the curve indicates that D < T2 / 4 and the eigenvalues are real
        • if T > 0, the system always has at least one positive eigenvalue
      • if D > 0, the system will have another positive eigenvalue
      • if D == 0, the system will also have a zero eigenvalue
      • if D < 0, the system will also have a negative eigenvalue
        if T == 0, the system always has at least one zero eigenvalue
      • if D > 0, the system will also have a positive eigenvalue
      • if D == 0, the system will have another zero eigenvalue
      • i D < 0, the system will also have a negative eigenvalue
        if T < 0, the system always has at least one negative eigenvalue
      • if D > 0, the system will also have a positive eigenvalue
      • if D == 0, the system will also have a zero eigenvalue
      • if D < 0, the system will have another negative eigenvalue

        if there is at least one positive eigenvalue but no negative eigenvalues, the origin is a source
        if there is both a negative & positive eigenvalue, the origin is a saddle
        if there is at least one negative eigenvalue but no positive eigenvalues, the origin is a sink
    The Parameter Plane
  • The trace-determinant plane is an example of a parameter plane
  • The entries of a matrix for a system are parameters that can be adjusted
  • This in turn would cause a change in the trace & determinant of the matrix and in the behaviour of the system entirely
  • However, the trace-determinant plane does not give information about which eigenvectors are linearly independent and the direction of the solution curves (either clockwise or anticlockwise)
  • let matrix A = [[a b] [c d]];
      • The solutions to the corresponding DE rotate clockwise if b is positive but anticlockwise if b is negative

  • Usually small changes in the parameters do not greatly affect the behaviour of the linear system
  • However, there are three such critical loci where a bifurcation along the trace-determinant plane does occur (a heavy change in behaviour):
      • The positive D-axis
    • When moving from left to right (increasing T), the system changes from a spiral sink to a center at the D-axis and lastly to a spiral source
      The T-axis
    • When moving from bottom to top (increasing D), the system changes from a saddle to a sink if T < 0, from a saddle to a center if D == 0, or from a saddle to a source if D > 0
      The repeated-root parabola
    • When moving from bottom to top (increasing D), the system changes from a sink to a spiral sink if T < 0, or a source to a spiral source if T > 0

  • Bifurcations always occur at D == 0 (across the T-axis) or D == T2/4 (across the repeated-root parabola)

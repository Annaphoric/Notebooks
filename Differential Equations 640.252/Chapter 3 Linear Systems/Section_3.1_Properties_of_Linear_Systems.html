Section 3.1: Properties of Linear Systems

    Linear Systems
  • A linear system is a system containing only first order/degree dependent variables
      • In other words, the differential equations within the system are linear functions of the dependent variables

  • let linear system {dx/dt = a x + b y; dy/dt = c x + d y}, where {a, b, c, d} = constant coefficients
  • This linear system is considered autonomous because its differentials depend only on dependent variables (a, b, c, d are constant parameters)
      • Since there are 2 dependent variables, the system is said to be 2-dimensional or planar

  • The constant coefficients can be written in matrix form:
      • let coefficient matrix A = [[a, b], [c, d]], column vector Y = {x; y}
    • The dot product of A & Y would be:
        • A Y = {{a, b} {c, d}} ⋅ {x; y} = {{a x + b y} {c x + d y}}

  • let vectorY(t) = {x(t); y(t)}
      • The derivative of the vector is written as:
    • dY/dt = {dx/dt; dy/dt}
      The equations can then be written as:
    • {dx/dt; dy/dt} = {{a, b} {c, d}} ⋅ {x; y}
    • {dx/dt; dy/dt} = A Y
    • dY/dt = A Y
  • The dimensions are number of dependent variables a vector has, corresponding to the number of its components
    Equillibrium Points of Linear Systems
  • let vector Y = (x, y), coefficient matrix A, vector field dY/dt = A Y
      • Point vector Y0 is a equillibrium point only if the vector given by the dY/dt at Y0 is the zero vector 0:
    • A Y0 = 0
      vector Y(t) = (0, 0) is always a solution (trivial) to the linear system
    • The point that determines this solution is Y0 = 0, which is an equillibrium point

      Solutions to the system must satisfy {{a x0 + b y0} {c x0 + d y0}} = {{0} {0}}
    • As long as there is at least 1 nonzero one coefficient, the solution can be solved based on the coefficients of the equations:
        • x0 = -b/a y0
        c (-b/a) y0 + d y0 = 0
        (a d - b c) y0 = 0
    • Based on these equations, if y0 == 0 is true, x0 is also true, giving a trivial solution
    • Nontrivial equillibrium solutions are determined by calculating (a d - b c = 0), which is known as the determinant
        • In other words, the determinant (cross product) of matrix A = {{a, b} {c, d}} is equal to scalar det(A) = a d - b c
        if det(A) == 0 is false, the only equillibrium point for the linear system is the origin or zero vector because the coefficients won't allow for a computable equillibrium solution
        Matrices whose determinant is zero are considered singular/degenerate and those that are not zero are considered nondegenerate
    The Linearity Principle
  • Linear systems have many useful and convenient properties that allow them to be a formidable model

  • let linear system dY/dt = A Y, where A = coefficient matrix, Y = dependent variable vector; the Linearity Principle or Principle of Superposition says:
      • if Y(t) is a solution to the system, then k Y(t) is also a solution, where k = constant
      if Y1(t) && Y2(t) are 2 different solutions to the system, then the sum of the solutions is also a solution
      In other words, any linear combination of solutions (k1 Y1(t) + k2 Y2(t)) is a solution to the system
  • The multiplication of matrices can be expressed as follow:
      • A(k Y) = k A Y

      A(Y1 + Y2) = A Y1 + A Y2
    Solving Initial Value Problems
  • The linear combination of any 2 solutions can be a solution at a specific initial value point/problem (IVP) even if the solutions individually do not coincide with the IVP
  • This combination of solutions can be determined by solving for the constants k1 & k2 in a 2-parameter family of solutions

  • 2 vectors that do not lie on the same line through the origin are considered linearly independent
  • let linearly independent vectors Y1 = (x1, y1), Y2 = (x2, y2), there exists any parameters k1 & k2 so that the IVP Y0(0) = (x0, y0) is satisfied:
      • k1 {{x1} {y1}} + k2 {{x2} {y2}} = {{x0} {y0}}
      This linear combination is also considered the general solution of the system
    Harmonic Oscillators
  • As described previously, a harmonic oscillator is a model used to study the motion of an object (mass) attached to a spring
  • The object is assumed to be to frictionless and unaffected by any forces except the restoring force & damping force
      • The restoring force is determined by Hooke's Law and the damping force is determined by Newton's 2nd Law of motion, giving the equation or system:
    • m d2y/dt2 + b dy/dt + k y = 0, where m = mass, b = damping coefficient, k = spring constant, y = position, t = time
    • {dy/dt = v; dv/dt = -(k y + b v) / m}
      Both differential equations within the system depend linearly on y and/or v

Section 3.2: Straight-Line Solutions

    Straight-Line Solutions
  • The geometry of a vector field came be used to determine special solutions for linear systems

  • Since the solutions to a linear system are always tangent to vectors within a vector field dY/dt, they can be drawn as straight lines with a slope according to the components of a solution vector Y(x, y) scaled to a common factor
      • This factor λ, also known as an eigenvalue, has to scale from the origin to some other point in the xy-plane

  • let coefficient-matrix A = {{a, b} {c, d}}, vector V = {{x} {y}}, there exists a factor λ to give a nonzero solution such that:
      • A V == λ V is true
      A V - λ V == 0 is also true
  • This can also be written as:
      • {{a, b} {c, d}} {{x} {y}} - λ {{x} {y}} = {{0} {0}}
      {{a x + b y} {c x + d y}} - {{λ x} {λ y}} = {{0} {0}}
      {{(a - λ) x + b y} {c x + (d - λ) y}} = {{0} {0}}
      {(a - λ) x + b y = 0; c x + (d - λ) y = 0} (This is the corresponding linear system)
      {{a - λ, b} {c, d - λ}} ⋅ {{x} {y}} = {{0} {0}}
      {{a - λ, b} {c, d - λ}} Y = 0

  • To determine if there is a nonzero solution, the determinant of the resulting matrix (A - λ I, where I is the identity matrix equal to {{1, 0} {0, 1}}) must equal zero:
      • det({{a - λ, b} {c, d - λ}}) = 0
      (a - λ) (d - λ) - b c = 0
      λ2 - (a + d) λ + (a d - b c) = 0
      λ2 - (a + d) λ + det(A) = 0
    • This equation is known as the characteristic polynomial
    • The roots (solutions) to the polynomial are the eigenvalues of the matrix A
    • If the solutions are not real, then they are considered complex eigenvalues
    • If the solutions lead to a solely a squared eigenvalue, this is known as a repeating root

  • if b || d == 0 is true, then there are nontrivial solutions for the values λ = {a, d}
  • if λ > 0, the vectors along the solution curve point away from the origin
  • if λ < 0, the vectors along the solution curve point towards the origin

  • The vector V that corresponds to an eigenvalue λ is known as an eigenvector
      • These vectors correspond to any vector within a straight-line solution of the linear system
      Eigenvectors can be solved by plugging in the corresponding eigenvalue into the previous equation:
    • A V - λ V = 0
    • A {{x} {y}} - λ {{x} {y}} = 0
    • {{a - λ, b} {c, d - λ}} V = 0
    • {(a - λ) x + b y = 0; c x + (d - λ) y = 0}

  • As long as there is an eigenvalue λ for a matrix A, then the solution Y0(t) for the linear system dY/dt = A Y using the corresponding eigenvector Vλ(x, y) is:
      • Y(t) = eλ t Vλ
      Y(t) = eλ t {{x} {y}}
      Y(t) = {{eλ t x} {eλ t y}}
  • The system at the solution Y0(t) is written as:
      • dY0/dt = {{λ eλ t x0} {λ eλ t y0}}
      dY0/dt = λ Y0(t)

  • If there are 2 distinct eigenvectors V1 & V2 for their respective corresponding real eigenvalues λ1 & λ2, these vectors are considered linearly independent
  • As long as they are linearly independent, the Linearity Principle can be applied to them:
      • For solution vectors Y1 = eλ1 t V1 & Y2 = eλ2 t V2, the general solution to the linear system is written as:
    • Y(t) = k1 Y1 + k2 Y2
    • Y(t) = k1 eλ1 t V1 + k2 eλ2 t V2

      Example 1:
        Given matrix A = {{4, 3} {-1, 0}}, determine the eigenvalues, eigenvectors & solutions to its corresponding linear system.
      1. Solving for the eigenvalues:
          2. det({{4 - λ, 3} {-1, -λ}}) = 0
          -λ (4 - λ) + 3 = 0
          λ2 - 4λ + 3 = 0
          (λ - 3) (λ - 1) = 0
          λ = {1, 3}
      2. Solving for the corresponding eigenvectors V:
          3. if λ = 1:
        • {{4 - (1), 3} {-1, -(1)}} ⋅ {{x} {y}} = 0
        • {{3, 3} {-1, -1}} ⋅ {{x} {y}} = 0
        • {3x + 3y = 0; -(x + y) = 0}
        • x + y = 0
        • y = -x

        • V1 = {{1} {-1}}

          if λ = 3:
        • {{4 - (3), 3} {-1, -(3)}} ⋅ 3 {{x} {y}} = 0
        • {x + 3y = 0; -(x + 3y) = 0}
        • x = -3y

        • V2 = {{-3} {1}}

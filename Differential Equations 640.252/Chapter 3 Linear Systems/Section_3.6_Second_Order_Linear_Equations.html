Section 3.6: Second-Order Linear Equations

    Second-Order Linear Equations
  • A method that solves the general solution to a second-order DE also solves the general solution of its associated system & vice versa

  • Given second-order DE a d2y/dt2 + b dy/dt + c y = 0, where (a, b, c) = constants, let solution y(t) = es t, where s = constant:
      • Since the DE is linear, homogenous, constant-coefficient & second-order, there is a simpler way to solve for the solution to the equation
      This method is much like the method described in the previous chapter, by differentiating the solution, substituting it into the equation & solving for s:
    • dy/dt = s es t
    • d2y/dt2 = s2 es t

    • a (s2 es t) + b (s es t) + c (es t) = 0
    • es t (a s2 + b s + c) = 0
    • a s2 + b s + c = 0
    • s = (-b +- sqrt(b2 - 4a c)) / 2a
    • s = {s1, s2}

    • Therefore, the solutions and general solution are:
        • y1(t) = es1 t
        y2(t) = es2 t

        y(t) = k1 es1 t + k2 es2 t

        Since y(t) is a solution to the second-order DE, this same solution & its derivative can be used to write the solution to its associated linear system dY/dt = [[0 1] [-10 -7]] Y, where Y(t) = [[y(t)] [v(t)]]:
      • Y(t) = [[k1 es1 t + k2 es2 t] [k1 s1 es1 t + k2 s2 es2 t]]
      • Y(t) = k1 es1 t [[1] [s1]] + k2 es2 t [[1] [s2]]

  • If an IC {y(0) = y0; v(0) = v0} is given, the constants k1 & k2 can be solved by setting the solution equations equal to y0 & y'0 at t = 0

  • This can also be applied the complex solutions where y1(t) is the real part of the solution & y2(t) is the imaginary part of the solution to find the general solution y(t) to the second-order DE
  • v1(t) & v2(t) is solved by differentiating the solutions to find the general solution Y(t) = [[y(t)] [v(t)]] of its associated linear system
    Classifying Harmonic Oscillators
  • The classification of a harmonic oscillator is dependent on how damped it is, that is, how much energy is dissipated into the surroundings

  • Given second-order DE (m d2y/dt + b dy/dt + k y) = 0, where m = mass, b = damping coefficient, k = spring constant, y = position, t = time, the characteristic polynomial is written as:
      • es t(m s2 + b s + k) = 0, where y(t) = es t
      m s2 + b s + k = 0
      s = (-b +- sqrt(b2 - 4mk)) / 2m
    • if b == 0:
        • This is considered an undamped or simple harmonic oscillator
        s = +-sqrt(-4mk) / 2m
        s = i sqrt(k) / sqrt(m)
        s = i sqrt(k/m)

        y(t) = ei ω t, where ω = sqrt(k/m)
        y(t) = cos(ω t) + i sin(ω t)
        y(t) = k1 cos(ω t) + k2 sin(ω t)
        v(t) = -k1 ω sin(ω t) + k2 ω cos(ω t)

        Y(t) = k1 [[cos(ω t)] [-ω sin(ω t)]] + k2 [[sin(ω t)] [ω cos(ω t)]]

        The natural period is 2π/ω and the natural frequency is ω/2π (angular frequency is ω itself)

    • if (b2 - 4mk) < 0:
        • This is considered an underdamped harmonic oscillator
        The general solution will solely be complex and separated into its real & imaginary parts in the form:
      • y(t) = k1 e-m t / 2 cos(t sqrt(b2 - 4mk) / 2) + k2 e-m t / 2 sin(t sqrt(b2 - 4mk) / 2)

    • if (b2 - 4mk) > 0:
        • This is considered an overdamped harmonic oscillator
        The general solution will solely be real in the form:
      • y(t) = k1 et (-b - sqrt(b2 - 4mk)) / 2 + k2 et (-b + sqrt(b2 - 4mk)) / 2

    • if (b2 - 4mk) == 0:
        • This is considered a critically damped harmonic oscillator
        The general solution will solely be real containing a repeated root in the form:
      • y(t) = k1 e-b t / 2 + k2 t e-b t / 2

  • Since m > 0 is always true, the nature of the origin or eigenline is always a sink

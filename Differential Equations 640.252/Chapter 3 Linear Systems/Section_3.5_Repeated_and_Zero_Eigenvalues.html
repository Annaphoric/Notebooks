Section 3.5: Repeated and Zero Eigenvalues

    Repeated Eigenvalues
  • Linear systems with only one eigenvalue or an eigenvalue having a value of zero are considered degenerate
      • They are fairly uncommon compared to linear systems with 2 distinct eigenvalues

  • A repeated eigenvalue is observed when the characteristic polynomial makes a perfect square
      • In this case, there will be a double root for one eigenvalue

  • Given linear system dY/dt = A Y at IC Y(t) = V0, having one distinct eigenvalue λ & two corresponding eigenvectors V0 (from the IC) & V1, the general solution Y(t) will have the form:
      • Y(t) = eλ t V0 + t eλ t V1
  • The solution can be checked by differentiating it and equating it to the system:
      • dY/dt = λ eλ t V0 + eλ t V1 + λ t eλ t V1
      dY/dt = eλ t (λ V0 + V1) + t eλ t (λ V1), separating like terms

      A Y = eλ t A V0 + t eλ t A V1, this is obtained by multiplying the solution by A

      eλ t (λ V0 + V1) + t eλ t (λ V1) = eλ t (A V0) + t eλ t (A V1), since dY/dt = A Y
    • In order for this equation to be true, the respective components must equal each other:
        • {A V0 = λ V0 + V1; A V1 = λ V1}
        {(A - λ I) V0 = V1; (A - λ I) V1 = 0}
      • The first equation explains that V0 is always an eigenvector as long as V1 == 0 is true, but V0 is not an eigenvector if V1 == 0 is false
      • The second equation explains that V1 is always an eigenvector as long as it itself is the nonzero vector
      • Since V1 depends on V0, Y(t) is a straight-line solution only if V0 is an eigenvector
  • As described, V1 can be solved by solving for (A - λ I) V0, where V0 is the IC
      • These vectors can then be used to find the general solution of a linear system based on its general form, giving an easier way to solve for the solution as long as the eigenvalue is repeated

  • The solution curves always spiral inwards or outwards with respect to the origin & tangent to one straight-line solution
      • if λ < 0, the origin is a sink
      if λ > 0, the origin is a source

  • Solutions curves with repeating eigenvalues appear to be in between systems that contain complex eigenvalues & systems that only contain real, distinct eigenvalues

  • In order for a system to have every vector considered to be an eigenvector is:
      • dY/dt = [[λ, 0] [0, λ]] Y
    • Since the system is completely decoupled, the solution curves form a ray that point towards the origin if λ < 0 (sink) or away from the origin if λ > 0 (source)
    Zero Eigenvectors
  • Given linear system dY/dt = A Y, having two real, distinct eigenvalues (λ1, λ2) = (0, λ0) and their corresponding eigenvectors V1 & V2, the general solution can be written as:
      • Y(t) = k1 eλ1 t V1 + k2 eλ2 t V2
      Y(t) = k1 V1 + k2 eλ2 t V2, since λ == 0 is true
    • This equation explains that if k2 == 0, all solutions are considered equillibrium solutions
    • Every point on the line of eigenvectors k1 V1 is an equillibrium
    • Every solution will point inwards or outwards from the equillibrium line tangent to the line made by all eigenvectors V2
        • if λ2 < 0, the solutions point inwards towards the equillibrium line
        if λ2 > 0, the solutions point outwards away from the equillibrium line

    Example 1:
      Find the general solution of system dY/dt = A Y, where A = [[-2 1] [0 -2]].
    1. This linear system can be writen as a system of equations:
        2. {dx/dt = -2x + y; dy/dt = -2y}
        This system is a linear, first-order, partially decoupled system because dx/dt is nonhomogenous & dy/dt is autonomous
        The solution for dy/dt is y(t) = ky e-2t
      • In this case, eigenvalue λ = -2 (Note: this can also be solved using the characteristic polynomial)

    2. Solving for solution x(t):
        3. dx/dt = -2x + ky e-2t
        dx/dt + 2x = ky e-2t, separating
        d((e2t) x) dt = d((e2t) ky e-2t)/dt, multiplying by integrating factor
        (e2t) x = ky t + kx, integrating
        x(t) = ky t e-2t + kx e-2t

    3. The general solution Y(t) for the system can now be written as:
        4. Y(t) = [[ky t e-2t + kx e-2t] [ky e-2t]]
        Y(t) = e-2t [[kx] [ky]] + t e-2t [[ky] [0]]
    Example 2:
      Determine the behaviour of the solutions for linear system dY/dt = A Y, where A = [[-3 1] [3 -1]].
    1. Solving for the eigenvalues:
        2. λ2 - ((-3) + (-1)) λ + det(A) = 0
        λ2 + 4λ = 0
        λ (λ + 4) = 0
        λ = {0, -4}
        Since one eigenvalue is zero and the other is negative, the solutions will point towards an equillibrium line
    2. Solving for the corresponding eigenvectors:
        3. {(-3 - 0) x + y = 0; 3x + (-1 - 0) y = 0}
        y = 3x
        V0 = [[1] [3]]

        {(-3 - (-4)) x + y = 0; 3x + (-1 - (-4)) y = 0}
        y = -x
        V-4 = [[1] [-1]]
    3. Writing the solutions and general solution:
        4. Y0 = e(0)t [[1] [3]]
        Y0 = [[1] [3]]

        Y-4 = e-4t [[1] [-1]]

        Y(t) = k0 [[1] [3]] + k-4 e-4t [[1] [-1]]
      • In this case:
          • x(t) = k0 + k-4 e-4t
          y(t) = 3k0 - k-4 e-4t

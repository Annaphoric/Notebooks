Section 3.4: Complex Eigenvalues

    Solutions with Complex Eigenvalues
  • Complex numbers are number that have the form:
      • a + b i, where {a, b} = real numbers & i = imaginary number corresponding to sqrt(-1)
    • 'a' is considered to be the real part of the number & "b i" is considered dto be the imaginary part

  • As mentioned in previous sections, real eigenvalues are used to determine straight-line solutions to a linear system in matrix form
  • Complex eigenvalues are eigenvalues in the form of a complex number
  • They are used to find non-straight-line solutions in the same way that real eigenvalues find straight-line solutions

  • Euler's Formula is used to determine real solutions from complex solutions:
      • eb i = cos(b) + i sin(b)
      ea eb i = ea (cos(b) + i sin(b))
      ea + b i = ea (cos(b) + i sin(b))
  • if b = π:
      • eπ i = cos(π) + i sin(π)
      eπ i = -1

  • A complex solution Y(t) = e(a + b i) t Vλ, where Vλ = [[x0] [y0]] (eigenvector of corresponding complex eigenvalue (λ = a + b i)) can be written as:
      • Y(t) = ea t (cos(b t) + i sin(b t)) Vλ
      Y(t) = ea t (cos(b t) + i sin(b t)) [[x0] [y0]]
      Y(t) = ea t (x0 [[cos(b t)] [sin(b t)]] + i y0 [[cos(b t)] [sin(b t)]])
    • The solution can vary depending on the eigenvector (value of x0 & y0), however, it will always be written as a complex number
    • The components of a complex solution are the real solution component YR & the imaginary solution component YZ (which is multiplied by i)
  • As long as both components to a complex solution exist, the linear system dY/dt = A Y0 can be written as:
      • A Y0 = A (YR + i YZ)
      dY/dt = A YR + i A YZ
      dYR/dt + i dYZ = A YR + i A YZ
    • This equation equates 2 complex numbers to each other
    • In order for the 2 complex numbers to truly be equal to each other, their respective real components & imaginary components must both equal to each other:
        • dYR/dt == A YR && dYZ/dt == A YZ must be true
  • So, the general solution Y(t) can be shown as the linear combination of both components:
      • Y(t) = k1 YR(t) + k2 YZ(t)
      Note: This only requires using one complex eigenvalue, since all complex eigenvalues for a given linear system have the same real & complex solution components
    Phase Portraits of Linear Systems having Complex Eigenvalues
  • The geometry of the vector field of a linear system containing complex solutions depends on 2 terms: its exponential component ea t & its trigonometric component (cos(b t) + i sin(b t))
      • if a > 0: the solution curve spirals outwards towards infinity as t increases
    • This is known as a spiral source
      if a < 0: the solution curve spirals inwards towards the origin as t increases
    • This is known as a spiral sink
      if a == 0: the solution curve has a constant amplitude, forming a circle
    • This is known as a center

      The trigonometric terms cos(b t) & sin(b t) cause the solution curve to oscillate with a period of 2π/b
    • This is the natural period
    • The natural frequency is the inverse of the natural period (in other words, the frequency is the amount of oscillations per unit of time)
    • The product of the natural period & frequency is always one
    • If the unit of the natural frequency is radians, its value is simply b, which corresponds to the angular frequency

    Example 1:
      Find the general solution of linear system dY/dt = A Y where A = [[-2, -3] [3, -2]], & Y(t) = [[x(t)] [y(t)]]
    1. Solving for the eigenvalues:
        2. λ2 - ((-2) + (-2)) λ + det(A) = 0
        λ2 + 4λ + 13 = 0
        λ = {(-4 - sqrt(-36)) / 2, (-4 + sqrt(-36)) / 2}
        λ = {-2 - 3i, -2 + 3i}

    2. Solving for corresponding eigenvectors:
        3. {(-2 - (-2 - 3i)) x - 3y = 0; 3x + (-2 - (-2 - 3i)) y = 0}
        {-3y + 3i x = 0; 3x + 3i y = 0}
        {y - i x = 0; x + i y = 0}
        y = i x
        V1 = [[1] [i]]

        {(-2 - (-2 + 3i)) x - 3y = 0; 3x + (-2 - (-2 + 3i)) y = 0}
        {-3y - 3i x = 0; 3x - 3i y = 0}
        {y + i x = 0; x - i y = 0}
        x = i y
        V2 = [[i] [1]]

    3. Accounting for Euler's Formula, the solutions are written as:
        4. Y1(t) = e(-2 - 3i) t [[1] [i]]
        Y1(t) = e-2t (cos(-3t) + i sin(-3t)) [[1] [i]]
        Y1(t) = e-2t [[cos(-3t) + i sin(-3t)] [i cos(-3t) - sin(-3t)]]
        Y1(t) = e-2t ([[cos(-3t)] [-sin(-3t)]] + i [[sin(-3t)] [cos(-3t)]])

        Y2(t) = e(-2 + 3i) t [[i] [1]]
        Y2(t) = e-2t (cos(3t) + i sin(3t)) [[i] [1]]
        Y2(t) = e-2t ([[-sin(3t)] [cos(3t)]] + i [[cos(3t)] [sin(3t)]])

    4. Since Y1(t) == Y2(t) is always true, the general solution can be written as:
        5. Y(t) = e-2 t (k1 [[-sin(3t)] [cos(3t)]] + k2 [[cos(3t)] [sin(3t)]])

Section 4.2: Sinusoidal Forcing

    Complexification of Sinusoidal Forcing
  • let forced harmonic oscillator DE $\frac{d^2t}{dt^2} + p \frac{dy}{dt} + q y = g(t)$, where $g(t) = $ sinusoidal function;
      • The external force $g(t)$ is periodic and has an average value of zero
      if $g(t) = sin(ω t)$ | $g(t) = cos(ω t)$: period $T = \frac{2π}{ω}$ & frequency $f = \frac{ω}{2π}$

  • To solve for the specific solution for a non-homogenous DE if the external force is sinusoidal, a constant-coefficient $α$ can be solved using the complex solution:
      • $y_c(t) = α e^{it}$
      $y_c(t) = α (cos(t) + i sin(t))$
      $y_c(t) = y_r(t) + i y_i(t)$, where $y_r(t) = $ real component of solution & $y_i(t) = $ imaginary component of solution

      This solution is used for its associated complexed DE $\frac{d^2t}{dt^2} + p \frac{dy}{dt} + q y = e^{it}$

  • The specific solution $y_p(t)$ can also be written in polar form by solving for the polar form of the complexed solution $y_c(t)$ given constant-coefficient $α$:
      • $α_θ = |α| e^{iθ}$, where $θ = atan(\frac{α_i}{α_r})$ (imaginary component & real component of constant-coefficient respectively)

      $y_c(t) = α_θ e^{iθ}$
      $y_c(t) = |α| e^{it} e^{iθ}$
      $y_c(t) = |α| e^{i(t + θ)}$
      $y_c(t) = |α| (cos(t + θ) + i sin(t + θ))$

  • A $sin(θ)$ function can be converted to a $cos(θ)$ function by the trigonometric formula:
      • $sin(θ) = cos(θ - \frac{π}{2})$

  • For any IC, the solutions of a general sinusoidal non-homogenous DE tends to the specific non-homogenous solution as t increases
    Example 1:
      Find the general solution of non-homogenous DE $\frac{d^2y}{dt^2} + 2 \frac{dy}{dt} + 2y = sin(t)$, then write it in polar form:

    1. Solving for the eigenvalues & general solution $y_h(t)$ for the associated homogenous DE:
        2. $λ^2 + 2λ + 2 = 0$
        $λ = \frac{-2 \pm \sqrt{-4}}{2}$
        $λ = \{-1 \pm i\}$

        General solution: $y_h(t) = k_1e^{-t} cos(t) + k_2e^{-t} sin(t)$

    2. Solving for the specific solution for the complexed non-homogenous DE and its components:
        3. let $y_c(t) = α e^{it}$;
        $\frac{dy_c}{dt} = αi e^{it}$
        $\frac{d^2y_c}{dt^2} = -α e^{it}$

        $\frac{d^2y_c}{dt^2} + 2 \frac{dy_c}{dt} + 2y_c = e^{it}$
        $-α e^{it} + 2αi e^{it} + 2α e^{it} = e^{it}$
        $α(1 + 2i) = 1$
        $5α = 1 - 2i$ (multiplying both sides by complex conjugate)
        $α = \frac{1 - 2i}{5}$
        $α_θ = \frac{\sqrt{5}}{5} e^{i atan(-2)}$

        Complexed solution: $y_c(t) = \frac{1 - 2i}{5} e^{it}$
        $y_c(t) = \frac{1 - 2i}{5} (cos(t) + i sin(t))$
        $y_c(t) = \frac{1}{5}((cos(t) + 2 sin(t)) + i(-2 cos(t) + sin(t)))$

        Real component: $y_r(t) = \frac{1}{5}(cos(t) + 2 sin(t))$
        Imaginary component: $y_i(t) = \frac{1}{5}(-2 cos(t) + sin(t))$

    3. Writing the specific & general solution for the given DE:
        4. Specific solution: $y_p(t) = y_i(t)$

        General solution: $y(t) = k_1e^{-t} cos(t) + k_2e^{-t} sin(t) + \frac{1}{5}(-2 cos(t) + sin(t))$
        $y(t) = k_1e^{-t} cos(t) + k_2e^{-t} sin(t) + \frac{\sqrt{5}}{5} cos(t + atan(-2) - \frac{π}{2})$
    Example 2:
      Find the general solution of non-homogenous DE $\frac{d^2y}{dt^2} + 2 \frac{dy}{dt} + 10y = 4 cos(2t)$ and write it in polar form:

    1. Solving for general solution of associated homogenous DE:
        2. $λ^2 + 2λ + 10 = 0$
        $λ = \frac{-2 \pm \sqrt{-36}}{2}$
        $λ = \{-1 \pm 3i\}$

        $y_h(t) = k_1 e^{-t} cos(3t) + k_2 e^{-t} sin(3t)$

    2. Solving for the specific solution of the complexed DE in cartesian form:
        3. $y_c(t) = α e^{2it}$
        $\frac{dy_c}{dt} = 2αi e^{2it}$
        $\frac{d^2y_c}{dt^2} = -4α e^{2it}$

        $\frac{d^2y}{dt^2} + 2 \frac{dy}{dt} + 10y = 4e^{2it}$
        $-4α e^{2it} + 4αi e^{2it} + 10α e^{2it} = 4e^{2it}$
        $α(3 + 2i) = 2$
        $α = \frac{2(3 - 2i)}{13}$

        $y_c(t) = \frac{2(3 - 2i)}{13} e^{2it}$
        $y_c(t) = \frac{2(3 - 2i)}{13} (cos(2t) + i sin(2t))$
        $y_c(t) = \frac{2}{13}((3 cos(2t) + 2 sin(2t)) + i(-2 cos(2t) + 3 sin(2t)))$

    3. Solving for the specific solution of the complexed DE in polar form:
        4. $α_θ = \frac{\sqrt{52}}{13} e^{i atan(-\frac{2}{3})}$
        $α_θ = \frac{2 \sqrt{13}}{13} e^{i atan(-\frac{2}{3})}$

        $y_{cθ}(t) = \frac{2 \sqrt{13}}{13} e^{i atan(-\frac{2}{3})} e^{2it}$
        $y_{cθ}(t) = \frac{2 \sqrt{13}}{13} e^{i(2t + atan(-\frac{2}{3})}$
        $y_{cθ}(t) = \frac{2 \sqrt{13}}{13} (cos(2t + atan(-\frac{2}{3})) + i sin(2t + atan(-\frac{2}{3})))$

    4. Writing the specific & general solution for the given DE in both cartesian & polar form:
        5. Specific cartesian solution: $y_p(t) = \frac{2}{13}(3 cos(2t) + 2 sin(2t))$

        General cartesian solution: $y(t) = k_1 e^{-t} cos(3t) + k_2 e^{-t} sin(3t) + \frac{2}{13}(3 cos(2t) + 2 sin(2t))$

        Specific polar solution: $y_{pθ}(t) = \frac{2 \sqrt{13}}{13} cos(2t + atan(-\frac{2}{3}))$

        General polar solution: $y_θ(t) = k_1 e^{-t} cos(3t) + k_2 e^{-t} sin(3t) + \frac{2 \sqrt{13}}{13} cos(2t + atan(-\frac{2}{3}))$

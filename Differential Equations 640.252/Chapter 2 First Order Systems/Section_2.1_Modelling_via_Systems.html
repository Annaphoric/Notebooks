Section 2.1: Modelling via Systems

    Predator-Prey Systems
  • let R(t) = population of prey, F(t) = population of predator, dR/dt = {2R - 1.2R F}, dR/dt = {0.9R F - F};
  • dR/dt shows how the population of prey changes over time and dF/dt shows how the population of predator changes over time
  • The proportionality coefficient (-1.2) shows how the predator interacts with the prey to decrease the population of the prey and (0.9) shows how the predator interacts with the prey to increase the population of the predator
  • At dR/dt = 0 && dF/dt = 0, the populations of both are not changing
      • A function of R(t) and F(t) that would case dR/dt && dF/dt to have a value of zero is known as one equillibrium solutions consisting of a pair of constant functions
      An obvious equillibrium solution would be where R(t) && F(t) both equal zero:
    • {R(t) = 0, F(t) = 0}
      Another way to solve for an equillibrium solution is by setting dR/dt && dF/dt equal to zero and setting the 2 equations within the system equal to each other:
    • dR/dt = (2 - 1.2F) R; dF/dt = (0.9R - 1) F
    • (2 - 1.2F) R = 0; (0.9R - 1) F = 0
    • (2 - 1.2F) R = (0.9R - 1) F
    • R(t) = 1.11; F(t) = 1.67

    • In other words, when the population of prey is R(t) = 1.11 && the population of predator F(t) = 1.67, the system is at equillibrium (the population are perfectly maintained indefinitely) as long as there are no confounding factors

  • if R(t) = 0: dR/dt == 0 & dF/dt == -F;
      • This means that there is no change in prey population; however, the predator population will continue to decrease and can never increase as long as there is no prey

  • if F(t) = 0: dF/dt == 0 & dR/dt == 2R;
      • This means that there is no change in predator poulation; however, the prey population will theoretically continue to increase indefinitely
      In reality, the prey population follows more of a logistic growth model rather than exponential

  • An initial condition solves for a specific solution of the DE by solving for an explicit value for the DE first, then equations for R(t) & F(t) are written afterwards in the form of a system
  • An initial condition of R(0) = R0 & F(0) = F0 where (R0, F0) = (1, 0.5)
  • A solution curve is the graph made by the systems of equations that consists of the initial condition
      • This graph is plotted as a plane where the axes are the values of R(t) & F(t) over time; it is also known as a phase plane, which is analagous to a phase line
      The initial point (R0, F0) is the point where the curve begins
      The curve forms a cycle that eventually returns back to the initial point after a certain amount of time
      A phase portrait is a collection of multiple solution curves
  • An equillibrium solution on a solution curve is shown as point rather than a curve because it is always constant (dR/dt = dF/dt = 0), known as an equillibrium point
    The Motion of an Object attached to a Spring
  • Assuming a mass is horizontally connected to a spring on a frictionless table, the undamped motion of the spring (ignoring air resistance & other forces) can relate to the the model described by the predator-prey system
  • The dependent variable is the force of the spring in motion
  • The 2 independent variables are the displacement from equillibrium (resting position) and time
  • The 2 quantities in the mass-spring model are the mass of the object & the spring constant (specific to the type of spring)
  • let y = displacement from equillibrium position, F = force due to acceleration, Fs = restoring force, t = time, m = mass, k = spring constant, a = acceleration, v = velocity;
  • A (-) y value indicates that the spring is compressed whilst a (+) y value indicates that the spring is stretched
  • The formula that describes the undamped motion of a spring is based on Newton's 2nd Law & Hooke's Law:
      • F = m a
      F = m d2y/dt2

      Fs = -k y, (Note: This equation is negative because the restoring force always points towards the direction opposite to the position of the mass)

      m d2y/dt2 = -k y
      d2y/dt2 + k y / m = 0
    • This equation shows a spring in simple harmonic oscillation
    • This is a second-order DE that relates the acceleration to the position over time

  • Another way to express acceleration is by using a first-order DE that relates the acceleration to the velocity over time
  • The system of equations is written as:
      • dy/dt = v, dv/dt = -k y / m

      if k / m = 1: dv/dt = -y;
  • The system of equations {dy/dt = v; dv/dt = -y} can be satisfied if y(t) = cos(t) && v(t) = -sin(t)
      • This solution can be written as (y(t), v(t)) = (cos(t), -sin(t))
      This forms a solution curve which happens to be a unit circle centered at the origin with a radius of one
      The circle is parsed clockwise with increasing time since the equation of v(t) is negative

  • The vt-plane, yt-plane & yv-plane can be combined together to form a unique yvt-graph, having the convenient shape of a coil

Section 2.2: The Geometry of Systems

    Vector Fields for Predator-Prey Systems
  • A vector is a matrix of dimensions (1 x m), where m = columns
  • Given system of equations {dR/dt = 2R - 1.2R F, dF = 0.9R F - F}, a solution curve (R(t), F(t)) can be written as a vector function P(t), having 2 components:
      • P(t) = (R(t); F(t))

  • The derivative of a vector function is the derivatives of all its components:
      • dP/dt = (dR/dt; dF/dt)
      dP/dt = (2R - 1.2R F; 0.9R F - F)

  • A vector field V can be expressed as:
      • V(R, F) = (2R - 1.2R F, 0.9R F - F)
    • This vector takes 2 inputs as solutions, one for R & one for F
  • The solutions R(t) = 2 & F(t) = 1 can be written as (R, F) = (2, 1) and as a vector field:
      • V(2, 1) = (2(2) - 1.2(2)(1), 0.9(2)(1) - (1))
      V(2, 1) = (1.6, 0.8)

  • Since P(t) uses the functions themselves as components and V(R, F) using the functions' derivatives as components, the derivative of vector function P is the vector field V itself
      • dP/dt = V(R, F)
      dP/dt = V(P)

  • The vectors that make up this field lie on the RF-plane, each having an R-component and F-component
      • This results in vectors varying in magnitude & direction: Magnitude = sqrt(R2 + F2), Direction = atan(F / R)
      To better visualise the graph made by the vector field, the vectors can be shown as unit vectors, having the same direction as an original vector but a magnitude of one
    • This creates a direction field

    The Geometry of Vector Fields & its Solutions
  • let systems of equations (SoE) {dx/dt = f(x, y); dy/dt = g(x, y)}, solution vector Y(t) = (x(t), y(t)), vector field F(x, y) = (f(x, y), g(x, y))
  • The derivative of the solution vector can be written as:
      • dY/dt = (dx/dt; dy/dt)
      dY/dt = (f(x, y), g(x, y))
      dY/dt = F(Y)

  • if dx/dt = x & dy/dt = y: the vectors with vector field F(x, y) = (x, y) always point away from the origin
  • if dx/dt = -x & dy/dt = -y: the vectors with vector field F(x, y) = (-x, -y) always point towards from the origin

  • The solution vector Y(t) = (x(t), y(t)) can be considered a parameterization of a solution curve on the xy-plane
  • Therefore, the derivative of the solution vector (dY/dt = F(Y)) represents the tangent vectors of the solution curves, which can also be represented by the vectors within the vector field themselves
  • Given dY/dt = F(Y) = F(x, y), a point Y0 = (x0, y0) is an equillibrium point if F(Y0) = 0 (zero vector)
      • Solution vector Y(t) = Y0 is then considered an equillibrium solution
  • In other words, dY/dt = (dx/dt, dy/dt) == (0, 0) must be true for all t
    Example 1:
      Given SoE {dx/dt = 3x + y; dy/dt = x - y}, determine the equillibrium point(s).
    1. Setting the derivatives equal to zero and solving the system:
        2. {3x + y = 0; x - y = 0}
        3x + y = x - y
        2x = -2y
        x = -y
        (x, y) = (0, 0)
      • The vectors never touch this point (the origin), they instead are repelled by it

Chapter 2: Describing Data

    Organising Data
  • Data can be organised into graphs depending on the type of variable

  • Pie Charts
      • Used for displaying categorial data
      The thickness of a "slice" represents the frequency of a variable's value
      Data is organised into a circle

  • Bar Graphs
      • Used for displaying categorial data
      The height of a bar represents the frequency of a variable's value
      Data is organised into evenly-spaced bars that do not overlap
      Segmented Bar Graph
    • Used for displaying 2 or more categorical variables
    • The variables are expressed as distinct colours
      Side-by-Side Bar Graph
    • Used for displaying 2 or more categorical variables
    • Different variables are placed in specific regions of the graph

  • Histogram
      • Used for displaying numerical data
      The height of the bar represents the frequency of a variable's value
      Data is organised into evenly-spaced bars that touch
      Data that increases in height then decreases at an equal amount is symmetric
      Data is considered bell-shaped if it has a parabolic shape (mean = median)
      Data that increases in height, but gradually decreases is asymmetric and skewed to the right (mean > median)
      Data that gradually increases in height, then decreases is asymmetric and skewed to the left (mean < median)

  • Stem and Leaf Plot
      • Used for displaying numerical data
      Data is organised as a table containing a "Stems" on the left & a "Leaves" columns on the right. The Stems column contains the tens, hundreds.. and so on (other than the ones place) place whilst the Leaves column contains solely the one place of the elements. The elements within the Leaves column are usually placed in ascending order starting from 0. If the elements in the Leaves column are not in the ones place, they must be explicitly stated as the smallest given decimal value

  • Dot Plot
      • Used for displaying numerical data
      Data is organised as a 1-dimesional plot containing stacked marks

  • Box Plot
      • Used for displaying numerical data
      Data is organised such that the values of a 5-Number Summary and outliers are all shown explicitly
      A box with length IQR is divided by the median. 2 lines stretch on opposite ends to demonstrate the lowest & highest relevant values

  • Side-by-Side Graphs
      • Used for displaying both numerical & categorical data
      Numerical data is first organised into a boxplot, dotplot or histogram. Then, every categorical variable corresponding to a set of numerical data is arranged side-by-side one another. Multiple graphs of numerical are therefore shown and arranged according to a categorical variable

  • Scatterplot
      • Used for displaying numerical data
      The values for 2 numerical variables are plotted as dots on a graph where one is plotted vertically and the otheris plotted horizontally
    Categorical Variables
  • As mentioned in the previous example, categorical variables are variables that represent a non-numerical identity such as gender or education level

  • The proportion or relative frequency/amount of participants in a certain category can be measured by dividing the number of participants in that category by the total number of participants
      • A frequency table may sometimes be used to observe the proportions in a study in the form of a table
      The proportion for a population is denoted by 'p'
      The proportion for a population is denoted by 'p' or p with a hat

  • If there are 2 or more categorical variables within a study, a 2-way table may be used to observe a possible relationship/association between the 2 variables

    Shape & Centering for One Quantitative Variable
  • As mentioned in the previous chapter, quantitative variables give a numerical value for a characteristic

  • An outlier is a value of a variable that is noticeably different from other variables. In other words, it lies outside a median range of values; it can be either much higher of much smaller than the average

  • Dotplots or histograms are often used to observe the distribution of data

  • The mean is the average of all data values for one variable. In a graph, it is the point that balances out the dataset. It is the sum of all values x divided by the total number n of values:
      • avg = Σk = 1n(xk) / n
      A common way to express mean in a sample is x with a line over it. In a population, μ is used

  • The median of a dataset is the middle value. If there is an even number of values in the dataset, the average of both middle values is found. The median gives information about each half of the dataset

  • Resistance demonstrates the impact of outliers within the dataset, that is, how they affect the median & mean. Generally, the more outliers, the more the mean is affected

    Measures of Spread for One Quantitative Variable
  • It is important to take into account how far spread out the data is. This is done by calculating the standard deviation & z-score of the dataset

  • The summary statistics is a table that lays out the crucial statistics of a corresponding dataset. This includes the size, mean, standard deviation, minimum, first quartile, median, third quartile, and maximum

  • The deviation of a dataset refers to how far it deviates from the norm or average. It is calculated by taking the difference of an element of interest & the mean:
      • deviation = element - mean

  • Standard deviation (std) is the statistic that measures the variability of the dataset as a whole. In other words, it is a measurement of how spread out the data is and how far each element deviates from the norm. This is commonly used (along with the mean) when every element in a dataset is relevant and must be taken into account. Standard deviation is the square root of the variance. It is calculated by taking the sum of squared deviations of each datset, dividing by n - 1 (where n = size) and square rooting the result:
      • std = sqrt(Σ((dataset[] - mean)2) / (size - 1))
      std = sqrt(Σk = 1n(xk - mean)2 / (n - 1))

      std is denoted as s in a sample and σ in a population

      The 95% Rule
    • To get an idea of the magnitude of standard deviation, in a symmetric, bell shaped dataset, 95% of the data lies within 2 standard deviations from the mean. In other words, this data falls within the interval of [mean ± 2(std)]

  • The z-score is a measure of how many standards deviations away an element lies from the mean. It is calculated by taking dividing the deviation by the standard deviation:
      • z-score = deviation / std
      z-score = (element - mean) / sqrt(Σ((dataset[] - mean)2) / (size - 1))

      Only about 5% of the data will have a z-score higher than 2 or lower than -2

  • Percentiles are the percentages of participants in a sample with a value less than or equal to a value of interest. For instance, a score in the 80th precentile means 80% of participants scored equal to or lower than the observed participant

  • 5-Number Summary
      • A tuple containing the minimum, first quartile (25th percentile), median, third quartile (75th percentile), and maximum in that order. This gives a summary of the overall distribution of a dataset

  • The range is the difference between the maximum & minimum values. This gives summary of the magnitude of how distributed the data is
  • The interquartile range (IQR) is the difference between the third quartile (Q3) & first quartile (Q1). This gives an approximate of the spread of most of the data. This is commonly used (along with the median) when there are no outliers

    Relationships between Categorical & Quantitative Variables
  • To mathematically determine whether an element is an outlier, it is compared to the value of (Q1 - 1.5 IQR) and (Q3 + 1.5 IQR). If the value of interest is less than the first comparison or last comparision, it is, indeed, an outlier:
      • if(element < (Q1 - (1.5 * IQR)) || element > (Q3 + (1.5 * IQR))) {outliers[n] = element}

  • Side-by-side graphs are optimal for comparing categorical & quantitative variables. An exaxmple of this is a study that measures the weekly average amount of time watching television according to a person's gender

  • Taking the difference of means within numerical data of 2 different categories gives insight to the relationship between the 2 categorical variables

  • Summary statistics, as mentioned previously, provides strong data of the distribution, spread and centre points of multiple categorical variables with associated numerical data
    Correlation between 2 Quantitative Variables
  • Scatterplots are an ideal tool to use for visualising the relationship between the values of 2 numerical variables

  • Correlation is a measure of the magnitude & direction (vector properties) of the linear association between 2 numerical variables
      • Correlation in a sample is denoted as r, whilst in a population, is denoted by ρ
      It is calculated as follows:
    • correlation = Σ(zScore1 * zScore2) / (size - 1)
    • correlation = Σ(((dataset1[] - mean1) / std1) * ((dataset2[] - mean2) / std2)) / (size - 1)
    • r = Σk = 1n(((xk - meanx) / sx) * ((yk - meany) / sy)) / (n - 1)
      Values of correlation range from -1 to 1 where a higher magnitude corresponds to a more linear association between 2 numerical variables. Negative values imply negative correlation and positive values imply positive correlation
      Strong correlations do not imply a cause & effect relationship
      Outliers may greatly affect the outcome of correlations

    Linear Regression
  • Linear Regression is the method used for approximating a line of best fit within a set of datapoints. It is commonly used to predict a value given the result of the slope of the regression line and an explanatory/independent variable
      • The regression line takes the form of a line, having the equation y = bx + c, where y = response/dependent variable, x = explanatory/independent variable, b = slope & c = constant
      x & y (the 2 variables) cannot be used interchangedly
      A predicted response is sometimes denoted by "y with a hat" or simply Y
      The equation of a regression line is also called the prediction equation
      Formally, the slope of the line represents the change in y divided by the change in x (differential of y). In other words, it represents the change in the predicted value for a specific change in independent variable (usually one). The intercept is the value of y when x == 0 (y-intercept)
      For a sample, the slope of the regression line is denoted by b. For a population, it is denoted as β
      Predicted data based on the prediction equation should only be used for valid values of an explanatory variable. Choosing a value for the independent variable that is too far from its normal range is referring to as extrapolating the data too far
      Data having a curved (exponential) trend cannot be regressed linearly
      Outliers are influential points that can greatly affect linear regression; it is good practice to remove them before regressing

  • The residual is the difference between the predicted value and the actual value of the response variable for a given explanatory variable. It is also referred to as the vertical deviation/distance from the regression line:
      • residual = actual - predicted
      residual = y - Y
      Deviations above the line have a positive residual & deviations below the line have a negative residual

  • The line of best fit corresponds to the regression line with predicted values that have the smallest squared residuals:
      • line = min(residuals[]2)
      line = min((actual[] - predicted[])2)

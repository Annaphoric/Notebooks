Chapter 6a: Inference for Proportions

    Distribution of a Proportion
  • The Central Limit Theorem (CLT) explains how the statistics of a sample follows a normal distribution as long as the sample size is large

  • Sample proportions p are centered around population proportions p

  • The standard deviation of a normal distribution in a sample based on population proportion is calculated as its standard error SE:
      • SE = sqrt((p * (1 - p)) / n), where n = sample size & p = population proportion
  • Since the sample size is inversely proportional to the standard error or variability, it is observed that increasing the sample size, decreases the standard error

  • The most optimal bell-shaped curve of the proportions is around 0.5. As proportions approach 0 or 1, the bell-shaped curve is almost indistinguishable
  • Larger sample sizes also favour optimal normal distribution (as a bell-shaped curve)
  • Normal distribution can be appropriately assumed if n * p ≥ 10 and n * (1 - p) ≥ 10:
      • if((n * p ≥ 10) && (n * (1 - p) ≥ 10)) {normalDist = true}
    Example 1:
  • The proportion of US adults who are college graduates is 0.275 (27.5%). What is the standard error based on a sample size of 200 adults?

      • Using the formula for standard error for proportions:
    • SE = sqrt((0.275 * (1 - 0.275)) / 200)
    • SE = 0.032
    Confidence Interval for a Proportion
  • The confidence interval CI for a sample statistic/proportion p is the range in which standard error SE exists based on an appropriate percentile of a standard normal distribution z*, a special type of z-score. The standard normal z* for 95% confidence is 1.96 and for 99% confidence is 2.576. The formula for the confidence interval is:
      • CI = [p ± z* * SE]
      CI = [p ± z* * sqrt((p * (1 - p)) / n)]
      The product of z* & SE is also referred to as the margin of error ME. Margin of error increases as the proportion is closer to 0.5 and the sample size decreases. In practise, the less ME, the better:
    • ME = z* * SE

  • The sample size n can be determined based on a known margin of error ME & predicted sample proportion p:
      • n = (p * (1 - p)) / (ME / z*)2
      n = p * (1 - p) * (z* / ME)2
      If the sample proportion is unable to be accurately predicted, then it is simply set to 0.5. Using a standard normal z* of 2 (around 95% confidence) & p = 0.5, the formula for sample size can be simplified to:
    • n = 1 / (ME)2
    Example 2:
  • In a sample size of n = 100 mixed nuts, 52 are peanuts. a) Is this sample size large enough to apply the Central Limit Theorem?; b) Find a 95% confidence interval for the proportion of peanuts;

      • a)
    • The application of the CLT can be determined by using the test:
        • n * p ≥ 10 && n * (1 - p) ≥ 10
        100 * 0.52 ≥ 10 && 100 * 0.48 ≥ 10
        52 ≥ 10 && 48 ≥ 10
        true && true (true)
        ∴ the CLT can be applied

      b)
    • For a 95% CI, z* = 1.96. This CI can be found using the formula:
        • CI = [p ± z* * SE]
        CI = [0.52 ± 1.96 * sqrt(0.52 * (1 - 0.52)) / 100)]
        CI = [0.52 ± 0.098]
        CI = [0.422, 0.618]int
        This means we are 95% confident that 42.2% to 61.8% of all mixed nuts are peanuts
    Hypothesis Test for a Proportion
  • A hypothesis test HT can be used to determine how well the sample proportion fits within the normal distribution of data when the null hypothesis H0 is true. The result of a hypothesis test can be quantified in terms of the p-value z. Assuming the sample size is appropriate for normal standard distribution, this value is calculated by subtracting the sample statistic (proportion) p by the hypothesised null parameter (value/proportion) p0, then dividing by the standard error SE:
      • z = (p - p0) / SE
      z = (p - p0) / sqrt((p0 * (1 - p0)) / n)
      z = (p - p0) * sqrt(n / (p0 * (1 - p0)))

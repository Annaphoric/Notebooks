Lecture 5.3: Cayley-Hamilton's Theorum

    Example 1:
  • Given a matrix of size 3 ⨯ 3 with eigenvalues λ = {-4, 2, 5}, is this matrix diagonalisable?

      • Since the multiplicity of the set of eigenvalues is one for each eigenvalue since none are repeated, the dimensions of the eigenspace made from each eigenvalue is one. In order for the matrix to be diagonalisable, the dimensions of each eigenspace must add up to the size of the matrix, 3:
    • dim(B(λ = -4)) + dim(B(λ = 2)) + dim(B(λ = 5)) == 3
    • 1 + 1 + 1 == 3 (true)
    • Since the sum of the dimensions is 3, the given matrix is diagonalisable
    Example 2:
  • Given a 5 ⨯ 5 matrix with eigenvalues λ = {-4, 6} with respective multiplicities of 3 & 2 and the number of dimensions of the eigenspace for λ = 6 is 2, under what conditions is the matrix diagonalisable and not diagonalisable?

      • The polynomial characteristic p(λ) for this matrix is:
    • (-1)5(λ + 4)3(λ - 6)2 = 0

      The dimensions of the eigenspace for the λ = -4 is based on its multiplicity (but it is not always true that they are, essentially, the same):
    • dim(B(λ = -4)) ≤ 3
    • If the number of dimensions of this eigenspace is 3, the matrix is diagonalisable (since the sum of the dimensions of both eigenspaces is 5). If it is less than 3, the matrix is not diagonalisable (since the sum of both eigenspaces is not 5)
    Cayley-Hamilton Theorum
  • For a matrix A of size n ⨯ n having with eigenvalues λ = {λ1, λ2...λn}, the expression cnAn + cn - 1An - 1 +...c0I (p(A)) is always equal to a zero matrix of the same size. The values of c are the coeficients of the λ within the characteristic polynomial:
      • cnAn + cn - 1An - 1 +...c0 = 0
      p(A) = 0

  • If the main diagonal of a matrix of size n ⨯ n is made up of all zeros, the lower triangle is made up of all zeros, but the upper triangle is filled with non-zero numbers, the matrix is said to be nilpotent. A nilpotent matrix raised to the n-th power is equal to the zero matrix. This is proved by Cayley-Hamilton's Theorum
    Example 3:
      Prove the Cayley-Hamilton Theorum using matrix A = [[7 6] [-1 2]].

    1. Computing p(λ):
        2. det([[(7 - λ) 6] [-1 (2 - λ)]]) = 0
        (λ2 - 9λ + 14) + 6 = 0
        λ2 - 9λ + 20 = 0

    2. Showing that p(A) == 0 is true:
        3. A2 = [[7 6] [-1 2]] [[7 6] [-1 2]]
        A2 = [[49 + (-6)] [-7 + (-2)]] & [[42 + 12] [-6 + 4]]
        A2 = [[43] [-9]] & [[54] [-2]]
        A2 = [[43 54] [-9 -2]]

        -9A = -9 [[7 6] [-1 2]]
        -9A = [[-63 -54] [9 -18]]

        p(A) = A2 - 9A + 20I == 0
        [[43 54] [-9 -2]] + [[-63 -54] [9 -18]] + [[20 0] [0 20]] == 0
        [[(43 - 63 + 20) (54 - 54)] [(-9 + 9)] [(-2 - 18 + 20)]] == 0
        [[0 0] [0 0]] == 0 (true)
        Since p(A) == 0 is true, the Cayley-Hamilton Theorum is obeyed

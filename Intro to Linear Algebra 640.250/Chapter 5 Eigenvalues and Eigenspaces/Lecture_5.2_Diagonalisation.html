Lecture 5.2: Diagonalisation

    Diagonalisation
  • For a matrix A, A is diagonalisable if there is an invertable matrix P so that A = P * D * P-1, where D = diagonal square matrix containing all eigenvalues of matrix A. The matrix P is constructed using the bases for eigenspaces by combining the eigenvectors in each set. P must be invertable & linearly independent
      • The diagonal matrix D must correspond to the positions (row/columns) of the eigenvectors placed in matrix P

  • Diagonalisation is only possible if the dimensions of each eigenspace is equal to the size of the matrix's rows/columns

  • When computing determinants, it is recommended to use a combination of row/column reduction & co-factor expansion

  • The multiplicity of an eigenvalue is the # of times it exists in the set of eigenvalues. It corresponds to the dimensions of the eigenspace/basis B formed by the corresponding eigenvalue.
      • The dimensions of the eigenspace must always be 1 or greater but less than or equal to the number of its multiplicity:
    • 1 ≤ dim(Bλ) ≤ mult(λ)

  • For a matrix A of size n, p(λ) is always a polynomial degree of n where the constant is equal to det(A)

  • The trace of A is the sum of the main diagonal. The constant of p(λ) is equal to trace(A)

  • If the multiplicity of eigenvalues for a matrix is one for each eigenvalue, the matrix is diagonalisable

  • The standard vectors are always eigenvectors for an identity matrix of the same size

  • The multiplicity of an eigenvalue for an identity matrix of size n is always n
    Example 1:
      Given matrix A = [[2 0 -1] [1 3 -1] [2 0 5]], determine if A is diagonalisable.
    1. Solving for the eigenvalues λ using (A - λI):
        2. [[(2 - λ) 0 -1] [1 (3 - λ) -1] [2 0 (5 - λ)]]
        [[(1 - λ) 0 -1] [0 (3 - λ) -1] [(7 - λ) 0 (5 - λ)]] (C1 = C1 + C3)
        Using 2nd column for co-factor expansion:
      • (3 - λ) * det([[(1 - λ) -1] [(7 - λ) (5 - λ)]]) * (-1)4 = 0
      • (3 - λ) * ((1 - λ)(5 - λ) - (-1)(7 - λ)) = 0
      • (3 - λ) * ((λ2 - 6λ + 5) + (7 - λ)) = 0
      • (3 - λ) * (λ2 - 7λ + 12) = 0
      • (3 - λ)(λ - 4)(λ - 3) = 0
      • (λ - 3)2(λ - 4)
      • λ = {3, 3, 4} (Since 3 appears twice, its multiplicity is 2)

    2. Finding the bases for the eigenvalues:
        3. For λ = 3:
      • [[-1 0 -1] [1 0 -1] [2 0 2]]
      • [[-1 0 -1] [0 0 0] [0 0 0]] (R2 = R2 + R1; R3 = R3 + 2 R1)
      • {-x - z = 0}
      • x(x) = x; y(x) = y; z(x) = -x
      • solutions = x [[1] [0] [-1]] + y [[0] [1] [0]]; B(λ = 3) = {[[1] [0] [1]], [[0] [1] [0]]}
      • This eigenspace has 2 dimensions

        For λ = 4:
      • [[-2 0 -1] [1 -1 -1] [2 0 1]]
      • [[-2 0 -1] [1 -1 -1] [0 0 0]] (R3 = R3 + R1)
      • [[1 -1 -1] [-2 0 -1] [0 0 0]] (R1, R2 = R2, R1)
      • {x - y - z = 0; -2x - z = 0}
      • y = x - z; z = -2x
      • x(x) = x; y(x) = 3x; z(x) = -2x
      • solutions = x [[1] [3] [-2]]; B(λ = 4) = {[[1] [3] [-2]]
      • This eigenspace has 1 dimension. Therefore, A is diagonalisable because sum of the dimensions of both eigenspaces is 3 which is the size of the matrix's rows/columns
      • Combining all the eigenvectors together gives matrix P = [[1 0 1] [0 1 -2] [1 0 -2]] and diagonal-matrix D = [[3 0 0] [0 3 0] [0 0 4]]. So:
          • A == PDP-1 is true
    Example 2:
      Given matrix B = [[-7 -1 -2] [0 c 0]] [-10 3 3] and p(λ) = -(λ - c)(λ + 3)(λ + 2), find values of c so that B is not diagonalisable.

    1. matrix B may not be diagonalisable if there are 2 or more of the same eigenvalues (multiplicity). So, for A to not be diagonalisable, c would have to be -3 or -2
        2. For c = -3:
      • -(λ + 3)2(λ + 2) = 0
      • λ = {-3, -3 -2}

      • [[(-7 - λ) -1 -2] [0 (-3 - λ) 0] [-10 3 (3 - λ)]] (eigenmatrix)

      • For λ = -3:
          • [[-4 -1 -2] [0 0 0] [-10 3 6]]
          [[-4 -1 -2] [-10 3 6] [0 0 0]] (R2, R3 = R3, R2)
          [[-4 -1 -2] [0 11/2 11] [0 0 0]] (R2 = R2 - 5 R1 / 2)
          dim(S(λ = -3)) = 1 (since the nullity is 1)

      • For λ = -2:
          • dim(S(λ = -2)) = 1 (since the multiplicity of (λ + 2) = 0 is 1)

      • dim(S(λ = -3)) + dim(S(λ = -2)) == 3 (3 is the size of the matrix)
      • 1 + 1 == 3 (false)
      • Since the sum of the dimensions of both eigenspaces is not 3, matrix B is not diagonalisable when c = -3

        For c = -2:
      • -(λ + 3)(λ + 2)2
      • λ = {-3, -2, -2}

      • [[(-7 - λ) -1 -2] [0 (-2 - λ) 0] [-10 3 (3 - λ)]]

      • For λ = -3:
          • dim(S(λ = -3)) = 1 (multiplicity = 1)

      • For λ = -2:
          • [[-5 -1 -2] [0 0 0] [-10 3 5]]
          [[-5 -1 -2] [-10 3 5] [0 0 0]] (R2, R3 = R3, R2)
          [[-5 -1 -2] [0 5 9] [0 0 0]] (R2 = R2 - 2 R1)
          dim(S(λ = -2)) = 1 (nullity = 1)

      • dim(S(λ = -3)) + dim(S(λ = -2)) == 3 (size of matrix)
      • 1 + 1 == 3 (false)
      • Since the sum of the dimensions of both eigenspaces is not 3, matrix B is not diagonalisable when c = -3

    2. In conclusion, matrix B is not diagonalisable when c = {-3, -2}
    Example 3:
      Given matrix A = [[6 -5 -4] [5 -3 -5] [4 -5 -2]] and p(λ) = -(λ + 3)(λ - 2)2, solve for the eigenspaces for each eigenvalue and determine if A is diagonalisable.

    1. The eigenvalues are λ = {-3, 2} and the eigenmatrix is [[(6 - λ) -5 -4] [5 (-3 - λ) -5] [4 -5 (-2 - λ)]]
        2. For λ = -3:
      • [[9 -5 -4] [5 0 -5] [4 -5 1]]
      • [[9 -5 -4] [1 0 -1] [4 -5 1]] (R2 = R2 / 5)
      • [[1 0 -1] [9 -5 -4] [4 -5 1]] (R1, R2 = R2, R1)
      • [[1 0 -1] [0 -5 5] [0 -5 5]] (R2 = R2 - 9 R1; R3 = R3 - 4 R1)
      • [[1 0 -1] [0 -5 5] [0 0 0]] (R3 = R3 - R2)
      • nullity = 1

      • {x - z = 0; -5y + 5z = 0}
      • x(z) = z; y(z) = z; z(z) = z
      • B(λ = -3) = {[[1] [1] [1]]}

        For λ = 2:
      • [[4 -5 -4] [5 -5 -5] [4 -5 -4]]
      • [[4 -5 -4] [1 -1 -1] [0 0 0]] (R2 = R2 / 5; R3 = R3 - R2)
      • [[1 -1 -1] [4 -5 -4] [0 0 0]] (R1, R2 = R2, R1)
      • [[1 -1 -1] [0 -1 0] [0 0 0]] (R2 = R2 - 4 R1)
      • nullity = 1

      • {x - y - z = 0; -y = 0}
      • x(z) = z; y(z) = 0; z(z) = z
      • B(λ = 2) = {[[1] [0] [1]]}

      • Since the nullity of both eigenspaces are 1. The sum of the dimensions of these eigenspaces is 2. Since this value is less than the size of the matrix, 3, matrix A is not diagonalisable


    2. Computing p(λ):
        3. det([[(6 - λ) -5 -4] [5 (-3 - λ) -5] [4 -5 (-2 - λ)]]) = 0
        det([[(6 - λ) -1 -4] [5 (2 - λ) -5] [4 (-3 + λ) (-2 - λ)]]) = 0 (C2 = C2 - C3)
        det([[(6 - λ) -1 0] [5 (2 - λ) (-13 + 4λ)] [4 (-3 + λ) (10 - 5λ)]]) = 0 (C3 = C3 - 4 C2)
        det([[(6 - λ) -1 0] [1 (5 - 2λ) (-23 + 9λ)] [4 (-3 + λ) (10 - 5λ)]]) = 0 (R2 = R2 - R3)

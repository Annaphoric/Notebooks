Lecture 5.1: Eigenvalues

    Eigenvalues
  • Eigenvalues are only applied to diagonal square matrices. These matrices have a size n ⨯ n, non-zero values on the main diagonal & all zeros outside the diagonal. The notation for eigenvalues is λ. They are also called characteristic values

  • A matrix λ is an eigenvalue for matrix A if and only if there is a nonzero vector v (known as an eigenvector) so that the product of A * v is equal to the product of λ * v:
      • if(A * v == λ * v && v != 0) {λ = eigenvalue}
      (Av = λv) is known as the eigenvalue equation

  • The eigenvalue equation can also be written in terms of a product of λ & an identity matrix I:
      • Av = λIv
      (A - λI)v = 0 (homogenous equation)

  • The set of eigenvectors {v} always form subspaces known as eigenspaces for λ. Eigenspaces, like subspaces, are closed under addition, where any vector can be added to a vector in the set and still have the same span, and closed in scalar multiplication, where any scalar can be multipled to a vector in the set and still have the same span
      • The eigenspace for λ is the nullspace of (A - λI):
    • eigen(λ) = null(A - λI)
  • let B = A - λI; If Bv = 0, then B has no inverse and the determinant of B is zero (det(B) == 0) because the nullity of B is greater than zero (null(B) > 0) && the rank of B is less than its size (rank(B) < n) && the columns/rows are dependent
      • If the determinant of B is zero, then the polynomial of λ is also zero. The polynomial p of λ is also called the characteristic polynomial:
    • det(A - λI) = det(B) = p(λ) = 0
      The roots of p(λ) are the eigenvalues for matrix A

  • let D = [[λ1 0...0] [0 λ2...0]...[0...0 λn]] (an infinite square diagonal matrix). The value of D raised to the m would be:
      • Dm = [[λ1m 0...0] [0 λ2m...0]...[0...0 λnm]]

  • If a matrix is able to be represented as a diagonal matrix, it is diagonalisable

  • Eigenvectors formed from different eigenvalues are linearly independent
    Example 1:
      Given matrix A = [[-9 -7] [14 12]], determine the eigenvalues for A, then find the basis for the nullspace of A.
    1. Solving for λ using the zero determinant of (A - λI):
        2. det(A - λI) = ((-9 - λ) * (12 - λ)) - (-7 * 14) == 0
        (λ2 - 12λ + 9λ - 108) + 98 = 0
        λ2 - 3λ - 10 = 0 (This is the polynomial of λ or p(λ))
        (λ - 5)(λ + 2) = 0
        λ = {-2, 5}

    2. Using the values of λ to find the basis for null(A):
        3. For λ = 5:
      • let B = A - 5I;
      • B = [[(-9 - 5) -7] [14 (12 - 5)]]
      • B = [[-14 -7] [14 7]]

      • [[-14 -7] [0 0]] (R2 = R2 + R1; rank = 1; null = 1)
      • {-14x - 7y = 0} (Written as a system of equations)
      • 7y = -14x
      • x(x) = x, y(x) = -2x (The variables written as a function in terms of x)
      • The solutions/nullspace to the system of equations is the vector w = [[x] [-2x]] = x [[1] [-2]] for all x (spanning a line). The corresponding eigenvector is simply v = [[1] [-2]]. The basis B for the eigenspace is the set containing v. This is written as:
          • B(λ = 5) = {[[1] [-2]]}
      • This shows that [[-9 -7] [14 12]] [[1] [-2]] = 5 [[1] [-2]]

        For λ = -2:
      • let C = A + 2I;
      • C = [[(-9 + 2) -7] [14 (12 + 2)]]
      • C = [[-7 -7] [14 14]]

      • [[-7 -7] [0 0]] (R2 = R2 + 2 R1)
      • {-(7x + 7y) = 0}
      • x(x) = x; y(x) = -x
      • solutions = x [[1] [-1]]; eigenvector v = [[1] [-1]]; eigenspace-basis B(λ = -2) = {[[1] [-1]]}

Lecture 1.3, 1.4: Gaussian Elimination

    Gaussian Elimination
  • An organised method for solving systems of equations

  • A row operation is a modification of an equation without changing its true values. These include:
      • Changing the order of equations
      Multiplying an equation by a nonzero scalar
      Adding equations together

      The following steps are used:
    1. Convert the system of equations into the product of a coefficient matrix & vector set equal to another vector
    2. Perform row operations to eliminate the same variable on all equations except one equation
    3. Use back substitution on the equations using found variable
    4. Repeat steps 2 & 3 until all variables are solved, if possible
    Example 1:
      Solve the following system of equations, if possible: {x - y - 3z + w = 0, -2x + y + 5z = -4, 4x - 2y - 10z + w = 5}.

    1. Rewriting the system of equations as a coefficient matrix and vector:
        2. [[1, -1, -3, 1], [-2, 1, 5, 0], [4, -2, -10, 1]] ⋅ [[x], [y], [z], [w]] = [[0], [-4], [5]]

    2. Row operations are now performed until the first variable is solved (x in this case):
        3. Note: Commas are removed for more efficient typing
        2 main row operations are performed here: The second row is replaced with the sum of the first equation multipled by 2 and the second equation; The third row is replaced with the sum of the first equation multiplied by -4 and the third equation
        [[1 -1 -3 1] [0 -1 -1 2] [0 2 2 -3]] ⋅ [[x] [y] [z] [w]] = [[0] [-4] [5]]
        Here, the element in the first row & column (1 in this case) is known as the pivot, which always contains zeros below it

    3. Row operations are performed until the second variable is solved (y in this case):
        4. [[1 -1 -3 1] [0 -1 -1 2] [0 0 0 1]] ⋅ [[x] [y] [z] [w]] = [[0] [-4] [-3]]
        The element in the second row & column (-1 in this case) becomes another pivot
        Since no more pivots can be defined, the entire equation is now said to be in a Row Echelon form

    4. Since there is a standard row (a row containing all zeros except for a one) in the coefficient matrix, the matrices can be converted into a system of equations and back substitution can be used to solve for the variables:
        5. {x - y - 3z + w = 0; -y - z + 2w = -4; w = -3}
        {x - y - 3z + (-3) = 0; -y - z + 2(-3) = -4} (substituting w = -3 into the equations)
        {x - y - 3z = 3; -y - z = 2}
        {x - y - 3z = 3; y = -(z + 2)}
        {x - (-(z + 2)) - 3z = 3} (substituting y = -(z + 2) into the equation)
        {x + z + 2 - 3z = 3}
        {x - 2z = 1}
        x = 2z + 1

        There are infinite solutions in the form of 3 equations that were found: {x = 2z + 1; y = -(z + 2); w = -3}. Since there is the same independent variable (or parameter) on 2 equations (z is the parameter of x and y), there is one degree of freedom. This means the solutions form a one dimensional line in 4-dimensional space (each dimension for one variable), having the point (1 + 2z, -z - 2, z, -3) as the corresponding solutions to the system of equations
    Example 2:
      Solve the following system of equations: {x + 3y + z + w = -1; -2x - 6y - z = 5; x + 3y + 2z + 3w = 2}.

    1. Converting the system of equations in a matrix form:
        2. [[1 3 1 1] [-2 -6 -1 0] [1 3 2 3]] ⋅ [[x] [y] [z] [w]] = [[-1] [5] [2]]

    2. Solving for the first pivot:
        3. The second row is replaced with the sum of the first equation multiplied by 2 and the second equation
        [[1 3 1 1] [0 0 1 2] [1 3 2 3]] ⋅ [[x] [y] [z] [w]] = [[-1] [3] [2]]

        The third row is replaced with the sum of the first equation multipled by -1
        [[1 3 1 1] [0 0 1 2] [0 0 1 2]] ⋅ [[x] [y] [z] [w]] = [[-1] [3] [3]]

    3. Since the second and third row are equivalent, the equations can be converted back into a systematic form and back substitution is used:
        4. {x + 3y + z + w = -1; z + 2w = 3}
        {x + 3y + z + w = -1; z = 3 - 2w}
        x + 3y + (3 - 2w) + w = -1 (substituting z = 3 - 2w)
        x + 3y - w = -4
        x = w - 3y - 4

        There are infinitely many solutions in the form of 2 algebraic equations: {x = w - 3y - 4; z = 3 - 2w}. There are 2 independent variables (y & z), so there are 2 degrees of freedom. Geometrically, the equations form a 2-dimensional plane in 4-dimensional space
    Example 3:
      Find the value of r in the following system of equations: {-x + ry = 2; rx - 9y = 6} to solve for a consistent solution.

    1. Converting the systematic format into a matrix form:
        2. [[-1 r] [r -9]] ⋅ [[x] [y]] = [[2] [6]]

    2. Solving for the first pivot:
        3. Replace the second equation by the sum of the first equation multiplied by r and the second equation
        [[-1 r] [0 (r2 - 9)]] ⋅ [[x] [y]] = [[2] [2r + 6]]

    3. Converting from matrix form back into systematic form and performing back substitution:
        4. {-x + ry = 2; y(r2 - 9) = 2r + 6}
        {x = ry - 2; y = (2r + 6)/(r2 - 9)}
        x = r((2r + 6)/(r2 - 9)) - 2 (Substituting y = (2r + 6)/(r2 - 9)

        There are 2 possible responses to this question:
      • The equations are consistent if r = -3. So, there are infinite solutions when r = -3, having the algebraic form: x = -(3y + 2). Since there is one independent variable, there is one degree of freedom. Geometrically, the solution is a line on a 2D plane

      • The equations are also consistent if r ≠ 3. So, there are infinite solutions when r ≠ 3, having the algebraic form: {x = r((2r + 6)/(r2 - 9)) - 2; y = (2r + 6)/(r2 - 9)}. This also means there is no solution at r = 3. Since there is one independent variable, there is one degree of freedom. Geometrically, the solution is a line in 3D space

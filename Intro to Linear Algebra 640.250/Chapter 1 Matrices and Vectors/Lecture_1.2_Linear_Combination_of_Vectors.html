Lecture 1.2: Linear Combination of Vectors

    Linear Combination of Vectors
  • Given 2 vectors v = [[a1], [a2]...[an]] & w = [[b1], [b2]...[bn]], the linear combination of these vectors (LCAV) results in the plane u = c1v + c2w. u can also be represented as u = [[c1a1 + c2b1], [c1a2 + c2b2] + ...]
      • The c scalars refer to any constant, the a elements refer to the entries of v and the b elements refer to the entries of w. A vector multiplied by any and every constant forms a line in the projected along the vector
      The resulting vector is a plane that includes the first 2 vectors

  • Rn refers to n-dimensional space. Geometrically, it is a vector containing all possible coordinates or components given an n-dimensional graph
    Example 1:
      Solve the following system of equations using the LCAV, if possible: {6x - 7y + 5z = 4, 2y - 3z + 4w = 5, x - 6z + 9w = 10}

    1. Rewriting the system of equations in the form of matrices (the variables act like constants):
        2. x [[6], [0], [1]] + y [[7], [2], [0]] + z [[5], [-3], [-6]] + w [[0], [4], [9]] = [[4], [5], [10]]
        Since the vector on the right side of the equation does not lie on the plane of the vectors on the left side of the equations (using the first 2 vectors for a simple proof), there is no solution to the system of equations
        The vectors used in the answer are called coefficient vectors
    Example 2:
      Is vector A = [[3], [5], [-5]] a linear combination of generating set S = {[[2], [0], [-1]], [[-1], [1], [0]]}?

    1. Rewriting the vectors using LCAV:
        2. x [[2], [0], [-1]] + y [[-1], [1], [0]] = [[3], [5], [-5]]

    2. Forming a system of equations:
        3. {2x - y = 3, y = 5, -x = -5}

    3. Substituting the appropriate variables into the system of equations:
        4. 2(5) - (5) = 3
        5 == 3 (false)
        Since the result is false, the system of equations is inconsistent and has no real solution. Geometrically, vector A lies outside the plane containing the vectors of set S
    Multiplying a Matrix & Vector
  • In order to multiply a matrix and a vector, the matrix's number of columns must be equal to the vector's number of rows. In other words, the matrix must have dimensions n ⨯ m and the vector must have dimensions m ⨯ 1. The dot product of a matrix's row is taken with the vector's column until the final row/column. The result is a matrix having dimensions n ⨯ 1
    Example 3:
      Given the following system of equations: {3a - 5b + 7c + 4d = 3, 2a + b + d = -1, 6a - c + 2d = 1}, rewrite this system as a coefficient matrix and variable matrix:

    1. Rewriting the matrices:
        2. [[3, -5, 7, 4], [2, 1, 0, 1], [6, 0, -1, 2]] ⋅ [[a], [b], [c], [d]] = [[3], [-1], 1]
    Example 4:
      Simplify [[3, -1, 4], [-2, 6, -1]] ⋅ [[0], [1], [0]].

    1. Performing the dot product:
        2. result = [[0 - 1 + 0], [0 + 6 + 0]]
        result = [[-1], [6]]
    Types of Matrices
  • Identity Matrix (In ⨯ n)
      • A square, symmetric matrix consisting of all zeros except for the diagonal which consists of ones

  • Standard vectors in Rn
      • Similar in physics, i or e1 represents the vector [[1, 0, 0...]], j or e2 represents the vector [[0, 1, 0...]], & k or e3 represents the vector [[0, 0, 1...]]
      A standard vector in n-dimensional space is en having the matrix [[0, 0, 0...1]]
      Multiplying a matrix by a standard matrix results in the column of the matrix containing the column of the vector's 1

  • Rotation matrix
      • A 2 ⨯ 2 matrix having the form: [[cosθ, -sinθ], [sinθ, cosθ]] (The matrix may have more dimensions but must be a square matrix)
      Used to multiply with a given 2-dimensional vector (2 ⨯ 1) to change its direction whilst keeping its magnitude constant, resulting in a vector of the same dimensions
    Example 5:
      Rotate vector u = [[-1], [-3]] by 210° (θ = 7π/6).

    1. Multiplying the vector by the corresponding rotation matrix:
        2. v = [[cosθ, -sinθ], [sinθ, cosθ]] ⋅ [[-1], [-3]]
        v = [[cos(7π/6), -sin(7π/6)], [sin(7π/6), cos(7π/6)]] ⋅ [[-1], [-3]]
        v = [[-sqrt(3)/2, 1/2], [-1/2, -sqrt(3)/2]] ⋅ [[-1], [-3]]
        v = [[sqrt(3)/2 - 3/2], [1/2 + 3 sqrt(3)/2]]

    2. To check if the lengths/magnitudes of both vectors are equivalent, the norm or magnitude of each vector is taken:
        3. ||u|| = sqrt(1 + 9)
        ||u|| = sqrt(10)

        ||v|| = sqrt((12 - 6 sqrt(3))/4 + (28 + 6 sqrt(3))/4)
        ||v|| = sqrt((6 - 3 sqrt(3) + 14 + 3 sqrt(3))/2)
        ||v|| = sqrt(20/2)
        ||v|| = sqrt(10)

        ||u|| == ||v|| (true)

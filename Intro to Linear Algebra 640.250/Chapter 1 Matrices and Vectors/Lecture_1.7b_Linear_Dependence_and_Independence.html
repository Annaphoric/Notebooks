Lecture 1.7b: Linear Dependence & Independence

    Example 1:
  • Given set S = {[[1] [2] [0] [-1]], [[1] [-3] [1] [2]], [[1] [2] [-2] [3]]}, is this an independent or dependent set?

      • Rewriting the vectors as a coefficient matrix and converting to Row Echelon Form:
    • [[1 1 1] [2 -3 2] [0 1 -2] [-1 2 3]] [[a] [b] [c]] = 0 (equality to a zero vector indicates a homogenous system)
    • [[1 1 1] [0 -5 0] [0 1 -2] [0 3 4]] [[a] [b] [c]] = 0 (E2 = E2 - 2 E1; E4 = E4 + E1)
    • [[1 1 1] [0 1 -2] [0 -5 0] [0 3 4]] [[a] [b] [c]] = 0 (E2, E3 = E3, E2;)
    • [[1 1 1] [0 1 -2] [0 0 -10] [0 0 10]] [[a] [b] [c]] = 0 (E3 = E3 + 5 E2; E4 = E4 - 3 E2)
    • [[1 1 1] [0 1 -2] [0 0 -10] [0 0 0]] [[a] [b] [c]] = 0 (E4 = E4 + E3)

    • {a + b + c = 0; b - 2c = 0; -10c = 0}
    • [a, b, c] = 0
    • Since each variable is equal to zero, we can conclude than the set is an independent set since the values of the variables can only be zero to form a zero matrix. The rank of the matrix is 3 since there are 3 pivots and the nullity is zero (free independent variables). Nullity also correlates to dependence; if nullity = 0, the vectors are independent and if nullity != 0, the vectors are dependent
    Example 2:
  • Given set S = {[[1] [0] [-1] [-1] [1]], [[-1] [1] [1] [0] [1]], [[-1] [-1] [2] [1] [0]], [[0] [-1] [3] [-2] [7]]}, determine if the set is dependent or independent.

      • Converting to a coefficient matrix and Row Echelon Form:
    • [[1 -1 -1 0] [0 1 -1 -1] [-1 1 2 3] [-1 0 1 -2] [1 1 0 7]]
    • [[1 -1 -1 0] [0 1 -1 -1] [0 0 1 3] [0 -1 0 -2] [0 2 1 7]] (E3 = E3 + E1; E4 = E4 + E1; E5 = E5 - E1)
    • [[1 -1 -1 0] [0 1 -1 -1] [0 0 1 3] [0 0 -1 -3] [0 0 3 9]] (E4 = E4 + E2; E5 = E5 - 2 E1)
    • [[1 -1 -1 0] [0 1 -1 -1] [0 0 1 3] [0 0 0 0] [0 0 1 3]] (E4 = E4 + E3; E5 = E5 / 3)
    • [[1 -1 -1 0] [0 1 -1 -1] [0 0 1 3] [0 0 0 0] [0 0 0 0]] (E5 = E5 - E3; Reduced Echelon Form)
    • This matrix has a rank of 3. Since there is one column available (the final vector), the nullity is one. Based on this info, we can conclude that the set containing the vectors is dependent since the final vector can be removed and still retain its corresponding span. The vectors can be solved in terms of the another, by choosing a value for one scalar multiple
    Example 3:
  • Given set S = {[[1] [2] [-1]], [[2] [1] [-3]], [[-1] [7] [r]]}, find values of r so that the set is classified as dependent.

      • Converting to a coefficient matrix & Row Echelon Form:
    • [[1 2 -1] [2 1 7] [-1 -3 r]]
    • [[1 2 -1] [0 -3 9] [0 -1 (r - 1)]] (E2 = E2 - 2 E1; E3 = E3 + E1)
    • [[1 2 -1] [0 1 -3] [0 -1 (r - 1)]] (E2 = E2 / -3)
    • [[1 2 -1] [0 1 -3] [0 0 (r - 4)]] (E3 = E3 + E2)
    • In order for the matrix to be considered dependent, there must be a column (vector) available to remove (having no pivots). This can occur by setting r = 4 so that the value is zero. This causes the matrix to have 2 pivots (rank = 2) and a nullity of 1 (for one column with no pivots)

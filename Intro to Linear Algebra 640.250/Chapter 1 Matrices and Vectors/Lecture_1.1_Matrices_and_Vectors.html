Lecture 1.1: Matrices & Vectors

    Introduction
  • Linear Algebra is the method used for evaluating linear equations

  • There are 2 aspects of linear algebra that are interrelated with each other:
      • Algebraic
      Geometric

  • Gaussian Elimination is the technique used to enlighten the relationship between the 2 aspects of solving linear equations

  • Linear equations do not include another exponents other than 0 or 1. In other words, quadratic or cubic functions are not linear functions

  • Solutions to linear equations can only have unique solutions, no solutions or infinite solutions
    Example 1:
  • Find the solutions to the system of equations {x + y = 1} & {y = x}.

      • There are various ways to solve for x & y:
    • Geometrically
        • The 2 equations are graphed and the respective coordinates for the point of intersection are the solutions for x & y
        The point of intersection is (1/2, 1/2)

    • Algebraically
        • The solutions for x & y are solved by substituting for each variable:
      • 2y = 1
      • y = 1/2
      • x = 1/2

    • There is a unique solution for this system of equations
    Matrices
  • A matrix is a rectangular array of numbers of rows & columns

  • They can be seen as a type of variable (like a list or array) that holds various numbers
  • For instance, matrix A = [[-3, 0, 4, 5], [2, 6, 1, 2], [0, -1, 1, π]]. In this matrix, there are 3 rows & 4 columns. So, A is a 3⨯4 matrix.
  • To choose a specific cell of the matrix, a certain notation is used. let b = A13. This assigns the integer 4 to variable b which is located in the 1st row & 3rd column. This is seen as b = A[0][2] in most programming languages. In general, Aij is the entry at the i-th row & j-th column

  • Matrices can have m⨯n dimensions

  • Column vectors have n⨯1 dimensions like vector v = [[1], [2], [3], [4]]. This matrix has 4 rows & 1 column

  • Row vectors have 1⨯m dimensions like vector w = [[1, 2, 3, 4, 5]]. This matrix has 5 columns & 1 row

  • Square matrices have m⨯n dimensions where m = n
      • A symmetric matrix is a square matrix that gives the same result after transposing it like matrix C = [[1, -2, 1], [-2, 2, 6], [1, 6, 5]]. In this matrix, 1 is the top left corner of the square, 2 is the centre & 5 is the bottom right corner. These specific elements are known as the principal diagonal or the "mirror"

  • A zero matrix has all components equal to zero regardless of number of dimensions. It is written as 0m⨯n

  • In order for 2 matricess to be equal to equal other, every one of its elements at their corresponding indeces must be equal to each other
    Algebra & Arithmetic using Matrices
  • Assume matrix A & B are both matrices that have m⨯n dimensions. The general matrix notation for A is written as A = [[Aij]]1≤i≤m, 1≤j≤n. The same notation is written for B

  • To add or subtract matrices, they both must have the same number of dimensions. The operation A+ B == (Aij + Bij). The subtraction operation is the same except the addition signs are replaced by minus signs. Addition is cumulative

  • To scale or multiply a vector/matrix v by a scalar multiple, the matrix is given the notation c v where c is a scalar number. This multiplies every component/index/element of a matrix by c. Multiplying a vector by -1 reverses the direction of the vector by taking the opposite of each component

    • To transpose a matrix, the matrix is given the notation At. Transposing means to convert all the rows to columns and vice versa. At has the dimensions n⨯m having the entries Aji. Transposing a matrix twice results in an identical matrix
    Example 2:
  • Given matrix A = [[-1, 2, 3, 4], [6, 2, 1, 0]], B = [[0, -1, 1, 1], [5, 1, 1, 1]], perform the following operations: a) A + B; b) A - B; c) 5 A; d) Bt

      • a)
    • The resulting matrix is [[-1, 1, 4, 5], [11, 3, 2, 1]]

      b)
    • The resulting matrix is [[-1, 3, 2, 3], [1, 1, 0, -1]]

      c)
    • The resulting matrix is [[-5, 10, 15, 20], [30, 10, 5, 0]]

      d)
    • Transposing B gives the resulting matrix [[0, 5], [-1, 1], [1, 1], [1, 1]]

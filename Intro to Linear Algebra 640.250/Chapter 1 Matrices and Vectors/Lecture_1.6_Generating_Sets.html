Lecture 1.6: Generating Sets

    Example 1:
      Find values of r so that vector u = [[2] [r] [-1]] lies within the span/plane of set S = {v, w} where v = [[1] [0] [-1]] and w = [[-1] [3] [2]]?

    1. In order for u to lie within the span/plane of the given set, it must be a legit LCOV of the set:
        2. av + bw = u
        a [[1] [0] [-1]] + b [[-1] [3] [2]] = [[2] [r] [-1]]
        [[a] [0] [-a]] + [[-b] [3b] [2b]] = [[2] [r] [-1]]
        [[a - b] [3b] [2b - a]] = [[2] [r] [-1]]
        [[1 -1] [0 3] [-1 2]] [[a] [b]] = [[2] [r] [-1]]

    2. Converting to Row Echelon Form:
        3. E3 = E3 + E1

        [[1 -1] [0 3] [0 1]] [[a] [b]] = [[2] [r] [1]]

        E3 = E3 - 3 E2

        [[1 -1] [0 3] [0 0]] [[a] [b]] = [[2] [r] [1 - 3r]]

    3. Converting to Reduced Echelon Form:
        4. E2 = E2 / 3

        [[1 -1] [0 1] [0 0]] [[a] [b]] = [[2] [r/3] [1 - 3r]]

    4. Solving system of equations for r:
        5. {a - b = 2; b = r/3; 0 = 1 - 3r}
        r = 1/3; b = 1/9; a = 19/9
    Example 2:
      Does vector p = [[-1] [0] [2] [3]] lie in the span/plane of set S = {[[1] [2] [-1] [1]], [[2] [-1] [1] [0]], [[-1] [2] [0] [3]]}?

    1. vector p must be a linear combination of the vectors in set S:
        2. x [[1] [2] [-1] [1]] + y [[2] [-1] [1] [0]] + z [[-1] [2] [0] [3]] = [[-1] [0] [2] [3]]
        [[1 2 -1] [2 -1 2] [-1 1 0] [1 0 3]] [[x] [y] [z]] = [[-1] [0] [2] [3]]
        Pivots of 1 are preferred

    2. Converting the matrices to Reduced Row Echelon Form:
        3. [[1 2 -1] [0 -5 4] [0 3 -1] [0 -2 4]] [[x] [y] [z]] = [[-1] [2] [1] [4]] (E2 = E2 - 2 E1; E3 = E3 + E1; E4 = E4 - E1)
        [[1 2 -1] [0 -5 4] [0 3 -1] [0 -1 2]] [[x] [y] [z]] = [[-1] [2] [1] [2]] (E4 = E4 / 2)
        [[1 2 -1] [0 -1 2] [0 3 -1] [0 -5 4]] [[x] [y] [z]] = [[-1] [2] [1] [2]] (E2 = E4; E4 = E2)
        [[1 2 -1] [0 -1 2] [0 0 5] [0 0 -6]] [[x] [y] [z]] = [[-1] [2] [7] [-8]] (E3 = E3 + 3 E2; E4 = E4 - 5 E2)
        [[1 2 -1] [0 -1 2] [0 0 5] [0 0 0]] [[x] [y] [z]] = [[-1] [2] [7] [2/5]] (E4 = E4 + 6 E3 / 4)
        Based on the last equation, there is no solution to the system of equations
    Generating Sets
  • A generating set of vectors S = {v1, v2...vn} in n-dimensional space is a set of vectors for which the span of S = Rn (all real numbers in any dimensional space)
    Example 3:
  • Is set S = {[[1] [0]], [[3] [0]]} a generating set?

      • This set is not a generating set because its span is [[a + 3b] [0]] where a & b are any real number. This includes any real number in a 1-dimensional line (x-axis) but not the 2-dimensional plane (xy-plane) in which the vectors lie in
    Example 4:
      Is the span of set S = {[[-1] [2] [1]], [[-1] [1] [3]], [[1] [-3] [1]]} equal to R3?

    1. If the system of equations is consistent or if the rank of S is equal to 3, the set is equal to R3. Converting to Row Echelon Form:
        2. [[-1 -1 1] [2 1 -3] [1 3 1]]
        [[-1 -1 1] [0 -1 -1] [0 -2 2]] (E2 = E2 + 2 E1; E3 = E3 + E1)
        [[-1 -1 1] [0 -1 -1] [0 0 4]] (E3 = E3 - 2 E2)
        Since there are 3 pivots, the rank of S is 3, which is equal to the number of dimensions where the vectors lie. This means the span of S is all real numbers in 3D

Lecture 2.7: Matrix Functions

    Matrix Functions
  • Given a matrix A of size m тип n, and a vector X of size n тип 1, the product of the matrix & vector will be vector Y of size m тип 1. The values of the entries of Y can be expressed as a function of X

  • To solve for the identity of matrix A given a matrix function, n-amount of standard vectors are set as an arguments for the function and each resulting vector is combined to form a matrix

  • The domain of the matrix function is all real numbers in n-dimensions and its range is all real numbers in m-dimensions

  • Matrix functions are also called linear transformations
    Example 1:
  • Given matrix-function A = f(x, y) = [[x - y] [2x + 3y] [4x - 5y]], What is the corresponding matrix that gives the identity of the function?

      • Matrix A can be solved by plugging in the 2 standard vectors into the function:
    • A = f([[1]] [0]) & f([[0] [1]])
    • A = [[1 - 0] [2 + 0] [4 - 0]] & [[0 - 1] [0 + 3] [0 - 5]]
    • A = [[1] [2] [4]] & [[-1] [3] [-5]]
    • A = [[1 -1] [2 3] [4 -5]]
    Example 2:
  • Given linear-transformation B = T([[x] [y] [z] [w]]) = [[2x - y + 3w] [-x + 2w] [3y - z]], find the identity of B.
      • Using the 4 standard vectors as arguments for T:
    • A = f([[1] [0] [0] [0]]) & f([[0] [1] [0] [0]]) & f([[0] [0] [1] [0]]) & f([[0] [0] [0] [1]])
    • A = [[2] [-1] [0]]] & [[-1] [0] [3]] & [[0] [0] [-1]] & [[3] [2] [0]]
    • A = [[2 -1 0 3] [-1 0 0 2] [0 3 -1 0]]

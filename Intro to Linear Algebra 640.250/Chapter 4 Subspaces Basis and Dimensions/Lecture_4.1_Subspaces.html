Lecture 4.1: Subspaces

    Subspaces
  • A subspace is a region in space like a plane or bounded object

  • In order for a set S of vectors v & w to lie within the span of a subspace E, the vectors and their sum (including linear combinations) must lie within the subspace regardless of a scalar multiple:
      • if({v, w, av + bw} in E): S is a subspace
      This works for k-vectors in Rn. The vectors are known as generators for the subspace closed under addition (any vector added to a vector within the set of vectors would still have the same span) & scalar multiplication (any scalar multiplied to a vector would still have the same span)

  • A zero vector lies within every subspace

  • When a coefficient matrix A is multiplied to a vector of variables and set equal to a zero vector (a homogenous system), the vector containing the variables forms a subspace known as a nullspace. Nullspace is also called the kernel
    Example 1:
  • Given set W = {[[r - s + 3t] [2r - t] [-r + 3s + 2t] [-2r + s + t]]} for all real numbers in 4D, is this set a subspace?

      • If W is in the span of every vector it contains, then W is a subspace. Rewriting the set as a sum of vectors:
    • [[r] [2r] [-r] [-2r]] + [[-s] [0] [3s] [s]] + [[3t] [-t] [2t] [t]]
    • r [[1] [2] [-1] [-2]] + s [[-1] [0] [3] [1]] + t [[3] [-1] [2] [1]]
    • Since this set of vectors are all dependent to each other, W is a subspace generated by the 3 vectors
    Example 2:
      Given system of equations S = {-3x + y + z + w = 0; x - y + z = 0; x + y - z + 2w = 0}, is this system a subspace?

    1. Rewriting the system in a matrix form:
        2. [[-3 1 1 1][1 -1 1 0] [1 1 -1 2]] [[x] [y] [z] [w]] = [[0] [0] [0]]
        This system is a homogenous system since the coefficient matrix is set equal to a zero vector. The variables (or vector of variables) that satisfies the homogenous equations form a nullspace
    Example 3:
  • Given matrix A = [[1 -2 -1 0] [0 1 3 -2] [-2 3 -1 2]], is vector v = [[1] [1] [-1] [1]] in null(A) (or in the nullspace of A)?

      • vector b must be a solution (by plugging in the values of the components of the vector into the variables) for the homogenous system set by the matrix A as a coefficient matrix. The matrix can be written as a system of equations or multiplied to A as a coefficient matrix. Multiplying A with v:
    • [[1 -2 -1 0] [0 1 3 -2] [-2 3 -1 2]] â‹… [[1] [1] [-1] [1]] == 0
    • [[0] [-4] [5]] == [[0] [0] [0]] (false)
    • Since this equation is false, vector v does not lie in the span of matrix A is therefore not a solution to the equation. This also means v is not in null(A)
    Example 4:
      Given matrix A = [[1 1 -1 4] [2 1 -3 5] [-2 0 4 -2]], what is the generating set (solutions) for null(A)?

    1. Converting the coefficient matrix A to a Row Echelon Form:
        2. [[1 1 -1 4] [2 1 -3 5] [-2 0 4 -2]]
        [[1 1 -1 4] [0 -1 -1 -3] [0 2 2 6]] (E2 = E2 - 2E1; E3 = E3 + 2 E1)
        [[1 1 -1 4] [0 -1 -1 -3] [0 0 0 0]] (E3 = E3 + 2E2)

    2. Solving for the variables of the system of equations:
        3. {x + y - z + 4w = 0; -y - z - 3w = 0}
        x = z - y - 4w; y = -(z + 3w)
        x = z + (z + 3w) - 4w
        x = 2z - w; y = -(z + 3w); z = z; w = w

        Therefore, the solution to the system of equations is [[2z - w] [-z - 3w] [z] [w]] or (z [[2] [-1] [1] [0]] + w [[-1] [-3] [0] [1]]). Since there are 2 vectors within the set of solutions for null(A), the solution is a 2D plane (subspace/nullspace) in 4D space
    Example 5:
  • Find the nullspace of matrix-function T([[x] [y] [z]]) = [[x + y - z] [0] [2x - z]].

      • Substituting each standard vector into the function:
    • T(i) = [[1] [0] [2]]

    • T(j) = [[1] [0] [0]]

    • T(k) = [[-1] [0] [-1]]

    • This results in the coefficient matrix [[1 1 -1] [0 0 0] [2 0 -1]] and system of equations {x + y - z = 0; 2x - z = 0}

      Solving for the variables:
    • {x + y - z = 0; 2x - z = 0}
    • y = z - x; z = 2x
    • y = 2x - x
    • x = x; y = x; z = 2x

    • Therefore, the vector containing the solutions is x [[1] [1] [2]]. This is a line (subspace/nullspace) in 3D space

Lecture 6.2: Orthonormal Sets

    Pairwise Orthogonality
  • Given a set S of k-amount of nonzero vectors for all real numbers n where k ≤ n, the vectors are said to be pairwise orthogonal if the dot product of any 2 vectors within the set is zero, having an angle of π/2 between the vectors

  • A set containing only standard unit vectors (i, j, k...) is always pairwise orthogonal. This is also known as an orthonormal set. Not all sets containing unit vectors w are orthonormal, only those which contain pairwise orthogonal unit vectors. To convert any set into an orthonormal set, the Gram-Schmidt method is performed:
      • wk = (vk - (vk ⋅ wk - 1) ⋅ wk = 1) / ||vk - (vk ⋅ wk - 1) ⋅ wk = 1||

  • The basis for an orthonormal set can be found by finding linear combinations of the vectors within the set. The constants are found by taking the dot product of the the projected vector & each vector within the basis

  • Pairwise orthogonal vectors are linearly independent

  • Since every subspace has a basis, every basis can be converted to an orthonormal form
    Example 1:
  • Given set S = {[[1] [2] [1]], [[1] [-1] [1]], [[2] [-1] [0]]}, is this an orthonormal set?

      • To determine if the set is orthonormal, the dot product of any 2 vectors in the set is taken:
    • [[1] [2] [1]] ⋅ [[1] [-1] [1]] == 0
    • 1 - 2 + 1 == 0 (true)

    • [[1] [2] [1]] ⋅ [[2] [-1] [0]] == 0
    • 2 - 2 == 0 (true)

    • [[1] [-1] [1]] ⋅ [[2] [-1] [0]] == 0
    • 2 + 1 == 0 (false)
    • Since the dot product of these 2 vectors is not zero, the set is not orthonormal
    Example 2:
      Given set S = {[[1] [0] [-1] [1]], [[2] [1] [-1] [0]], [[2] [-1] [-1] [3]]}, convert this set into an orthonormal set.

    1. The Gram-Schmidt method will have to be performed. First, the first vector is converted to a unit vector u:
        2. u = [[1] [0] [-1] [1]] / ||[[1] [0] [-1] [1]]||
        u = [[1] [0] [-1] [1]] / sqrt(3)

    2. Next, the projection onto the first vector is removed from the second vector to form an orthonormal vector v:
        3. p = [[2] [1] [-1] [0]] - (([[2] [1] [-1] [0]] ⋅ u) u)
        p = [[2] [1] [-1] [0]] - ((2 + 1) [[1] [0] [-1] [1]] / sqrt(9))
        p = [[2] [1] [-1] [0]] - [[1] [0] [-1] [1]]
        p = [[1] [1] [0] [-1]]

        v = p / ||p||
        v = [[1] [1] [0] [-1]] / ||[[1] [1] [0] [-1]]||
        v = [[1] [1] [0] [-1]] / sqrt(3)

    3. Finally, the 2 projections onto the first & second vectors are removed from the third vector to form an orthonormal vector w:
        4. q = [[2] [-1] [-1] [3]] - (([[2] [-1] [-1] [3]] ⋅ u) u) - (([[2] [-1] [-1] [3]] ⋅ v) v)
        q = [[2] [-1] [-1] [3]] - ((2 + 1 + 3) / sqrt(9) [[1] [0] [-1] [1]]) - ((2 - 1 - 3) / sqrt(9) [[1] [1] [0] [-1]])
        q = [[2] [-1] [-1] [3]] - 2 [[1] [0] [-1] [1]] + 2/3 [[1] [1] [0] [-1]]
        q = [[2] [-1] [-1] [3]] - [[2] [0] [-2] [2]] + [[2/3] [2/3] [0] [-2/3]]
        q = [[0] [-1] [1] [1]] + [[2/3] [2/3] [0] [-2/3]]
        q = [[2/3] [-1/3] [1] [1/3]]
        v = q / ||q||
        v = [[2/3] [-1/3] [1] [1/3]] / ||[[2/3] [-1/3] [1] [1/3]]||
        v = [[2/3] [-1/3] [1] [1/3]] / sqrt(4/9 + 1/9 + 1 + 1/9)
        v = [[2/3] [-1/3] [1] [1/3]] / (sqrt(15) / 3)
        v = [[2] [-1] [3] [1]] / sqrt(15)

        Therefore, the orthonormal basis for the given set is B = {u, v, w}
    Example 3:
      Given set W = {[[1] [-2] [0] [-1]], [[2] [1] [1] [0]], [[1] [0] [-2] [1]]} and vector u = [[6] [9] [9] [0]] = a [[1] [-2] [0] [-1]] + b [[2] [1] [1] [0]] + c [[1] [0] [-2] [1]], determine the constants of u.

    1. Solving for the dot product of any 2 vectors:
        2. [[1] [-2] [0] [-1]] ⋅ [[2] [1] [1] [0]] == 0
        2 - 2 == 0 (true)

        [[1] [-2] [0] [-1]] ⋅ [[1] [0] [-2] [1]] == 0
        1 - 1 == 0 (true)

        [[2] [1] [1] [0]] ⋅ [[1] [0] [-2] [1]] == 0
        2 - 2 == 0 (true)

        Since the dot product of any vector is zero, ever vector is orthogonal to each other, but not unit vectors (orthonormal)

    2. As long as the set is orthogonal, the constants are solved by taking the dot product of u & each vector and dividing by the square of the vector's magnitude (if they were orthonormal, then each vector is divided by simply its magnitude instead):
        3. a = (u ⋅ [[1] [-2] [0] [-1]]) / ||[[1] [-2] [0] [-1]]||2
        a = ([[6] [9] [9] [0]] ⋅ [[1] [-2] [0] [-1]]) / sqrt(1 + 4 + 1)2
        a = (6 - 18) / 6
        a = -2

        b = (u ⋅ [[2] [1] [1] [0]]) / ||[[2] [1] [1] [0]]||2
        b = (12 + 9 + 9) / 6
        b = 5

        c = (u ⋅ [[1] [0] [-2] [1]]) / ||[[1] [0] [-2] [1]]||2
        c = (6 - 18) / 6
        c = -2

Section 16.1: Line Integrals of Scalar Functions

  • A line or path integral integrates a function over a bounded curved region
  • Assume vector-function r(t) = {g(t)i + h(t)j + k(t)k} is the parametrisation of function f(x, y, z) for t in [a, b]. This function can be expressed as a composite function f(g(t), h(t), k(t)) or f(r(t)) where x = g(t), y = h(t), z = k(t). To integrate the function along a curve C, the curve is partitioned N-times, forming subarcs. The k-th subarc at point P = (xk, yk, zk) has a length of Δsk. The sum of the areas of every subarc is a similar concept as a Riemann sum SN:
      • SN = ∑k = 1N(f(xk, yk, zk)) Δsk

  • The norm of the partition occurs as N (the number of subarcs) approaches infinity. Like a single integral, this is also the definition of a line integral:
      • limN ➝ ∞(SN) = ∫C(f(x, y, z) ds)

  • The integral can be true only as long as the limit exists. In order for the limit to exist, the curve must be smooth; that is the tangent vector (velocity vector; v = dr/dt) of the curve can never be equal to zero. As described in section 13.3, the arc length s(t) is the integral of the velocity vector in terms of t along a curve bounded by [a, b] or [t0, t]

  • To integrate the function f(x, y, z) along the parametrised curve C between [a, b], ds is rewritten as the magnitude of the velocity vector function multiplied by the differential of t:
      • ds = ||v(t)|| dt

  • Therefore, the line integral can be rewritten as:
      • ∫C(f(x, y, z) ds = ∫ab(f(g(t), h(t), k(t)) ||v(t)||) dt
      The integral on the left is the line integral of the function whilst the integral on the right is the simple definite integral of the function

  • The initial point of the curve C in terms of the parametric vector function r(t) is at the position r(a) and the terminal point is the position r(b)

  • Line integrals have an additive property where multiple integrals can be summed for every subcurve in a piecewise curve. Depending on which path is taken to reach the end of the last subcurve, there is a different value for integrating over an individual subcurve
    Example 1:
      Integrate function f(x, y, z) = {x - 3y2 + z} over the line segment C made from the origin to point P = (1, 1, 1)
    1. The simplest parametrisation of the curve is by using the parametric vector r(t) = {ti + tj + tk for t in [0, 1]}. Therefore, the tangent vector v(t) and its magnitude are written as:
        2. v(t) = dr/dt
        v(t) = i + j + k

        ||v(t)|| = sqrt(3)

    2. The integral is written and evaluated as shown:
        3. ∫01(f(r(t)) sqrt(3)) dt
        sqrt(3) ∫01((t) - 3(t)2 + (t)) dt
        sqrt(3) [t2 - t3]01
        sqrt(3) (1 - 1)
        0
    Example 2:
      Using the same function as the previous example, another path is chosen to reach the point (1, 1, 1). This time, an initial path C1 runs from the origin to point (1, 1, 0) and a second path C2 runs from point (1, 1, 0) to (1, 1, 1). Integrate this function over C1∪C2 (piecewise curve addition).

    1. The simplest parametrisation of C1 is found by using the parametric vector r1(t) = {ti + tj for t in [0, 1]}. The velocity vector and its magnitude are written as:
        2. v1(t) = i + j

        ||v1(t)|| = sqrt(2)

    2. The integral of the function along this curve is expressed as:
        3. ∫C1(f(x, y, z)) ds = ∫01(f(r1(t)) ||v1(t)||) dt
        ∫C1(f(x, y, z)) ds = sqrt(2) ∫01((t) - 3(t)2) dt
        ∫C1(f(x, y, z)) ds = sqrt(2) [t2/2 - t3]01
        ∫C1(f(x, y, z)) ds = sqrt(2) (1/2 - 1)
        ∫C1(f(x, y, z)) ds = -sqrt(2)/2

    3. The simplest parametrisation of C2 is found by using the parametric vector r2(t) = {i + j + tk for t in [0, 1]}. The velocity vector and its magnitude are:
        4. v(t) = k

        ||v(t)|| = 1

    4. So, the integral of the function along this curve is:
        5. ∫C2(f(x, y, z)) ds = ∫01(f(r2(t)) ||v2(t)||) dt
        ∫C2(f(x, y, z)) ds = ∫01((1) - 3(1)2 + (t)) dt
        ∫C1(f(x, y, z)) ds = ∫01(t - 2) dt
        ∫C1(f(x, y, z)) ds = [t2/2 - 2t]01
        ∫C1(f(x, y, z)) ds = 1/2 - 2
        ∫C1(f(x, y, z)) ds = -3/2

    5. The integral of the function along the entire path (C1 & C2) is the sum of the 2 individuals line integrals made from them:
        6. ∫(C1, C2)(f(x, y, z)) ds = -sqrt(2)/2 - 3/2
        ∫(C1, C2)(f(x, y, z)) ds = -(sqrt(2) - 3)/2
    Example 3:
      Find the line integral given function f(x, y, z) = {2xy + sqrt(z)} and helical vector r(t) = {cos(t)i + sin(t)j + tk, for t in [0, π]}.

    1. The velocity vector function & its magnitude based on the given positional vector function are:
        2. v(t) = -sin(t)i + cos(t)j + k

        ||v(t)|| = sqrt(2)

    2. Therefore, the line integral for the function is written as:
        3. ∫0π(f(r(t)) ||v(t)||) dt
        sqrt(2) ∫0π(2cos(t)sin(t) + sqrt(t)) dt
        sqrt(2) ∫0π(sin(2t) + sqrt(t)) dt
        sqrt(2) [2t sqrt(t)/3 - cos(2t)/2]0π
        sqrt(2) (2π sqrt(π)/3 - 1/2 + 1/2)
        2π sqrt(2π)/3
    Applying Line Integrals to Mass & Moment
  • In the case of coils or springs, the mass of the coil is distributed along it length. The density δ(x, y, z) represents the amount of mass per unit of length. let C be the curve made by the coil, r(t) = {x(t)i + y(t)j + z(t)k, for t in [a, b]} is the positional vector function that makes up the curve. Using this information, the total mass m of the coil is the line integral of the density multiplied by the magnitude of the tangent vector function v(t) between [a, b]. This is written as:
      • m = ∫C(δ) ds
      m = ∫ab(δ(x(t), y(t), z(t)) ||v(t)||) dt
      This formula also applies to thin rods and wires
    Example 4:
      A slender metal arch that is denser towards the bottom lies along the semicircle {y2 + z2, z >= 0}. Find the centre of the arch's mass given its density δ(x, y, z) = {2 - z}.
    1. let y = cos(t), z = sin(t); Since the arch lies within the yz-plane, the x & y coordinates of its centre of mass are both at 0. To find the z-coordinate, the semicircle is parametrised as a parametric vector function r(t) using the defined parametric equations. This positional vector, the velocity vector & its magnitude are:
        2. r(t) = cos(t)j + sin(t)k, for t in [0, π]

        v(t) = -sin(t)j + cos(t)k

        ||v(t)|| = 1

    2. The mass m of the arch is evaluated as:
        3. m = ∫C(δ) ds
        m = ∫0π(δ(0, cos(t), sin(t))) dt
        m = ∫0π(2 - sin(t)) dt
        m = [2t + cos(t)]0π
        m = 2π - 1 - 1
        m = 2(π - 1)

    3. The z-moment about the xy-plane (Mxy) is evaluated as:
        4. Mxy = ∫C(z δ) ds
        Mxy = ∫0π(sin(t) δ(0, cos(t), sin(t))) (Every z is replaced with sin(t) as given in the positional vector)
        Mxy = ∫0π(sin(t) (2 - sin(t))) dt
        Mxy = ∫0π(2sin(t) - sin2(t)) dt
        Mxy = ∫0π(2sin(t) + (cos(2t) - 1)/2) dt
        Mxy = [sin(t)/4 - t/2 - 2cos(t)]0π
        Mxy = 2 - π/2 + 2
        Mxy = 4 - π/2

    4. The z-coordinate c is determined by dividing the z-moment by the mass:
        5. c = Mxy / m
        c = (4 - π/2) / (2(π - 1))
        c ≈ 0.57
        Therefore, the centre of mass is the point (0, 0, 0.57)
    Line Integrals for Curved Planes
  • Assume a smooth curve C is parametrised by positional vector r(t) = {x(t)i + y(t)j, for t in [a, b]}, a curved plane is formed perpendicular to the xy-plane by letting z = f(x, y) = any real positive number. As long as z is positive, the graph of f will always lie above the curve, forming and curved plane with the height of f(x, y). The Riemann Sum (SN) and its line integral to solve for the area of the curved plane are then written as:
      • SN = ∑k = 1N(f(xk, yk)) Δsk
      limN ➝ ∞(SN) = ∫C(f) ds

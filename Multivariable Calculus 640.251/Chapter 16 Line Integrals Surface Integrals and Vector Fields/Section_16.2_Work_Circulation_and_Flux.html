Section 16.2: Work, Circulation & Flux

    Vector Fields
  • Gravitational & electrostatic forces are expressed as vectors, having both a magnitude & direction. These forces produce a vector field as work is performed on an object affected by the force. Work can be calculated using line integrals using the vector field
      • A vector field is a vector function that assigns a vector to every one of its components within their domain. It is continuous/differentiable if each one of its component functions are continuous/differentiable
      Examples include:
    • F(x, y) = M(x, y)i + N(x, y)j
    • G(x, y, z) = Q(x, y, z)i + R(x, y, z)j + S(x, y, z)k

  • A velocity vector function can also be represented a vector field such as a fluid particle's velocity. Vector arrows are drawn with its tail at the originating position and the arrow pointing towards the direction of the vector field where the function increases most rapidly. The vector functions are always evaluated at the originating point. The length of the arrow represents its speed or magnitude

  • A radial field is demonstrated by the vector field F(x, y) = {xi + yj} and a spinning field is demonstrated by G(x, y) = {(-yi + xj) / sqrt(x2 + y2)}
    Gradient Fields
  • A gradient field is a vector field made of gradient vectors. Its component functions are the partial derivatives of the original vector function's components:
      • ∇F = (∂F/∂x)i + (∂F/∂y)j + (∂F/∂z)k

  • Gradient vectors are used to determine the vectors that point toward the greatest increase of f at a given point

  • In many physics applications, F is represented as potential energy, usually expressed as negative to show the areas of decreasing potential energy
    Example 1:
      Suppose a material is heated and the resulting temperature T at position P = (x, y, z) given by the equation T = {100 - x2 - y2 - z2}. let F(x, y, z) be the gradient vector of T. Solve for F.
    1. The gradient of T is found by finding the partial derivatives of the function T and arranging them into a vector:
        2. ∂f/∂x = -2x; ∂f/∂y = -2y; ∂f/∂z = -2z;
        F(x, y, z) = -2xi - 2yj - 2zk
        This gradient vector function reveals the direction of greatest increase of temperature according to the position
    Line Integrals of Vector Fields
  • Fluid flow, work, energy, electrical & gravitational fields along a curved region can be measured using line integrals of given vector fields
  • let continuous vector-field F(x, y, z) = {M(x, y, z)i + N(x, y, z)j + P(x, y, z)k}; and vector-function r(t) = {g(t)i + h(t)j + k(t), for t in [a, b]} is the parametrised vector for the smooth curve C. Increasing t from a to b reveals the forwards direction of the curve. The tangent (velocity) vector v(t) is found by differentiating r(t). The unit tangent vector T is the direction of v(t) (or v/|v|); this can also be found by differentiating r in terms of the arc length s
    • . Therefore, the line integral of F along C is expressed as shown:
      ∫C(F ⋅ T) ds = ∫C(F ⋅ (dr/ds)) ds
      ∫C(F ⋅ T) ds = ∫C(F ⋅ dr)
  • Converting the line integral into a definite integral for evaluation:
      • ∫C(F ⋅ dr) = ∫ab(F(r(t)) ⋅ dr/dt) dt
      ∫C(F ⋅ dr) = ∫ab(F(r(t)) ⋅ v(t)) dt
    Example 2:
      Evaluate the line integral of F(x, y, z) = {zi + xyj - y2k} along curve C parametrised by vector function r(t) = {t2i + tj + sqrt(t)k, for t in [0, 1]}.

    1. To write this line integral, F must be expressed in terms of t by letting r(t) be its argument:
        2. F(r(t)) = sqrt(t)i + (t2)(t)j - (t)2k
        F(r(t)) = sqrt(t)i + t3j - t2k

    2. The velocity vector v(t) is the derivative of r(t):
        3. v(t) = dr/dt
        v(t) = 2ti + j + 1/(2 sqrt(t))k

    3. Writing & evaluating the line integral:
        4. ∫C(F ⋅ dr) = ∫ab(F(r(t)) ⋅ v(t)) dt
        ∫C(F ⋅ dr) = ∫01(< sqrt(t), t3, -t2 > ⋅ < 2t, 1, 1/(2 sqrt(t)) >) dt
        ∫C(F ⋅ dr) = ∫01((2t sqrt(t) + t3 - t sqrt(t)/2) dt
        ∫C(F ⋅ dr) = ∫01((t3 + 3t sqrt(t)/2) dt
        ∫C(F ⋅ dr) = [t4/4 + 3t2 sqrt(t)/5]01
        ∫C(F ⋅ dr) = 1/4 + 3/5
        ∫C(F ⋅ dr) = 17/20
    Line Integrals in Cartesian Form
  • In some applications, only a specific component is necessary when evaluating the motion of a vector field. In the case of a vector field F(x, y, z) = M(x, y, z)i, only an x-component exists. If this vector field is to be evaluated over curve C parametrised by r(t) = {g(t)i + h(t)j + k(t)k}, the dot product of F & dr will always be M(x, y, z) g'(t) dt. The line integral can therefore be simplified as:
      • ∫C(F ⋅ dr) = ∫C(M(x, y, z) g'(t)) dt
      ∫C(F ⋅ dr) = ∫C(M(x, y, z)) dx
      ∫C(F ⋅ dr) = ∫ab(M(r(t)) g'(t)) dt
      This is the evaluation of the line integral of F along C where F has only an x-component

  • If F has only a y-component of N(x, y, z), the line integral is:
      • ∫C(F ⋅ dr) = ∫C(N(x, y, z)) dy
      ∫C(F ⋅ dr) = ∫ab(N(r(t)) h'(t)) dt

  • If F has only a z-component of P(x, y, z), the line integral is:
      • ∫C(F ⋅ dr) = ∫C(P(x, y, z)) dz
      ∫C(F ⋅ dr) = ∫ab(P(r(t)) k'(t)) dt

  • The sum of the 3 line integrals of F for each component is written as:
      • ∫C(M(x, y, z)) dx + ∫C(N(x, y, z)) dy + ∫C(P(x, y, z)) dz = ∫C(M dx + N dy + P dz)
    Example 3:
      Evaluate the line integral ∫C(-y dx + z dy + 2x dz) where helix C is parametrised by vector r(t) = {cos(t)i + sin(t)j + tk, for t in [0, 2π]}
    1. Solving for the velocity vector v(t):
        2. v(t) = -sin(t)i + cos(t)j + k

    2. Expressing the integrand in terms of t:
        3. dx = -sin(t) dt; dy = cos(t) dt; dz = dt
        -y dx = sin2(t) dt; z dy = t cos(t) dt; 2x dz = 2 cos(t) dt

    3. Making appropriate substitutions & evaluating the given line integral:
        4. ∫C(-y dx + z dy + 2x dz) = ∫02π(sin2(t) + t cos(t) + 2 cos(t)) dt
        ∫C(-y dx + z dy + 2x dz) = ∫02π((1 - cos(2t))/2 + t cos(t) + 2 cos(t)) dt
        ∫C(-y dx + z dy + 2x dz) = [t/2 - sin(2t)/4 + t sin(t) + cos(t) + 2 sin(t)]02π
        ∫C(-y dx + z dy + 2x dz) = π
    Work and Circulation Along a Curve in Space
  • Assume vector field F(x, y, z) = {M(x, y, z)i + N(x, y, z)j + P(x, y, z)k} is a force (gravitational, electromagnetic, or other) that occurs along a curve C parametrised by r(t) = {g(t)i + h(t)j + k(t)k, for t in [a, b]}. To solve for the work done by this force throughout the curve, the curve is partitioned into N amount of subarcs (s). The k-th subarc exists from point Pk - 1 to Pk. Within this subarc lies a point (xk, yk, zk) with a tangent unit vector of T(xk, yk, zk). The work done by the force along the k-th subarc of the curve is found by multiplying the projection of F onto the tangent vector (dot product with T) with the arclength Δsk. So, the total work of the force along the curve is the Riemann sum of all the work values for all subarcs. Letting N approach infinity reveals the line integral of the total work:
      • W = ∑k = 1N(F(xk, yk, zk) ⋅ T(xk, yk, zk)) Δsk
      W = ∫C(F ⋅ T) ds
      W = ∫ab(F(r(t)) ⋅ dr/dt) dt
      W = ∫ab(F(r(t)) ⋅ v(t)) dt
  • The same integral can be used to calculate the flow of a fluid within a given region. This integral would then be known as a flow integral. If the initial position happens to be the same point as the terminating position (a = b), this type of flow is referred to as circulation around the given curved region
      • Any vector field tangent to the curve has maximum circulation
      Any vector field normal to the curve has zero circulation
    Example 4:
      Find the work done by the force field F(x, y, z) = {(y - x2)i + (z - y2)j + (x - z2)k} in moving an object along the parametrised curve r(t) = {ti + t2j + t3k, for t in [0, 1]} from the origin to point P = (1, 1, 1).

    1. The velocity vector function v(t) is written as:
        2. v(t) = i + 2tj + 3t2

    2. Writing the force vector F in terms of t:
        3. F(r(t)) = (t2 - t2)i + (t3 - t4)j + (t - t6)k
        F(r(t)) = (t3 - t4)j + (t - t6)k

    3. The work W done by the force along the parametrised curve is the integral of the dot product of the force vector & the velocity vector:
        4. W = ∫01(F(r(t)) ⋅ v(t)) dt
        W = ∫01(< 0, t3 - t4, t - t6 > ⋅ < 1, 2t, 3t2 >) dt
        W = ∫01(2t4 - 2t5 + 3t3 - 3t8) dt
        W = [2t5/5 - t6/3 + 3t4/4 - t9/3]01
        W = 2/5 - 1/3 + 3/4 - 1/3
        W = 24/60 - 40/60 + 45/60
        W = 29/60
    Example 5:
      Find the work done by the force field F(x, y, z) = {xi + yj + zk} in moving an object along curve C parametrised by vector r(t) = {cos(πt)i + t2j + sin(πt)k, for t in [0, 1]}.

    1. The velocity vector function is the derivative of the positional vector:
        2. v(t) = {-π sin(πt)i + 2tj + π cos(πt)k}

    2. Setting r(t) as the argument of F:
        3. F(r(t)) = cos(πt)i + t2j + sin(πt)k

    3. The work W is solved using the following integration:
        4. W = ∫01(F(r(t)) ⋅ v(t)) dt
        W = ∫01(< cos(πt), t2, sin(πt) > ⋅ < -π sin(πt), 2t, π cos(πt) >) dt
        W = ∫01(-π sin(2πt)/2 + 2t3 + π sin(2πt)/2) dt
        W = [t4/2]01
        W = 1/2
    Example 6:
      A fluid's velocity field is F(x, y, z) = {xi + zj + yk}. Find the fluid's flow along the helix r(t) = {cos(t)i + tj + sin(t)k, for t in [0, π/2]}.

    1. The velocity vector v of the helix is:
        2. v(t) = -sin(t)i + j + cos(t)k

    2. The velocity field vector in terms of t is:
        3. F(r(t)) = cos(t)i + sin(t)j + tk

    3. The flow of the fluid is determined as shown:
        4. flow = ∫0π/2(< cos(t), sin(t), t > ⋅ < -sin(t), 1, cos(t) >) dt
        flow = ∫0π/2(-sin(2t)/2 + sin(t) + t cos(t)) dt
        flow = [cos(2t)/4 - cos(t) + t sin(t) + cos(t)]0π/2
        flow = (-1/4 - 1/4) + π/2
        flow = (π - 1)/2
    Example 7:
      Find the circulation of the field F(x, y) = {(x - y)i + xi} around the circle r(t) = {cos(t)i + sin(t)j, for t in [0, 2π]}.

    1. The velocity vector v is:
        2. v(t) = -sin(t)i + cos(t)j

    2. Setting r(t) as the argument of F:
        3. F(r(t)) = (cos(t) - sin(t)i + cos(t)j)

    3. The circulation of the field vector is:
        4. circulation = ∫02π(< cos(t) - sin(t), cos(t) > ⋅ < -sin(t), cos(t) >) dt
        circulation = ∫02π(-sin(2t)/2 + sin2(t) + cos2(t)) dt
        circulation = ∫02π(1 - sin(2t)/2) dt
        circulation = [t + cos(2t)/4]02π
        circulation = 2π
        This assumes the field vector circulates around the circle in a counter-clockwise direction, hence why it's a positive result
    Flux Across Plane Curves
  • A curve on the xy-plane can be categorised as simple or complex (non-simple)
      • Simple curves do not overlap at any time (one-to-one curve; one coordinate point for any t)
      Complex curves overlap (more than one same coordinate point for any t)

  • A curve can also be classified using its closed or open (not closed) nature
      • Closed curves always form a loop (a terminating point is an initial point)
      Open curves do not form a loop (distinct initial & terminating point)

  • To determine the rate at which a fluid enters or leaves a region enclosed by a smooth simple closed curve C on the xy-plane, the line integral of the dot product vector field F & the normal unit vector n of C (pointing outwards) is calculated. The flow of F along C is its flux:
      • flux = ∫C(F ⋅ n) ds

  • The normal unit vector of a parametrised curve on the xy-plane is determined by taking the cross product of its unit tangent vector T & the z-unit vector k. Since the curve in terms of t can have 2 directions: clockwise or counterclockwise, the order of the cross product matters when determining the outward normal unit vector
      • For a clockwise motion, the normal unit vector that points outward is the resulting vector of k ⨯ T
      For a counter-clockwise motion, the normal unit vector that points outward is the resulting vector of T ⨯ k. A counter-clockwise direction of a curve is the standard direction in Calculus

  • Assume curve C has the following parameteric equations: x = g(t), y = h(t), for t in [a, b]; and vector-field F(x, y) = {M(x, y)i + N(x, y)j}; then the following must be true:
      • T = (dx/ds)i + (dy/ds)j

      n = T ⨯ k
      n = < dx/ds, dy/ds, 0 > ⨯ < 0, 0, 1 >
      n = (dy/ds)i - (dx/ds)j

  • The line integral of F of the curve flowing towards the counter-clockwise direction can now be rewritten as:
      • ∫C(F ⋅ n) ds = ∫C(M(x, y) (dy/ds) - N(x, y) (dx/ds)) ds
      ∫C(F ⋅ n) ds = ∳C(M(x, y) dy - N(x, y) dx)
      ∫C(F ⋅ n) ds = ∫ab(M(r(t)) h'(t) - N(r(r)) g'(t) dt)
      The counter-clockwise integral ∳ hints the counter-clockwise direction of the curve which traces around the curve only once
    Example 8:
      Find the flux of F(x, y) = {(x - y)i + xj} across the circle C = {x2 + y2 = 1} in the xy-plane.

    1. Parametrising the curve in terms of t:
        2. let x = cos(t), y = sin(t), for t in [0, 2π]
        vector r(t) = cos(t)i + sin(t)j

        dx = -sin(t) dt; dy = cos(t) dt

    2. Assigning the vector field in terms of r(t):
        3. F(r(t)) = (cos(t) - sin(t))i + cos(t)j

    3. The c-clockwise integral and its evaluation to solve for the solve of F along the circle are:
        4. flux = ∳C((x - y) dy - x dx)
        flux = ∫02π((cos(t) - sin(t))(cos(t)) - (cos(t))(-sin(t))) dt
        flux = ∫02π(cos2(t) - sin(2t)/2 + sin(2t)/2) dt
        flux = 1/2 ∫02π(1 + cos(2t)) dt
        flux = 1/2 [t + sin(2t)/2]02π
        flux = 1/2 (2π)
        flux = π
        The answer is positive, correctly demonstrating outward flux

Section 14.5: Directional Derivatives & Gradient Vectors

    Directional Derivatives
  • The directional unit vector at a given point points towards the direction of a curve
  • Assume function f(x, y) & point P = (a, b) are in a certain region, the unit vector {u = uxi + uyj} points towards the direction of the curve. The components of this vector are found by parametrising f through point P parallel to u according to the arc length s. These parametric equations are written as:
      • x = a + sux; y = b + suy
  • The directional derivative (Duf(P) or dfu(P)) in terms of s is found by taking the derivative of f(x, y) at evaluated at point P = (a, b) in the direction of the directional unit vector u. It is shown by the equation:
      • dfu(P)/ds = lims -> 0((f(a + sux, b + suy) - f(a, b)) / s)
  • Another way to determine the directional derivative is to assign the partial derivatives of f as their respective components evaluated at point P = (a, b) for the directional vector. Let {x = a + s(dx/ds), y = b + s(dy/ds)} be the line the point P = (a, b) parametrised by the arc length s and {u = (dx/ds)i + (dy/ds)j} be the unit vector in the direction of increasing s, the direction derivative of f evaluated at point P in the direction of u is as follows:
      • (df(P)/ds)u = (∂f(P)/∂x)(dx/ds) + (∂f(P)/∂y)(dy/ds)
      (df(P)/ds)u = ((∂f(P)/∂x)i + (∂f(P)/∂y)j) ⋅ ((dx/ds)i + (dy/ds)j)
      (df(P)/ds)u = ∇f ⋅ u = F ⋅ u
      {∇f = F = (∂f/∂x)i + (∂f/∂y)ji} in this case is known as the gradient vector of f(x, y)

  • The value of the gradient vector is found by evaluating the partial derivatives of the function at a given point
    Example 1:
  • Find the derivative of {f(x, y) = x2 + xy} at point P = (1, 2) in the direction of the unit vector {u = (1/sqrt(2))i + (1/sqrt(2))j}

      • Using the equation for directional derivatives:
    • dfu(1, 2)/ds = lims -> 0((f(1 + s(1/sqrt(2)), 2 + s(1/sqrt(2))) - f(1, 2)) / s)
    • dfu(1, 2)/ds = lims -> 0(((1 + s/sqrt(2))2 + (1 + s/sqrt(2))(2 + s/sqrt(2))) - (1 + 2)) / s)
    • dfu(1, 2)/ds = lims -> 0(((1 + s sqrt(2) + s2/2) + (2 + (3s sqrt(2))/2 + s2/2)) - 3) / s)
    • dfu(1, 2)/ds = lims -> 0((s2 + (5s sqrt(2))/2 + 3) - 3) / s)
    • dfu(1, 2)/ds = lims -> 0(s + (5 sqrt(2))/2)
    • dfu(1, 2)/ds = (5 sqrt(2)) / 2
    Example 2:
      Find the derivative of {f(x, y) = x ey + cos(xy)} at point Q = (2, 0) in the direction of {v = 3i - 4j}.

    1. let u = the directional unit vector of v. u is found as shown:
        2. u = v/||v||
        u = < 3, -4 > / 5
        u = (3/5)i - (4/5)j

    2. The gradient vector of f(x, y) (F) is expressed as:
        3. F = (∂f/∂x)i + (∂f/∂y)j
        F = (ey - y sin(xy))i + (x ey - x sin(xy))j

    3. Evaluating F at the point Q:
        4. F(2, 0) = i + 2j

    4. dfu evaluated at point Q can now be solved (using the dot product):
        5. dfu(2, 0) = F ⋅ u
        dfu(2, 0) = < 1, 2 > ⋅ < 3/5, -4/5 >
        dfu(2, 0) = 3/5 - 8/5
        dfu(2, 0) = -1
    Properties of Directional Derivatives
  • Since the dot product can be used to find the directional derivative of a function, another way of calculating this vector is by using the angle between the 2 vectors. It can be calculated as shown:
      • dfu = ||f|| ||u|| cosθ
      dfu = ||f|| cosθ
      dfu = f ⋅ u

  • The function f(x, y) increases most rapidly when cosθ = 1 (θ = 0 or 2π). In the direction of this angle, the directional derivative would be equal to the magnitude of the gradient of f:
      • df(u, θ = 0) = ||F||
  • Similarly, the function decreases most rapidly when cosθ = -1 (θ = π). In the direction of this angle, the directional derivative would be equal to the opposite of the magnitude of F:
      • df(u, θ = π) = -||F||
  • The function is constant (zero change in f) when cosθ = 0 (θ = π/2). The unit vector would be orthogonal to the gradient of f, leading to a dot product equal to 0
      • df(u, θ = π/2) = 0
    Example 3:
  • Find the directions in which {f(x, y) = x2/2 + y2/2} a) Increases most rapidly; b) Decreases most rapidly; c) There is no change in f; evaluated at the point P = (1, 1).

      • a)
    • The gradient vector of f (F) is solved as follows:
        • F = (∂f/∂x)i + (∂f/∂y)j
        F = xi + yj

    • Evaluating this vector at point P:
        • F(1, 1) = i + j

    • The direction where the function increases most rapidly is the direction u of the evaluated gradient vector:
        • u = F(1, 1)/||F(1, 1)||
        u = < 1, 1 > / sqrt(2)
        u = (1/sqrt(2))i + (1/sqrt(2))j

      b)
    • The direction where the function decreases most rapidly is the opposite if the directional unit vector:
        • -u = -(1/sqrt(2)i - (1/sqrt(2)))

      c)
    • The direction where the function has no change is an orthogonal version of u, normal to the function (n) and its opposite:
        • n = -(1/sqrt(2))i + (1/sqrt(2))j
        n = (1/sqrt(2))i - (1/sqrt(2))j
    Gradients & Tangents to Level Curves
  • Assume a differentiable function f(x, y) is equal to a constant value c (forming a level curve at c) along a smooth curve {r(t) = g(t)i + h(t)j}, the following must be true:
      • f(g(t), h(t)) = c
      df(g(t), h(t))/dt = dc/dt
      (∂f/∂x)(dg/dt) + (∂f/∂y)(dh/dt) = 0 (via Chain Rule; dg = dx & dh = dy)
      ((∂f/∂x)i + (∂f/∂y)j) ⋅ ((dg/dt)i + (dh/dt)j) = 0
      F ⋅ dr/dt = 0
      F ⋅ v = 0
      This shows that the gradient vector of f(x, y) at a level curve of a constant is always normal (orthogonal) to the velocity vector made from the tangent vector of the positional vector

  • The tangent line made in the direction of the velocity vector made from a level curve of f(x, y) and normal to the gradient vector of f at point P = (a, b) is expressed as follows:
      • (∂f(a, b)/∂x)(x - a) + (∂f(a, b)/∂y)(y - b) = 0

  • Gradient vectors have the same vector algebra as an ordinary vector

  • Gradient vectors can also be made by a triple-variable function like f(x, y, z) and will have the following components:
      • F = (∂f/∂x)i + (∂f/∂y)j + (∂f/∂z)k
    Example 4:
      Find the equation for the line tangent to the ellipse made by {x2/4 + y2 = 2} at point Q = (-2, 1)

    1. let g(x, y) = x2/4 + y2. The partial derivatives of g are:
        2. ∂g/∂x = x/2; ∂g/∂y = 2y;

    2. Evaluating the partial derivatives at point Q:
        3. ∂g(-2, 1)/∂x = -1; ∂g(-2, 1)/∂y = 2;

    3. This means the gradient vector of f at Q will have the form:
        4. F = -i + 2j

    4. Therefore, the tangent line L is expressed as:
        5. L = {-(x - (-2)) + 2(y - (1)) = 0} (Original constant 2 of level curve is disregarded)
        L = {-x - 2 + 2y - 2 = 0}
        L = {2y - x = 4}
    Example 6:
      Find the derivative of {f(x, y, z) = x3 - xy2 - z} evaluated at point P = (1, 1, 0) in the direction of vector {r = 2i - 3j + 6k} Then find the max, min and zero changes of f.

    1. The 3 partial derivatives of f are as shown:
        2. ∂f/∂x = 3x2 - y2; ∂f/∂y = -2xy; ∂f/∂z = -1;

    2. Evaluating each partial derivative at point P:
        3. ∂f(1, 1, 0)/∂x = 2; ∂f(1, 1, 0)/∂y = -2; ∂f(1, 1, 0)/∂z = -1;

    3. let F = the gradient vector of f(x, y, z) evaluated at point P. F is expressed as:
        4. F(1, 1, 0) = 2i - 2j - k

    4. let u = the direction of r. u is evaluated as:
        5. u = r / ||r||
        u = < 2, -3, 6 > / 7
        u = (2/7)i - (3/7)j + (6/7)k

    5. The directional derivative of f at P in the direction of u is the dot product of the gradient vector of f & the unit vector u:
        6. dfu(1, 1, 0) = F(1, 1, 0) ⋅ u
        dfu(1, 1, 0) = < 2, -2, -1 > ⋅ < 2/7, -3/7, 6/7 >
        dfu(1, 1, 0) = 4/7 + 6/7 - 6/7
        dfu(1, 1, 0) = 4/7

    6. let D = the direction in which the function increases most rapidly. D is the magnitude of the gradient vector of f(P):
        7. D = F / ||F(1, 1, 0)||
        D = < 2, -2, -1 > / 3
        D = (2/3)i - (2/3)j - k

    7. The function decreases most rapidly at the opposite direction of D:
        8. -D = -(2/3)i + (2/3)j + k

    8. The function has a change of zero at the direction orthogonal to D (by taking the opposite of one component)
    Derivatives along Paths
  • Assume r(t) = x(t)i + y(t)j + z(t)k is the positional vector function for a smooth path C, v(t) is the derivative of r in terms of t, and w = f(r(t)) is a scalar function evaluated along C. The derivative of w or f in terms of t is calculated using the chain rule or dot product:
      • ∂w/∂t = (∂w/∂x)(∂x/∂t) + (∂w/∂y)(∂y/∂t) + (∂w/∂z)(∂z/∂t)
      df(r(t))/dt = F(r(t)) ⋅ v(t)

Section 14.2: Limits & Continuity

    Limits of Multivariable Functions
  • Given an input of (a, b), if f(x, y) is arbitrarily close to a real number L, L is referred to as the limit of f(x, y). The distance between (x, y) & (a, b) cannot be zero but can be infinitely close to zero. The limit can be reached afromany direction, unlike single-variable functions which can only be reached from the left & right. In order for the limit to exist, it must be able to be reached from any direction. This limit is shown as:
      • lim(x, y) -> (a, b)(f(x, y)) = L

  • The follow assumptions must also be true:
      • For all ε > 0, there exists a δ > 0 for all (x, y) in the domain of f. δ is the radius r of the circle centered at (x0, y0). (x, y) can exist within any points within the boundaries of (L ± ε), making up the circle
      |f(x, y) - L| < ε
      r = sqrt((x - x0)2 + (y - y0)2)
      0 < r < δ or r ∈ (0, δ)

  • Limits for multivariable functions share very similar principles as single-variable functions

    Example 4:
  • Given f(x, y) = y/x, does lim(x, y) -> (0, 0)(f(x, y)) exist?

      • Letting y = x leads to the expression f(x, x) = x/x = 1. This implies that on the limit is always 1 on the line {y = x} regardless of the limit inputs. If y were any other value, other than x, and the limit was applied to the expression, an indeterminant expression error would be raised. Therefore, this specific limit does not exist because it is only rational at the direction of the line {y = x} but not any other region
    Continuity of Multivariable Functions
  • In order for a function f(x, y) to be continuous at point (a, b), it must follow 3 rules:
      • f(a, b) == true
      lim(x, y) -> (a, b)(f(x, y)) == true = L (from ALL directions)
      lim(x, y) -> (a, b)(f(x, y)) = f(a, b)
  • The function is continuous at every point if the above rules apply for (a, b) ∈ (domain of f)
  • If both a multivariable function f(x, y) & single-variable function g(t) are continuous, then the composition h(x, y) = g(f(x, y)) is also continuous

  • The 2-path test is the concept of testing the function for a specific limit along different paths. If a different result is achieved when attempting to find a limit along 2 different routes, then the limit is classified as non-existent (false)

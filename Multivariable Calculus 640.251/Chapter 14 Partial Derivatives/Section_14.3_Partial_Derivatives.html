Section 14.3: Partial Derivatives

    Introduction
  • A partial derivative is the differentiation of an independent variable with respect to another independent variable
  • The tangent line of the curve z = f(x, y) at the point P = (a, b, f(a, b)) is given by the limit of the distance between f(a, b) & f(a + h, b) when dividing by h as h approaches zero. Since this limit involves a change in the x-coordinate, the result is referred to as the partial derivative of f with respect to x (∂f/∂x or fx). This partial derivative is shown as:
      • ∂f(a, b)/∂x = limh -> 0((f(a + h, b) - f(a, b)) / h) == true
    • A partial derivative of f with respect to y (∂f/∂y or fy)is shown identical to the previous expression, except f(a + h, b) is replaced f(a, b + h)

  • Continuous partial derivatives of a function don't necessarily imply that the function is continuous itself
    Example 1:
  • Given f(x, y) = {x2 + 3xy + y - 1}, find a) ∂f/∂x; b) ∂f/∂y; at point (4, -5).

      • a)
    • To find ∂f/∂x, y is treated as a constant and both sides are differentiated with respect to x:
        • ∂f/∂x = ∂(x2 + 3xy + y - 1)/∂x
        ∂f/∂x = 2x + 3y

    • Substituting the respective variables with the given point:
        • ∂f(4, -5)/∂x = 2(4) + 3(-5)
        ∂f(4, -5)/∂x = -7

      b)
    • To find ∂f/∂y, x is treated as a constant and both sides are differentiated with respect to y:
        • ∂f/∂y = ∂f(x2 + 3xy + y - 1)/∂y
        ∂f/∂y = 3x + 1

    • Substituting the respective variables with the given point:
        • ∂f(4, -5)/∂y = 3(4) + 1
        ∂f(4, -5)/∂y = 13
    Example 2:
  • Find ∂f/∂y given f(x, y) = y sin(xy).

      • This partial derivative is solved by letting x equal a constant and differentiating both sides with respect to y (Using Product Rule):
    • ∂f/∂y = ∂(y sin(xy))/∂y
    • ∂f/∂y = sin(xy) + xy cos(xy)
    Example 3:
  • Given f(x, y) = 2y / (y + cos(x)), find a) ∂f/∂x; b) ∂f/∂y;

      • a)
    • Letting y be a constant (Using quotient rule):
        • ∂f/∂x = ∂(2y / (y + cos(x)))/∂x
        ∂f/∂x = (2y)(sin(x)) / (y + cos(x))2

      b)
    • Letting x be a constant (Using Quotient Rule):
        • ∂f/∂y = ∂(2y / (y + cos(x)))/∂y
        ∂f/∂y = (2(y + cos(x)) - 2y(1)) / (y + cos(x))2
        ∂f/∂y = cos(x) / (y + cos(x))2
    Example 4:
  • Given the equation {yz - ln(z) = x + y} find ∂z/∂x.

      • Treating y as a constant (Using Implicit Differentiation):
  • ∂(yz - ln(z))/∂x = ∂(x + y)/∂x
  • y(∂z/∂x) - (1/z)(∂z/∂x) = 1
  • (y - 1/z)(∂z/∂x) = 1
  • ((zy - 1) / z)(∂z/∂x) = 1
  • ∂z/∂x = z / (zy - 1)
    Example 5:
      The plane {x = 1} intersects the paraboloid {z = x2 + y2} at a parabola. Find the slope of the line tangent to the parabola at point (1, 2, 5).

    1. The cross-section parabola at the corresponding plane exists within yz-plane. This means that the tangent line must also be within this same plane. In this case, the relevant independent variable is y and x would be a constant. Solving for the partial derivative of z with respect to y:
        2. ∂z/∂y = ∂(x2 + y2)/∂y
        ∂z/∂y = 2y

    2. Substituting the variable for the respective point:
        3. ∂z(1, 2, 5)/∂y = 2(2)
        ∂z(1, 2, 5)/∂y = 4
    Example 6:
  • Given f(x, y, z) = x sin(y + 3z), find ∂f/∂z

      • x & y are treated as constants (Using Product Rule):
    • ∂f/∂z = ∂(x sin(y + 3z))/∂z
    • ∂f/∂z = 3x cos(y + 3z)
    Example 7:
  • Assume 3 resistors R1, R2, & R3 are connected in parallel. Find ∂Req/∂R2 given the equation (1/Req) = {(1/R1) + (1/R2) + (1/R3)} when R1 = 30Ω, R2 = 45Ω, R3 = 90Ω

      • R1 & R3 are treated like constants:
    • ∂(1/Req)/∂R2 = ∂((1/R1) + (1/R2) + (1/R3))/∂R2
    • -(1/Req2)(∂Req/∂R2) = -1/R22
    • ∂Req/∂R2 = Req2/R22
    • ∂Req/∂R2 = (Req/R2)2

      Req is solved by plugging in every resistance value into the given equation:
    • 1/Req = 1/(30) + 1/(45) + 1/(90)
    • Req = (1/15)-1
    • Req = 15Ω

      Substituting the variables in the partial derivative with the resistance values given & solved:
    • ∂Req/∂R2 = ((15)/(45))2
    • ∂Req/∂R2 = (1/3)2
    • ∂Req/∂R2 = 1/9
    • This means a change in R2 affects Req 1/9 times as much
    Example 8:
      Given f(x, y) = {0 for xy != 0, 1 for xy == 0}, prove that f is not continuous at the origin when approaching the point along the line {y = x}, but its partial derivatives are.

    1. In order to determine continuity of f at the origin, we observe what happens when approaching the origin in the direction of {y = x}
        2. let y = x;
      • Based on the definition of f, f is always equal to 0 when x or y is not equal to 0 except when either x or y is equal to 0. Based on the 2-path test, there are different values for the function when approaching the origin from {y = x} even before applying the limit. So, the function is not continuous towards the origin

    2. To find the partial derivative of f with respect to x (∂f/∂x), y is set to be the constant value 0. Regardless of the value of x, f(x, y) will always be 1 & ∂f/∂x will always be 0. Similarly, to find ∂f/∂y, x is set to the constant value of 0. Then, regardless of the x-value, f(x, y) will also be 1 & ∂f/∂y will always be 0
    Second-order Partial Derivatives
  • Assume there is a function f(x, y), to solve for its 2nd order partial derivative with respect to one independent variable, the derivative will have to be taken twice while treating every other independent variable as a constant value. For instance, the 2nd partial derivative of f with respect to x is represented as ∂2f/∂x2 or fxx. This same notation is applied when taking the 2nd partial derivative of f with respect to y
  • The partial derivative of the partial derivative of f can also be mixed. When the function is differentiated first with respect to y, then with respect to x, the 2nd partial derivative is shown as ∂2f/(∂x ∂y) or fyx. When the function is differentiated first with respect x, then with respect to y, the 2nd partial derivative is shown as ∂2f/(∂y ∂x) or fxy

  • Clairaut's Theorem or The Mixed Derivative Theorem
      • As long as a function f(x, y) and its first & second partial derivatives are all continuous on a point (a, b), then the mixed 2nd-order partial derivatives must both equal each other. This is represented as:
    • fxy(a, b) = fyx(a, b)
    • ∂2f(a, b)/(∂y ∂x) = ∂2f(a, b)/(∂x ∂y)
      This also applies to n-th order mixed derivatives as well!
    Example 9:
  • Given f(x, y) = x cos(y) + y ex, find all 4 linear combinations of 2nd-order partial derivatives.

      • Solving for ∂2f/∂x2 in 2 steps:
    • ∂f/∂x = ∂(x cos(y) + y ex)/∂x
    • ∂f/∂x = cos(y) + y ex

    • ∂2f/∂x2 = ∂(cos(y) + y ex)/∂x
    • ∂2f/∂x2 = y ex

      Solving for ∂2f/∂y2 in 2 steps:
    • ∂f/∂y = ∂(x cos(y) + y ex)/∂y
    • ∂f/∂y = -x sin(y) + ex

    • ∂2f/∂y2 = ∂(ex - x sin(y))/∂y
    • ∂2f/∂y2 = -x cos(y)

      Solving for ∂2f/(∂y ∂x) using the first partial derivative with respective to x that was found (∂f/∂x):
    • ∂2f/(∂y ∂x) = ∂(cos(y) + y ex)/∂y
    • ∂2f/(∂y ∂x) = ex - sin(y)

      Solving for ∂2f/(∂x ∂y) using the first partial derivative with respective to y that was found (∂f/∂y):
    • ∂2f/(∂x ∂y) = ∂(ex - x sin(y))/∂x
    • ∂2f/(∂x ∂y) = ex - sin(y)
    Example 10:
      Given w = xy + ey / (y2 + 1), find ∂2w/(∂x ∂y).

    1. Since the function & its partial derivatives up to the 2nd-order are all continuous, ∂2w/(∂x ∂y) == ∂2w/(∂y ∂x) based on Clairaut's Theorem. In this case, ∂w/∂x is solved first instead of ∂w/∂y, simplifying the calculations:
        2. ∂w/∂x = ∂(xy + ey / (y2 + 1))/∂x
        ∂w/∂x = y

    2. Now, the partial derivative with respect to y of this partial derivative is calculated to determine ∂2w/(∂y ∂x) and as a result, ∂2w/(∂x ∂y):
        3. ∂2w/(∂x ∂y) = ∂2w/(∂y ∂x) (Clairaut's Theorem)
        ∂2w/(∂x ∂y) = ∂(y)/∂y
        ∂2w/(∂x ∂y) = 1
    Example 11:
      Find fyxyz if f(x, y, z) = 1 - 2xy2z + x2y.

    1. Solving for fy:
        2. fy = -4xyz + x2

    2. Solving for fyx:
        3. fyx = -4yz + 2x

    3. Solving for fyxy:
        4. fyxy = -4z

    4. Solving for fyxyz:
        5. fyxyz = -4
    Differentiability
  • A trace is a curve created by a vertical plane that passed through the surface of a graph. Partial derivatives have the slope of the tangent line of the trace curves
  • If a function z = f(x, y) is differentiable & continuous at (a, b) and {fx(a, b) && fy(a, b)} == true, the following assumptions must be true as c, d, Δx, & Δy all approach zero:
      • Δz = f(a + Δx, b + Δy) - f(a, b)
      Δz = fx(a, b) Δx + fy(a, b) Δy + c Δx + d Δy
      f(a + Δx, b + Δy) - f(a, b) = fx(a, b) Δx + fy(a, b) Δy + c Δx + d Δy
      This theorem is derived from the Linear Approximation Theorem used for the tangent line of a single-variable function

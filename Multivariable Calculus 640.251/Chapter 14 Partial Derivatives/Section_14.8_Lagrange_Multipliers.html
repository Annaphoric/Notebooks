          Find the points on the hyperbolic cylinder {x2 - z2 = 1} that are closest to the origin.

            2. x2 = z2 + 1

        Introduction
      • There will occasionally be cases where the variables of an interval or bounded region of a function are not easily solvable. An alternate and more convenient method for finding extreme values of a constrained function is by using Lagrange multipliers. This involves solving for the gradients of both the given function f(x, y, z) and its bounded equation g(x, y, z) = 0. The point at which the gradient vectors F & G respectively exist must share a common tangent plane & normal line. The gradients should also be equal to each other by a specific scalar multiple known as a Lagrange multiplier (λ). This is shown as:
          • F = λG for λ != 0
          (∂f/∂x)i + (∂f/dy)j + (∂f/∂z)k = λ(∂g/∂x)i + λ(∂g/∂y)j + λ(∂g/∂z)k
        Example 2b:
          Find the points on the hyperbolic cylinder {x2 - z2 = 1} that are closest to the origin using Lagrange multipliers.

        1. The cylinder can be represented as a function f & the distance formula can be represented as function g. function f(x, y, z) = x2 - z2 - 1, g(x, y, z) = x2 + y2 + z2 - r2. The first partial derivatives of both functions in terms of z, y, & z are:
            2. ∂f/∂x = 2x; ∂f/∂y = 0; ∂f/∂z = -2z; ∂g/∂x = 2x; ∂g/∂y = 2y; ∂g/∂z = 2z;

        2. gradient vector F = 2xi - 2zj, G = 2xi + 2yj + 2zk. Equating the gradient vectors by a Lagrange multiplier:
            3. F = λG
            < 2x, 0, -2z > = λ < 2x, 2y, 2z >
            2x = λ 2x; 0 = λ 2y; -2z = λ 2z;
            x != 0; λ = 1;
            y = 0; z = 0;

        3. Solving for x using the plane equation:
            4. x = sqrt((0) + 1)
            x = ±1

        4. Therefore the points closest to the origin are:
            5. (-1, 0, 0) & (1, 0, 0)
        Example 3:
          Find the greatest & smallest values that function f(x, y) = xy takes on the ellipse {x2/8 + y2/2 = 1}

        1. function g(x, y) = x2 + y2 - 1 == 0. The first partial derivatives of both functions are:
            2. ∂f/∂x = y; ∂f/∂y = x; ∂g/∂x = x/4; ∂g/∂y = y;

        2. gradient vector F = 2xi + 2yj; G = (x/4)i + yj; Applying the Lagrange multiplier method:
            3. F = λ G
            < y, x > = λ < x/4, y >
            y = λ x/4; x = λ y;
            y = λ (λ y)/4
            y = (λ2 y)/4

        3. y cannot equal to zero because x = λ y for any λ and the point (0, 0) does not satisfy g(x, y). So, λ must equal ±2 using y = (λ2 y)/4 so that x = ±2y. Solving for the variables using g(x, y):
            4. (±2y)2/8 + y2/2 = 1
            y2/2 + y2/2 = 1
            y2 = 1
            y = ±1

            x2/8 + (±1)2/2 = 1
            x2/8 = 1/2
            x = ±2

        4. Therefore, the 4 extreme points of the f(x, y) along g(x, y) are:
            5. (±2, ±1)
        Example 4:
          Find the max & min values of function {f(x, y) = 3x + 4y} on the circle {g(x, y) = x2 + y2 = 1}
      • The first partial derivatives & gradient vectors of the functions and applying the Lagrange multiplier method:
          • ∂f/∂x = 3; ∂f/∂y = 4; ∂g/∂x = 2x; ∂g/∂y = 2y;

          gradient vector F = 3i + 4j, G = 2xi + 2yj

          3 = λ 2x; 4 = λ 2y;
          x = 3/(2λ); y = 2/λ

      • Solving for λ using the constrained function g(x, y)
          • (3/2λ)2 + (2/λ)2 = 1
          9/4λ2 + 4/λ2 = 1
          λ = sqrt(25/4)
          λ = ±5/2

      • Using the value of λ to determine x & y:
          • x = 3/(2(±5/2)); y = 2/(±5/2)
          x = ±3/5; y = ±4/5;
          The extreme points of f are: (±3/5, ±4/5). These are the points where the 2 gradient vectors have the same direction & are farthest from the origin

      • The constant of the shared level curves for both functions is found by finding the values of f(x, y) at the extreme points:
          • f(3/5, 4/5) = 3(3/5) + 4(4/5); f(-3/5, -4/5) = 3(-3/5) + 4(-4/5)
          f(3/5, 4/5) = (9 + 16)/5; f(-3/5, -4/5) = -(9 + 16)/5
          f(3/5, 4/5) = 5; f(-3/5, -4/5) = -5
        Lagrange Multipliers with 2 Constrained Functions
      • A similar concept applies when dealing with solving for extreme values of a function bounded by 2 functions. Assume function f(x, y, z); & constraining functions g(x, y, z) = 0, h(x, y, z) = 0; the sum of the gradient vectors G & H for the respective bounded functions must be equal to the gradient vector F via 2 Lagrange multipliers λ & μ where neither multiplier is equal to zero. This can be written as:
          • F = λG + μH
          (∂f/∂x)i + (∂f/∂y)j + (∂f/∂z)k = λ{(∂g/∂x)i + (∂g/∂y)j + (∂g/∂z)k} + μ{(∂h/∂x)i + (∂h/∂y)j + (∂h/∂z)k}
        Example 5:
          plane E = {x + y + z = 1} cuts through the cylinder {x2 + y2 = 1}, having the shape of an ellipse. Find the points on the ellipse that lie closest & farthest from the origin (using the distance formula).

        1. function f(x, y, z) = x2 + y2 + z2; constraining function g(x, y, z) = x + y + z - 1 == 0, h(x, y, z) = x2 + y2 - 1 == 0; The first partial derivatives & gradients vectors of each function are:
            2. ∂f/∂x = 2x; ∂f/∂y = 2y; ∂f/∂z = 2z; ∂g/∂x = 1; ∂g/∂y = 1; ∂g/∂z = 1; ∂h/∂x = 2x; ∂h/∂y = 2y; ∂h/∂z = 0;

            gradient vector F = 2xi + 2yj + 2zk, G = i + j + k, H = 2xi + 2yj

        2. Applying the Lagrange Multiplier Method:
            3. F = λG + μH
            < 2x, 2y, 2z > = λ < 1, 1, 1 > + μ < 2x, 2y, 0 >

            2x = λ + μ 2x; 2y = λ + μ 2y; 2z = λ
            2x = (2z) + μ 2x
            x = z + μ x
            z = x(1 - μ)

            2y = (2z) + μ 2y
            y = z + μ y
            z = y(1 - μ)

            x(1 - μ) = y(1 - μ)
            x = y

        3. try: y = x; Using this equation & the constraining functions to solve for variables:
            4. h(x, x) = x2 + (x)2 - 1 == 0
            h(x, x) = 2x2 = 1
            x = ±1/sqrt(2); y = ±1/sqrt(2)

            g(x, x, z) = x + (x) + z - 1 == 0
            g(x, x, z) = 2x + z = 1
            z = 1 - 2(±1/sqrt(2))
            z = 1 ∓ sqrt(2)

        4. The extreme points are point A = (1/sqrt(2), 1/sqrt(2), 1 - sqrt(2)), point B = (-1/sqrt(2), -1/sqrt(2), 1 + sqrt(2)) where point B is farthest from the origin

        5. try z = 0; Using this value & the constraining functions to solve for variables:
            6. g(x, y, 0) = x + y + (0) - 1 == 0
            g(x, y, 0) = x + y = 1
            y = 1 - x; x = 1 - y

            h(x, 1 - x) = x2 + (1 - x)2 - 1 = 0
            h(x, 1 - x) = 2x2 - 2x = 0
            x = 0; y = 1

            h(1 - y, y) = (1 - y)2 + y2 - 1 == 0
            h(1 - y, y) = 2y2 - 2y = 0
            y = 0; x = 1;
        6. point C = (0, 1, 0) & D = (1, 0, 0) are also considered extreme points where either one of these 2 are closest to the origin

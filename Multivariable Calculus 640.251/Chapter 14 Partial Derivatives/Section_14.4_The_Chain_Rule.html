Section 14.4: The Chain Rule

    Introduction
  • Assume w = f(x) is a differentiable function of x and x = g(t) is a differntiable function of t, the differential of w in terms of t can be solved by multiplying the 2 derivatives together. This is shown as:
      • dw/dt = (dw/dx)(dx/dt)
      In this case, w is the dependent variable, x is the intermediate variable (since it is being cancelled out) & t is the independent variable

  • This shares a similar connotation for double-variable functions. Assume w = f(x, y) is differentiable for both x & y and x = g(t) & y = h(t) are differentiable for t. dw/dt is solved as shown:
      • dw/dt = (∂w/∂x)(dx/dt) + (∂w/∂y)(dy/dt)
  • Formally, each derivative is evaluated at a specific point, expressed as:
      • dw(t0)/dt = (∂w(x0, y0)/∂x)(dx(t0)/dt) + (∂w(x0, y0)/∂y)(dy(t0)/dt)

  • Likewise, this same concept is applied with triple-variable functions & beyond. Assume w = f(x, y, z) is differentiable for each of its independent variables and x = g(t), y = h(t) & z = i(t) are all differentiable for t. dw/dt is solved as shown:
      • dw/dt = (∂w/∂x)(dx/dt) + (∂w/∂y)(dy/dt) + (∂w/∂z)(dz/dt)
    Example 1:
      Use the Chain Rule to find the derivative of w = xy with respect to t along the path {x = cos(t)} & {y = sin(t)} at t0 = π/2.

    1. Solving for both partial derivatives of w:
        2. ∂w/∂x = y; ∂w/∂y = x;

    2. Calculating the derivatives of the functions x & y:
        3. dx/dt = -sin(t); dy/dt = cos(t);

    3. Using this info to determine the differential of w in terms of t:
        4. dw/dt = (∂w/∂x)(dx/dt) + (∂w/∂y)(dy/dt)
        dw/dt = (y)(-sin(t)) + (x)(cos(t))
        dw/dt = x cos(t) - y sin(t)

    4. Substituting x & y for its given respective functions:
        5. dw/dt = (cos(t)) cos(t) - (sin(t)) sin(t)
        dw/dt = cos2(t) - sin2(t)
        dw/dt = cos(2t)

    5. Substituting t for its given value:
        6. dw(π/2)/dt = cos(2(π/2))
        dw(π/2)/dt = -1
        Note: This value can also be solved by directly substituting the x & y functions into f(x, y), differentiating, and last, substituting the t-value
    Example 2:
      Given w = xy + z, x = cos(t), y = sin(t), z = t, use the chain rule to solve for dw/dt at t = 0.

    1. Solving for each respective partial derivative of w:
        2. ∂w/∂x = y; ∂w/∂y = x; ∂w/∂z = 1;

    2. Finding the derivatives of x, y & z:
        3. dx/dt = -sin(t); dy/dt = cos(t); dz/dt = 1;

    3. dw/dt is calculated as shown:
        4. dw/dt = (∂w/∂x)(dx/dt) + (∂w/∂y)(dy/dt) + (∂w/∂z)(dz/dt)
        dw/dt = (y)(-sin(t)) + (x)(cos(t)) + (1)(1)
        dw/dt = x cos(t) - y sin(t) + 1

        dw/dt = (cos(t)) cos(t) - (sin(t)) sin(t) + 1
        dw/dt = cos2(t) - sin2(t) + 1
        dw/dt = cos(2t) + 1

    4. Substituting for the given t-value:
        5. dw(0)/dt = cos(2(0)) + 1
        dw(0)/dt = 2
    Continued
  • Assume there is a triple-variable function w = f(x, y, z) and double-variable functions x = g(r, s), y = h(r, s), & z = i(r, s), ∂w/∂r & ∂w/∂s can be calculated as shown:
      • ∂w/∂r = (∂w/∂x)(∂x/∂r) + (∂w/∂y)(∂y/∂r) + (∂w/∂z)(∂z/∂r)
      ∂w/∂s = (∂w/∂x)(∂x/∂s) + (∂w/∂y)(∂y/∂s) + (∂w/∂z)(∂z/∂s)
    Example 3:
  • Find a) ∂w/∂r; b) ∂w/∂s; given {w = x + 2y + z2}, x = r/s, y = {r2 + ln(s)} & z = 2r.

      • a)
    • Solving for the first partial derivatives of w:
        • ∂w/∂x = 1; ∂w/∂y = 2; ∂w/∂z = 2z;

    • Finding the differentials of x, y, & z with respect to r:
        • ∂x/∂r = 1/s; ∂y/∂r = 2r; ∂z/∂r = 2;

    • Solving for ∂w/∂r:
        • ∂w/∂r = (∂w/∂x)(∂x/∂r) + (∂w/∂y)(∂y/∂r) + (∂w/∂z)(∂z/∂r)
        ∂w/∂r = (1)(1/s) + (2)(2r) + (2z)(2)
        ∂w/∂r = 1/s + 4r + 4z

        ∂w/∂r = 1/s + 4r + 4(2r)
        ∂w/∂r = 1/s + 12r

      b)
    • Finding the differentials of x, y, & z with respect to s:
        • ∂x/∂s = -r/s2; ∂y/∂s = 1/s; ∂z/∂s = 0;

    • Solving for ∂w/∂s:
        • ∂w/∂s = (∂w/∂x)(∂x/∂s) + (∂w/∂y)(∂y/∂s) + (∂w/∂z)(∂z/∂s)
        ∂w/∂s = (1)(-r/s2) + (2)(1/s)
        ∂w/∂s = 2/s - r/s2
        ∂w/∂s = (2 - r/s)/s
    Example 4
  • Given {w = x2 + y2}, {x = r - s}, & {y = r + s}, determine a) ∂w/∂r; b) ∂w/∂s;

      • a)
    • Solving for the first partial derivatives of w:
        • ∂w/∂x = 2x; ∂w/∂y = 2y;

    • Finding the differentials of x & y with respect to r:
        • ∂x/∂r = 1; ∂y/∂r = 1;

    • Solving for ∂w/∂r:
        • ∂w/∂r = (∂w/∂x)(∂x/∂r) + (∂w/∂y)(∂y/∂r)
        ∂w/∂r = (2x)(1) + (2y)(1)
        ∂w/∂r = 2(x + y)

        ∂w/∂r = 2((r - s) + (r + s))
        ∂w/∂r = 4r

      b)
    • Finding the differentials of x & y with respect to s:
        • ∂x/∂s = -1; ∂y/∂s = 1;

    • Solving for ∂w/∂s:
        • ∂w/∂s = (∂w/∂x)(∂x/∂s) + (∂w/∂y)(∂y/∂s)
        ∂w/∂s = (2x)(-1) + (2y)(1)
        ∂w/∂s = 2(y - x)

        ∂w/∂s = 2((r + s) - (r - s))
        ∂w/∂s = 4s
    Implicit Differentiation
  • Assume a function {w = f(x, y) = 0} & {y = g(x)} are both differentiable, since the derivative of w in terms of x is also equal to zero, then the following must be true:
      • dy/dx = -fx/fy
      dy/dx = -(∂w/∂x)(∂y/∂w)

  • Mixed multivariable functions work similarly. Assume a function {w = f(x, y, z) = 0} & {z = g(x, y)}, the following must be true:
      • dz/dx = -fx/fz; dz/dy = -fy/fz;
      dz/dx = -(∂w/∂x)(∂z/∂w); dz/dy = -(∂w/∂y)(∂z/∂w)
    Example 5:
      Find dy/dx if {y2 - x2 - sin(xy) = 0}.
    1. let w = y2 - x2 - sin(xy). The partial derivatives of w are as follows:
        2. ∂w/∂x = -2x - y cos(xy); ∂w/∂y = 2y - x cos(xy);
        ∂w/∂x = -(2x + y cos(xy)); ∂y/∂w = 1/(2y - x cos(xy));

    2. dy/dx is calculated as shown:
        3. dy/dx = -(∂w/∂x)(∂y/∂w)
        dy/dx = (2x + y cos(xy)) / (2y - x cos(xy))
    Example 6:
  • Find a) ∂z/∂x; b) ∂z/∂y; at the origin if {x3 + z2 + y exz + z cos(y) = 0}

      • a)
    • let w = x3 + z2 + y exz + z cos(y). The 3 partial derivatives of w are:
        • ∂w/∂x = 3x2 + yz exz; ∂w/∂y = exz - z sin(y); ∂w/∂z = 2z + xy exz + cos(y);
        ∂z/∂w = 1/(2z + xy exz + cos(y))

    • ∂z/∂x is determined as follows:
        • ∂z/∂x = -(∂w/∂x)(∂z/∂w)
        ∂z/∂x = -(3x2 + yz exz) / (2z + xy exz + cos(y))

    • Evaluating this derivative at the origin:
        • ∂z(0, 0, 0)/∂x = 0

      b)
    • ∂z/∂y is determined as follows:
        • ∂z/∂y = -(∂w/∂y)(∂z/∂w)
        ∂z/∂y = -(exz - z sin(y)) / (2z + xy exz+ cos(y))

    • Evaluating this derivative at the origin:
        • ∂z(0, 0, 0) = -1

Section 14.7: Extreme Values & Saddle Points

    Extreme Values
  • An extreme value relative extreme is an absolute min or max of a function
  • A local extreme value is a local min or max of a function. For double-variable functions, local maxes are referred to as mountain peaks & local mins are referred to as valley troughs

  • For single variable functions, a local min or max is found by determining one or more horizontal tangent lines. This is found by calculating the first derivative of a function, setting the derivative equal to zero & solving for the resulting independent + dependent value (a point as (x, y))
  • For double-variable functions, a local min or max is found by determining one or more tangent planes. This can be determined similarly as what was done with single-variable functions. The difference is finding every first partial derivative of a function, resulting in 2 independent variables for one dependent variable (giving a point as (x, y, z)). Saddle points can also be found in the process. All local extrema are explained further

  • The First Derivative Test for Double-Variable Functions
      • The point P = (a, b) is a local maximum or minimum if the partial derivatives of f(x, y) are both equal to 0

  • A point where both partial derivatives are equal to zero or one or both partial derivatives do not exist is known as a critical point. This is observed as a horizontal or any vertical tangent plane
      • A critical point where there are domain points of the function all around it is known as a saddle point. Saddle points have a mix of local mins & maxes at the observed point
    Example 1:
      Find the local extrema of {f(x, y) = x2 + y2 - 4y + 9}.


  • The partial derivatives of f(x, y) are:
      • ∂f/∂x = 2x; ∂f/∂y = 2y - 4;

  • Setting the partial derivatives equal to zero and solving for each variable:
      • 0 = 2x; 0 = 2y - 4;
      x = 0; y = 2;
      let Q = (0, 2); This means that this point is a critical point

  • The value of f at point Q is:
      • f(0, 2) = 0 + (2)2 - 4(2) + 9
      f(0, 2) = 5
      Since f(x, y) can never be lower than 5, point Q is a local minimum
    Example 2:
      Find the local extrema (if any) of {f(x, y) = y2 - x2}.

    1. The partial derivatives of f(x, y) are:
        2. ∂f/∂x = -2x; ∂f/∂y = 2y;

    2. Setting the partial derivatives equal to zero and solving for the variables:
        3. 0 = -2x; 0 = 2y;
        x = 0; y = 0;
        let point Q = (0, 0); This means Q is a critical point

    3. The value of f at point Q is:
        4. f(0, 0) = 0;
        In this case, f(x, y) can be greater than 0 for x2 <= y2 & lower than 0 for x2 <= y2. This fact indicates that point Q is neither a local min nor local max, but rather a saddle point, which can be confirmed using the second derivative test
    The Second Derivative Test
  • Suppose f(x, y) and its first & second derivatives are continuous and its first derivatives at point Q = (a, b) are both equal to zero:
      • let the discriminant or Hessian of f = (∂2f(a, b)/∂x2)(∂2f(a, b)/∂y2) - (∂2f(a, b)/(∂y ∂x))2. This expression can also be observed as the cross product of < ∂2f(a, b)/∂x2, ∂2f(a, b)/(∂y ∂x) > & < ∂2f(a, b)/(∂y ∂x), ∂2f(a, b)/∂y2 >

      Point Q is a local maximum if {∂2f(a, b)/∂x2 < 0} && {the discriminant > 0}

      Point Q is a local maximum if {∂2f(a, b)/∂x2 > 0} && {the discriminant > 0}

      Point Q is a saddle point if {the discriminant < 0}

      The test is inconclusive if the discrimant == 0
    Example 3:
      Find the local extreme values of {f(x, y) = xy - x2 - y2 - 2x - 2y + 4}.

    1. The first partial derivatives of f(x, y) are:
        2. ∂f/∂x = y - 2x - 2; ∂f/∂y = x - 2y - 2;

    2. The second partial derivatives of f(x, y) are:
        3. ∂2f/∂x2 = -2; ∂2f/∂y2 = -2; ∂2f/(∂y ∂x) = 1;

    3. Setting the first partial derivative equal to zero and solving for each variable:
        4. 0 = y - 2x - 2; 0 = x - 2y - 2;
        y = 2x + 2; x = 2y + 2;

        x = 2(2x + 2) + 2
        x = 4x + 6
        x = -2;

        y = 2(-2) + 2
        y = -2;

        let point Q = (-2, -2). Point Q is a critical point

    4. The discriminant of f at Q is:
        5. disc(f, Q) = (∂2f(-2, -2)/∂x2)(∂2f(-2, -2)/∂y2) - (∂2f(-2, -2)/(∂y ∂x))2
        disc(f, Q) = (-2)(-2) - (1)2
        disc(f, Q) = 3;
        Since the discriminant of f at point Q is greater than 3, point Q cannot be a saddle point. Additionally, the second partial derivative of f at Q in terms of only x is less than 0. Therefore, it can be concluded that point Q is a local maximum

    5. The value of f at Q is:
        6. f(-2, -2) = (-2)(-2) - (-2)2 - (-2)2 - 2(-2) - 2(-2) + 4
        f(-2, -2) = 4 - 4 - 4 + 4 + 4 + 4
        f(-2. -2) = 8
    Example 4:
      Find the local extreme values of {f(x, y) = 3y2 - 2y3 - 3x2 + 6xy}.

    1. The first partial derivatives of f(x, y) are:
        2. ∂f/∂x = 6y - 6x; ∂f/∂y = 6y - 6y2 + 6x;

    2. The second partial derivatives of f(x, y):
        3. ∂2f/∂x2 = -6; ∂2f/∂y2 = 6 - 12y; ∂2f/(∂y ∂x) = 6;

    3. Solving for critical points:
        4. 0 = 6(y - x); 0 = 6(y - y2 + x)
        y = x; y2 = y + x

        y2 = y + (y)
        y = 0; x = 0;
        y = 2; x = 2;
        critical point Q = (0, 0); R = (2, 2);

    4. The discriminant of f at Q is:
        5. disc(f, Q) = (-6)(6 - 12(0)) - (6)2
        disc(f, Q) = -36 - 36
        disc(f, Q) = -72 < 0
        Since this value is found to be less than zero, critical point Q must be a saddle point

    5. The discriminant of f at R is:
        6. disc(f, R) = (-6)(6 - 12(2)) - (6)2
        disc(f, R) = -6(-18) - 36
        disc(f, R) = 108 - 36
        disc(f, R) = 72 > 0
        Since the discriminant of f at R is greater than zero, it can be concluded that point Q is not a saddle point. The second partial derivative of f in terms of x will always be -6 regardless of the point. This confers that the critical point R is a local max since the value is (-)

    6. The value of f(x, y) at the determined local max R is:
        7. f(2, 2) = 3(2)2 - 2(2)3 - 3(2)2 + 6(2)(2)
        f(2, 2) = 12 - 16 - 12 + 24
        f(2, 2) = 8
    Example 5:
      Find the critical points of function f(x, y) = 10xy e-(x2 + y2) and identify their true nature

    1. The first partial derivative of f(x, y) with respect to x is:
        2. ∂f/∂x = 10y e-(x2 + y2) + 10xy(-2x) e-(x2 + y2)
        ∂f/∂x = 10y e-(x2 + y2) - 20x2y e-(x2 + y2)

    2. The first partial derivative of f(x, y) with respect to y is:
        3. ∂f/∂y = 10x e-(x2 + y2) + 10xy(-2y) e-(x2 + y2)
        ∂f/∂y = 10x e-(x2 + y2) - 20xy2 e-(x2 + y2)

    3. The second partial derivative f(x, y) with respect to solely x is:
        4. ∂2f/∂x2 = -20xy e-(x2 + y2) + (-40xy e-(x2 + y2)) + (-20x2y(-2x) e-(x2 + y2))
        ∂2f/∂x2 = -20xy e-(x2 + y2) - 40xy e-(x2 + y2) + 40x3y e-(x2 + y2)

    4. The second partial derivative of f(x, y) with respect to solely y is:
          5. ∂2f/∂y2 = -20xy e-(x2 + y2) + (-40xy e-(x2 + y2)) + (-20xy2(-2y) e-(x2 + y2))
        ∂2f/∂y2 = -20xy e-(x2 + y2) - 40xy e-(x2 + y2) + 40xy3 e-(x2 + y2)

    5. The mixed second partial derivative of f(x, y) is:
        6. ∂2f/(∂y ∂x) = 10 e-(x2 + y2) + 10y(-2y) e-(x2 + y2) - 20x2 e-(x2 + y2) + (-20x2y(-2y) e-(x2 + y2))
        ∂2f/(∂y ∂x) = 10 e-(x2 + y2) - 20y2 e-(x2 + y2) - 20x2 e-(x2 + y2) + 40x2y2 e-(x2 + y2))

    6. Solving for ∂f/∂x = 0:
        7. 0 = 10y e-(x2 + y2) - 20x2y e-(x2 + y2)
        20x2y = 10y
        2x2 = 1
        x = ±1/sqrt(2)
        y = 0

    7. Solving for ∂f/∂y = 0:
        8. 0 = 10x e-(x2 + y2) - 20xy2 e-(x2 + y2)
        20xy2 = 10x
        2y2 = 1
        y = ±1/sqrt(2)
        x = 0
        critical points = {(0, 0), (1/sqrt(2), 1/sqrt(2)), (-1/sqrt(2), 1/sqrt(2)), (1/sqrt(2), -1/sqrt(2)), (-1/sqrt(2), -1/sqrt(2))}
    Absolute Maximum & Minimum Points
  • Absolute max or mins can be either internal (critical) points or terminal (boundary) points if the function is enclosed with a certain interval or region
  • To find the critical points of a double-variable function, we use the first & second derivative test as described earlier. A critical point may also be considered an absolute min or max in specific cases
  • If the function is bounded by a region, the boundary points are the points located at the ends of the function within the region. These points are solved by comparing all the values of f at every relevant point
    Example 6:
      Find the absolute maximum & minimum values of function {f(x, y) = 2 + 2x + 4y - x2 - y2} bounded by the triangular region made by lines {x = 0, y = 0, y = 9 - x}.

    1. The 3 boundary points are solved by solving for variables using 2 boundary equations at a time:
        2. boundary point A = (0, 0), B = (9, 0), C = (0, 9);

    2. The values of the function at the boundary points are:
        3. f(0, 0) = 2; f(9, 0) = 2 + 2(9) + 4(0) - (9)2 - (0); f(0, 9) = 2 + 2(0) + 4(9)2 - (0) - (9)2
        f(0, 0) = 2; f(9, 0) = 2 + 18 - 81; f(0, 9) = 2 + 36 - 81
        f(0, 0) = 2; f(9, 0) = -61; f(0, 9) = -43;

    3. Solving for the first partial derivatives of f(x, y):
        4. ∂f/∂x = 2 - 2x; ∂f/∂y = 4 - 2y;

    4. Solving for the second partial derivatives of f(x, y):
        5. ∂2f/∂x2 = -2; ∂2/∂y2 = -2; ∂2/(∂y ∂x) = 0;

    5. Determining critical point(s) using the first derivative test:
        6. 0 = 2 - 2x; 0 = 4 - 2y;
        x = 1; y = 2;
        critical point P = (1, 2)

    6. The value of f(x, y) at P is:
        7. f(1, 2) = 2 + 2(1) + 4(2) - (1)2 - (2)2
        f(1, 2) = 12 - 1 - 4
        f(1, 2) = 7;

    7. On the line y = 0, the function is expressed as:
        8. f(x, 0) = 2 + 2x + 4(0) - x2 - (0)
        f(x, 0) = 2 + 2x - x2

    8. Performing the first derivative test for f(x, 0):
        9. df(x, 0)/dx = 2 - 2x

        0 = 2 - x
        x = 1;
        boundary point D = (1, 0);

    9. The value of f(x, y) at D is:
        10. f(1, 0) = 2 + 2(1) - (1)2
        f(1, 0) = 3;

    10. On the line x = 0, the function is expressed as:
        11. f(0, y) = 2 + 2(0) + 4y - (0) - y2
        f(0, y) = 2 + 4y - y2

    11. Performing the first derivative test for f(0, y):
        12. df(0, y)/dy = 4 - 2y;

        0 = 4 - 2y
        y = 2;
        boundary point E = (0, 2);

    12. The value of f at E is:
        13. f(0, 2) = 2 + 4(2) - (2)2
        f(0, 2) = 6;

    13. Along the line y = 9 - x, the function is expressed as:
        14. f(x, 9 - x) = 2 + 2x + 4(9 - x) - x2 - (9 - x)2
        f(x, 9 - x) = 2 + 2x + 36 - 4x - x2 - (x2 - 18x + 81)
        f(x, 9 - x) = 16x - 2x2 - 43

    14. Performing the first derivative test for f(x, 9 - x):
        15. df(x, 9 - x)/dx = 16 - 4x

        0 = 16 - 4x
        x = 4;
        boundary point F = (4, 5);

    15. The value of f at F:
        16. f(4, 5) = 2 + 2(4) + 4(5) - (4)2 - (5)2
        f(4, 5) = 2 + 8 + 20 - 16 - 25
        f(4, 5) = 30 - 41
        f(4, 5) = -11;

    16. Comparing all the values of f at every point {f(0, 0) = 2, f(9, 0) = -61, f(0, 9) = -43, f(1, 2) = 7, f(1, 0) = 3, f(0, 2) = 6, f(4, 5) = -11} it is observed that:
        17. The absolute max of f(x, y) is at point (1, 2) and its absolute min is at point (9, 0)
    Example 7:
      A delivery company only accepts rectangular boxes whose sum of length & girth (perimeter of saggital cross-section) does not exceed 108in. Find the dimensions of an acceptable box at the largest volume.

    1. Setting the variables:
        2. let x = length, y = width, z = height; function g(y, z) = 2(y + z), V(x, y, z) = xyz; plane E = {x + 2y + 2z = 108}

    2. Rewriting the equations:
        3. x = 108 - 2y - 2z

        V(y, z) = (108 - 2y - 2z)yz
        V(y, z) = 108yz - 2y2z - 2yz2

    3. The first partial derivatives of V(y, z) are:
        4. ∂V/∂y = 108z - 4yz - 2z2; ∂V/∂z = 108y - 2y2 - 4yz;

    4. The second partial derivatives of V(y, z) are:
        5. ∂2V/∂y2 = -4z; ∂2V/∂z2 = -4y; ∂2V/(∂z ∂y) = 108 - 4y - 4z;

    5. Performing the first derivative test for both first partial derivatives:
        6. 0 = 108z - 4yz - 2z2; 0 = 108y - 2y2 - 4yz;
        4y + 2z = 108; 4z + 2y = 108;
        z = 54 - 2y; y = 54 - 2z;

        4(54 - 2y) + 2y = 108
        216 - 6y = 108
        y = {0, 18, 54};
        z = 54 - 2(0)
        z = {0, 18, 54};

    6. Since the values of the volume when any variable is zero is zero, the only available point is (18, 18). This can be reinforced by determining the discriminant of V at this point which is found to be a local maximum. Solving for x(18, 18):
        7. x(18, 18) = 108 - 2(18) - 2(18)
        x(18, 18) = 108 - 72
        x = 36;

    7. Therefore, the dimensions that most optimise the volume is when {x = 36, y = 18, z = 18} or point (36, 18, 18). The volume at this max point is:
        8. V(36, 18, 18) = (36)(18)(18)
        V(36, 18, 18) = 11 664in3 = 0.19m3

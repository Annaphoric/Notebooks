Section 14.6: Tangent Planes & Differentials

    Tangent Planes & Normal Lines
  • Tangent planes are made by the linearisation of a triple-variable function w = f(x, y, z) at a point on a level surface (3D cross-section of a 4D graph).
  • Assume there is a point P = (a, b, c) that is located on a level surface made by f(x, y, z) = d. The tangent plane E at this point is given by the equation:
      • E = (∂f(P)/∂x)(x - a) + (∂f(P)/∂y)(y - b) + (∂f(P)/∂z)(z - c) = 0

  • A normal line L is a line through the same point and is parallel to the gradient vector of the function. It is given by the equation:
      • L = {x = a + (∂f(P)/∂x) t, y = b + (∂f(P)/∂y) t, z = c + (∂f(P)/∂z) t}

  • If z can be isolated to simplify the function as z = f(x, y) where ∂f/∂z = 1, the equation for the tangent plane E of the surface at point P = (a, b, c) can be simplified as:
      • E = (∂f(P)/∂x)(x - a) + (∂f(P)/∂y)(y - b) - (z - c) = 0
    Example 1:
  • Find the a) tangent plane; b) normal line; of the level surface (circular paraboloid) made by {f(x, y, z) = x2 + y2 + z = 9} at point Q = (1, 2, 4).

      • a)
    • The partial derivatives of f are:
        • ∂f/∂x = 2x; ∂f/∂y = 2y; ∂f/∂z = 1;
    • Evaulating the partial derivatives at point Q:
        • ∂f(Q)/∂x = 2; ∂f(Q)/∂y = 4; ∂f(Q)/∂z = 1;
    • The evaluated gradient vector of f (F(P))is:
        • F(P) = 2i + 4j + k
    • The tangent plane E can now be written as:
        • E = {2(x - 1) + 4(y - 2) + (z - 4) = 0}
        E = {2x - 2 + 4y - 8 + z - 4 = 0}
        E = {2x + 4y + z = 14}

      b)
    • The normal line is written as:
        • L = {x = 1 + 2t, y = 2 + 4t, z = 1 + t}
    Example 2:
      Find the plane tangent to the surface made by {z = x cos(y) - y ex} at the origin.
    1. let f(x, y) = x cos(y) - y ex. The partial derivatives of f are:
        2. ∂f/∂x = cos(y) - y ex; ∂f/∂y = -(x sin(y) + ex)
    2. Evaluating the partial derivatives at the origin:
        3. ∂f(0, 0, 0)/∂x = 1; ∂f(0, 0, 0)/∂y = -1;
    3. The tangent plane E will have the form:
        4. E = {x - y - z = 0}
    Example 3:
      The cylinder {f(x, y, z) = x2 + y2 = 2} & plane {g(x, y, z) = x + z = 4} make an intersecting ellipse. Point P = (1, 1, 3) exists at the this ellipse. Find the parametric equations for the line tangent to the ellipse at the given point.
    1. The tangent line is orthogonal to both gradient vectors made by the equations. The partial derivatives of f evaluated at point P are:
        2. ∂f(P)/∂x = 2x; ∂f(P) = 2y;
        ∂f(1, 1, 3)/∂x = 2; ∂f(1, 1, 3)/∂y = 2;

    2. The gradient vector of f evaluated at P F(P) is:
        3. F(P) = 2i + 2j

    3. The partial derivatives of g evaluated at point P are:
        4. ∂g(P)/∂x = 1; ∂g(P)/∂z = 1;
        ∂g(1, 1, 3)/∂x = 1; ∂g(1, 1, 3)/∂z = 1;

    4. The gradient vector of g evaluated at P G(P) is:
        5. G(P) = i + k

    5. The tangent line is also parallel to the resulting vector w from the cross product of the 2 evaluated gradient vectors. The cross product of shown as:
        6. w = F ⨯ G
        w = < 2, 2, 0 > ⨯ < 1, 0, 1 >
        w = 2i - 2j - 2k

    6. The components of this vector and the given point can now be used to express the parametrised tangent line L:
        7. L = {x = 1 + 2t, y = 1 - 2t, z = 3 - 2t}
    Estimating Change in a Specific Direction
  • The distance between 2 points of a function f is expressed as ds. If u is the direction from initial point P to terminal point Q, the change in f (df) is given by the formula:
      • df = (F(P) ⋅ u) ds
    Example 4:
      Given {f(x, y, z) = y sin(x) + 2yz}, estimate how much the change in f (df) is if the initial point P = (0, 1, 0) moves 0.1 units towards the terminal point Q = (2, 2, -2).

    1. The partial derivatives of f evaluated at initial point P are:
        2. ∂f(P)/∂x = y cos(x); ∂f(P)/∂y = sin(x) + 2z; ∂f(P)/∂z = 2y;
        ∂f(0, 1, 0)/∂x = 1; ∂f(0, 1, 0)/∂y = 0; ∂f(0, 1, 0)/∂z = 2;

    2. The evaluated gradient vector of f has the form:
        3. F(P) = i + 2k

    3. The vector made between the 2 points (PQ) is:
        4. PQ = < 2, 2 - 1, -2 >
        PQ = 2i + j - 2k

    4. The direction u of PQ is:
        5. u = PQ / ||PQ||
        u = < 2, 1, -2 > / 3
        u = (2/3)i + (1/3)j - (2/3)k

    5. The dot product of the evaluated gradient vector F(P) & the directional unit vector uis:
        6. F ⋅ u = < 1, 0, 2 > ⋅ < 2/3, 1/3, -2/3 >
        F ⋅ u = 2/3 - 4/3
        F ⋅ u = -2/3

    6. Finally, the change in f (df) from the initial point to the terminal point can be evaluated by multiplying the result of the dot product with the given distance between the points (ds):
        7. df = (F ⋅ u) ds
        df = (-2/3)(1/10)
        df = -2/30
    Linearisation of a Double-Variable Function
  • To approximate an unknown value of a differentiable function z = f(x, y) at a point Q = (c, d), a given nearby point P = (a, b) is chosen to approximate f(c, d) in the form of a plane. As long as the distance between the 2 points is arbitrarily small, the linear approximation L(x, y) of the function at the given point (for nearby points) is:
      • f(c, d) ≈ L(x, y) = f(a, b) + (∂f(a, b)/∂x)(x - a) + (∂f(a, b)/∂y)(y - b)
      L(x, y) is the plane tangent to the surface made by z = f(x, y) at the point P. This is analagous to the line made by linearisation of a single-variable function using a relative point
    Example 5:
      Find the linearisation of {f(x, y) = x2 - xy + y2/2 + 3} at point P = (3, 2).

    1. Evaluating the function at the given point:
        2. f(3, 2) = 9 - 6 + 2 + 3
        f(3, 2) = 8

    2. The partial derivatives of f at point P are:
        3. ∂f(P)/∂x = 2x - y; ∂f(P)/∂y = y - x;
        ∂f(3, 2)/∂x = 4; ∂f(3, 2)/∂y = -1;

    3. The linearisation of the function at point P is written as:
        4. L(x, y) = f(a, b) + (∂f(a, b)/∂x)(x - a) + (∂f(a, b)/∂y)(y - b)
        L(x, y) = 8 + 4(x - 3) - (y - 2)
        L(x, y) = 4x - 12 - y + 2 + 8
        L(x, y) = 4x - y - 2
    Standard Linear Approximation Error
  • Let M = the upper bound (highest value) of the absolute values of the 2nd-order partial derivatives of a function f(x, y). The error E(x, y) of performing a linear approximation can be bounded within a rectangle with a centre point P = (a, b). This value is expressed as:
      • E(x, y) = |f(x, y) - L(x, y)|
      E(x, y) <= ±(M(|x - a| + |y - b|)2) / 2
    Differentials of Multivariable Functions
  • For single-variable functions like y = f(x), the change in f (Δf) indicates the change in the value of the function as x changes from a to {a + Δx}:
      • Δf = f(a + Δx) = f(a)
  • The differential of f is written as:
      • df = f'(a) Δx

  • The same concept applies for multivariable functions. Assume z = f(x, y) is differentiable and its partial derivatives exist at a point P = (a, b). When moving from this point to a nearby point, the change in f (Δf) is respresented as:
      • Δf = f(a + Δx, b + Δy) - f(a, b)
  • Applying linearisation to the function at the given point, the change in linearisation (ΔL) is demonstrated as:
      • ΔL = L(a + Δx, b + Δy) - L(a, b)
      ΔL = (∂f(a, b)/∂x)Δx + (∂f(a, b)/∂y)Δy
  • The total differential (d_) of a function is defined as its linearisation from a given point to a nearby point:
      • df = (∂f(a, b)/∂x) dx + (∂f(a, b)/∂y) dy
    Example 6:
      Suppose a cylindrical can is expected to have a radius r = 1in & height h = 5in. When making the can, it is found that the radius & height are off by dr = +0.03 & dh = -0.1. Estimate the absolute change in the volume of making this can relative to its blueprint.

    1. The volume of a cylinder is given by the formula V = πr2h. The partial derivatives of V(r, h) in terms of r & h are:
        2. ∂V/∂r = 2πrh; ∂V/∂h = πr2;

    2. Evaluating the partial derivatives according to the blueprint values:
        3. ∂V(1, 5)/∂r = 10π; ∂V(1, 5)/∂h = π;

    3. The change in volume of the actual can relative to the blueprint values is most approximated to be the value of its absolute differential (dV)
        4. ΔV ≈ dV = (∂V(1, 5)/∂r)dr + (∂V(1, 5)/∂h)dh
        dV = (10π)(0.03) + (π)(-0.01)
        dV = 0.29π in
    Example 7:
      My company manufactures stainless steel circular cylindrical molasses storage tanks that are 25ft high and have a radius of 5ft. How sensitive are the tanks' volumes to small variations in height & radii?

    1. let r = 5, h = 25; The partial derivatives of V(r, h) evaluated at the given radius & height are:
        2. ∂V(r, h)/∂r = 2πrh; ∂V(r, h)/∂h = πr2;
        ∂V(5, 25)/∂r = 250π; ∂V(5, 25)/∂h = 25π;

    2. The absolute differential dV is calculated as:
        3. dV = (∂V(5, 25)/∂r)dr + (∂V(5, 25)/∂h)dh
        dV = 250π dr + 25π dh
        Based on my result, a change in radius would cause a more drastic affect in volume (by a factor of 250π) compared to the same change in height (by only a factor of 25π). A large partial derivative indicates a more sensitive change in an independent variable. Nevertheless, if the original blueprint value of height was higher by a certain amount, the change in height would be more sensitive than the radius. Therefore, it can be concluded that the sensitivity of the volume based on its radius & height is more affected by the original values, rather than the change in independent variables
    Linearisation & Differentials of Triple-Variable Functions & Beyond
  • The linear approximation L(x, y, z) of f(x, y, z) at a P = (a, b, c) to a nearby point is:
      • L(x, y, z) = f(P) + (∂f(P)/∂x)(x - a) + (∂f(P)/∂y)(y - b) + (∂f(P)/∂z)(z - c)

  • The error E created when making this approximation is based on the upper bound (highest value) of any 2nd partial derivative of f (assuming they are continuous) at a point P = (a, b, c). The error is given by:
      • E(x, y, z) = |f(x, y, z) - L(x, y, z)|
      E(x, y) <= ±(M(|x - a| + |y - b| + |z - c|)2) / 2

  • The absolute, total differential of f(x, y, z) is calculated as:
      • df = (∂f(P)/∂x) dx + (∂f(P)/∂y) dy + (∂f(P)/∂z) dz
    Example 8:
  • Find the linearisation L(x, y, z) of {f(x, y, z) = x2 - xy + 3 sin(z)} at point Q = (2, 1, 0).

      • The function at the given point has value of:
  • f(2, 1, 0) = 4 - 2
  • f(2, 1, 0) = 2

      • The partial derivatives of f evaluated at Q are:
  • ∂f(x, y, z)/∂x = 2x - y; ∂f(x, y, z)/∂y = -x; ∂f(x, y, z)/∂z = 3 cos(z);
  • ∂f(2, 1, 0)/∂x = 3; ∂f(2, 1, 0)/∂y = -2; ∂f(2, 1, 0)/∂z = 3;

  • The linearisation of the function is expressed as:
      • L(x, y, z) = 2 + (3)(x - 2) + (-2)(y - 1) + (3)(z)
      L(x, y, z) = 3x - 2y + 3z + 2 - 6 + 2
      L(x, y, z) = 3x - 2y = 3z - 2

Section 14.1: Multivariable Functions

    Introduction
  • A function having several variables has a domain expressed as an ordered pair (tuple) containing input variables. The range of this function is the output variable when an input is given. For example, a function F with a domain (x, y) has a range of F(x, y); if is given the input tuple (a, b), its output will be f(a, b)
  • An interior point is a circle of points that satisfies a function within its domain. The set of every interior point makes up the domain of a function. A boundary point is a circle of points that contains points that both satisfy and does not satisfy a function
  • A region is considered open if it contains no boundaries and considered closed if it does
    Example 1:
  • Find the domains & ranges of the given functions a) z = sqrt(y - x2); b) z = 1/(xy); c) z = sin(xy); d) w = sqrt(x2 + y2 + z2); e) w = 1/(x2 + y2 + z2); f) w = xy ln(z);

      • a)
    • Domain: (y >= x2)
    • Range: [0, ∞)

      b)
    • Domain: (x != 0 || y != 0) (xy != 0)
    • Range: (-∞, ∞)

      c)
    • Domain: (-∞, ∞) (entire plane; R2)
    • Range: [-1, 1]

      d)
    • Domain: (-∞, ∞) (entire space; R3)
    • Range: [0, ∞)

      e)
    • Domain: (x, y, z) != (0, 0, 0)
    • Range: (0, ∞)

      f)
    • Domain: (z > 0) (half space)
    • Range: (-∞, ∞))
    Example 2:
  • Describe the domain of function f(x, y) = sqrt(y - x2).

      • This function has an interior made from the set (y >= x2) or (y - x2 >= 0). Its boundary is made by the set (y = x2) or (y - x2 = 0). The exterior is made from the set (y <= x2) or (y - x2 <= 0)
    Graphs, Level Curves & Contours of Double Variable Functions
  • A double variable function has 2 independent variables for one dependent variable
  • The set of points (x, y) on a plane that satisfies the function f at a constant value c is a level curve of f at c shown as f(x, y) = c
  • The set of points (x, y, f(x, y)) in space within the domain of f is the graph of f. It is also referred to as the surface of z where z = f(x, y)
  • A contour map is given by the graphs plane made by z = c. It is always parallel to the level curve
    Example 3:
  • Describe the graph z = f(x, y) where f(x, y) = 100 - x2 - y2.
      • The domain of this graph is all real numbers for both x & y ((-∞, ∞) or R2). The range is (-∞, 100]
      The graph of z = 100 - x2 - y2 is a paraboloid
      A level curve of zero (f(x, y) = 0) is expressed as:
    • 100 - x2 - y2 = 0
    • x2 + y2 = 100
    • r = 10 units
      A level curve of 51 (f(x , y) = 51) is expressed as:
    • 100 - x2 - y2 = 51
    • x2 + y2 = 49
    • r = 7 units
      A level curve of 75 (f(x, y) = 75) is expressed as:
    • 100 - x2 - y2 = 75
    • x2 + y2 = 25
    • r = 5 units
    Graphs of Triple Variable Functions
  • A triple variable function has 3 independent variables for one dependent variable
  • The set of points (x, y, z) in space that satisfy the function f(x, y, z) at a constant value c is a level surface of f at c shown as f(x, y, z) = c
  • The set of points (x, y, z, f(x, y)) in space within the domain of f is the graph of f, which happens to exist in a 4-dimensional space. It is also referred to as the surface of w where w = f(x, y, z)
  • The definitions of a triple-variable function are shared with general multivariable functions
    Example 4:
  • Describe the level surfaces of the function f(x, y, z) = sqrt(x2 + y2 + z2).
      • A level surface of f exists at any constant value c where f(x, y, z) = c. The surface is a sphere on the xyz-coordinate graph that has a radius of c

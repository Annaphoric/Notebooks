Section 15.3: Area by Double Integration

    Area of Bounded Regions for Planes
  • Using the plane f(x, y) = 1, the area A of the bounded region R for the plane is given by the integral:
      • A = ∬RdA
    Example 1:
      Find the area of the region R bounded by y = x && y = x2 in the first quadrant.

    1. Solving for the point of intersection:
        2. x2 = x
        x = {0, 1}

        y(0) = 0; y(1) = 1;
        point P = (0, 0), Q = (1, 1)

    2. Based on the points found, their x-coordinates correspond to the line in which intersects the points parallel to the x-axis. These are the lines x = 0 & x = 1; these are used as the bounds of the integral in terms of x. Within this bound, y = x2 is seen to be the lower curve & y = x is the upper curve. Therefore, the area of the region is:
        3. A = ∫01(∫x2x(1) dy) dx
        A = ∫01(x - x2) dx
        A = [x2/2 - x3/3]01
        A = 1/2 - 1/3
        A = 1/6
    Example 2:
      Find the area A of the region R enclosed by the parabola y = x2 & the line y = {x + 2}.

    1. The points of intersection of the 2 given curves are:
        2. x2 = x + 2
        x2 - x - 2 = 0
        (x - 2)(x + 1) = 0
        x = {-1, 2}

        y(-1) = 1; y(2) = 4;
        point P = (-1, 1), Q = (2, 4)

    2. Based on these points, the bound for the integral in terms of y is x = -1 & x = 2. Within this interval, y = x2 is the lower curve & y = {x + 2} is the upper curve. The total area using this info is:
        3. A = ∫-12(∫x2x + 2(1) dy) dx
        A = ∫-12(x + 2 - x2) dx
        A = [x2/2 + 2x - x3/3]-12
        A = ((2)2/2 + 2(2) - (2)3/3) - ((-1)2/2 + 2(-1) - (-1)3/3)
        A = (2 + 4 - 8/3) - (1/2 - 2 + 1/3)
        A = 8 - 3 - 1/2
        A = 9/2
    Example 3:
  • Find the area of the playing field given by the region R = {x ∈ [-2, 2], y ∈ [-(1 + sqrt(4 - x2)), 1 + sqrt(4 - x2)]}.

      • The area can be solved using Fubini's Theorem of Double Integrals:
    • A = ∫-22(∫-(1 + sqrt(4 - x2))1 + sqrt(4 - x2)(1) dy) dx
    • A = ∫-22(2(1 + sqrt(4 - x2))) dx
    • A = 2(2 - (-2)) + 2 ∫-22(sqrt(4 - x2)) dx
    • A = 8 + 2 ∫-22(4 - x2) dx

    • let θ = asin(x/2), x = 2 sinθ, dx = 2 cosθ dθ (trig substitution)
    • A = 8 + 2 ∫-22(sqrt(4 - 4 sin2θ)(2 cosθ)) dθ
    • A = 8 + 8 ∫-22(sqrt(cos2θ) cosθ) dθ
    • A = 8 + 8 ∫-22(cos2θ) dθ
    • A = 8 + 4 ∫-22(1 + cos(2θ)) dθ

    • A = 8 + 4 [θ + sin(2θ)/2]-22
    • A = 8 + 4 [asin(x/2) + sin(2 asin(x / 2))]-22
    • A = 8 + 4(asin(1) - asin(-1)) + 4(sin(2 asin(1)) - sin(2 asin(-1)))
    • A = 8 + 4(π/2 + π/2) + 4(0 - 0)
    • A = 8 + 4π

      Another way to solve for the area is by observing the sketch made by the regions and calculating the areas of the individuals shapes that are distinguishable. Based on the sketch, a rectangle & 2 semicircles make up the graph. The rectangle is seen to have a length of 4 & height of 2 and the circle has a radius of 2. The area of the entire region is:
    • Atotal = Arectangle + Acircle
    • Atotal = (4)(2) + π(2)2
    • Atotal = 8 + 4π
    Average Values of Double Integrals
  • The average value of a function integrated within a bound is the value of the intergral divided by the area of the bounded region
    Example 4:
      Find the average value of function f(x, y) = x cos(xy) over the bounded rectangle R = {x ∈ [0, π] y ∈ [0, 1]}.

    1. The value A of the function integrated within the bound is:
        2. A = ∫0π(∫01(x cos(xy)) dy) dx
        A = ∫0π([sin(xy)]01) dx
        A = ∫0π(sin(x)) dx
        A = [cos(x)]π0
        A = 2

    2. The area of the bounding rectangle R:
        3. Arectangle = π

    3. Dividing the value of the function by the area of the bound:
        4. Avg = 2 / π

Section 15.8: Substitutions in Multiple Integrals

    Substitutions in Double Integrals
  • In the case of a complicated double integral, the substitution method can be used to simplify the integral, whether it be the integrand, limits of integration, or both. A region R originally in the Cartesian xy-coordinate graph can be transformed into a region G in a Cartesian uv-coordinate graph. A function f(x, y) bounded within R creates the final image relative to G. By setting x equal to a function g(u, v) and y equal to a function h(u, v), a pre-image of G can be formed relative to R. Based on this information, the function can now be written as f(g(u, v), h(u, v)). Since this function now includes 4 partial derivatives (2 for g(u, v) & 2 for h(u, v)), there must be a simpler way to express all 4 in one integrand. This is the role of the Jacobian determinant (J(u, v)) which is the cross product of the 2 gradient vector functions:
      • gradient-vector G = (∂x/∂u)i + (∂x/∂v)j; H = (∂y/∂u)i + (∂y/∂v)j
      J(u, v) = G ⨯ H
      J(u, v) = < ∂x/∂u, ∂x/∂v > ⨯ < ∂y/∂u, ∂y/∂v >
      J(u, v) = (∂x/∂u)(∂y/∂v) - (∂x/∂v)(∂y/∂u)
      J(u, v) = ∂(x, y)/∂(u, v)

  • The Jacobian determinant is viewed as a scalar multiple or scaling factor of region G to achieve the differential area that is present in the region R. The transformation of the integration of f(x, y) can now be written as:
      • ∬R(f(x, y)) dx dy = ∬G(f(g(u, v), h(u, v)) J(u, v)) du dv
      This is analagous to the using the chain rule on a single variable function f(x) where x = g(u):
    • ∫g(a)g(b)(f(x)) dx = ∫ab(f(g(u)) g'(u)) du
    Example 1:
      Find the Jacobian determinant for converting a Cartesian function into a polar function. Then, express the integral {∬R(f(x, y)) dx dy} as a polar integral within region G.

    1. Using the Cartesian-to-polar equations {x = r cosθ, y = r sinθ}, the gradient vector functions can be defined:
        2. ∂x/∂r = cosθ; ∂x/∂θ = -r sinθ; ∂y/∂r = sinθ; ∂y/∂θ = r cosθ;
        gradient G = cosθ i - (r sinθ)j; H = sinθ i + (r cosθ)j

    2. The Jacobian determinant J(r, θ) is the cross product of the 2 gradient vectors:
        3. J(r, θ) = G ⨯ H
        J(r, θ) = < cosθ, -r sinθ > ⨯ < sinθ, r cosθ >
        J(r, θ) = r cos2θ + r sin2θ
        J(r, θ) = r

    3. The Cartesian integral can now be converted to a polar one:
        4. ∬R(f(x, y)) dx dy = ∬G(f(r cosθ, r sinθ) r) dr dθ
    Example 2:
      Transform and evaluate {∫04(∫y/2y/2 + 1((2x - y)/2) dx) dy} by using the following uv-coordinate equations: {u = (2x - y)/2, v = y/2}.

    1. To find the Jacobian determinant, the uv-coordinate equations must be converted to equations of x and y in terms of u & v:
        2. 2u = 2x - y; 2v = y;
        x = (2u + y)/2; y = 2v;
        x(u, v) = u + v; y(u, v) = 2v;

Section 13.1: Curves & their Tangents in Space

    Curves in Space
  • A curve C(t) in space is given by 3 parametrised equations that make up the points of the curve according to the x, y, & z axes. The equations are calculated as functions in terms of t (time) within a time interval I as shown:
      • x = f(t); y = g(t); z = h(t)
  • Therefore, the curve will have the points:
      • C(t) = (x, y, z)
      C(t) = (f(t), g(t), h(t)) for t ∈ I
  • These set of points can also be written as a vector r(t) originating from the origin and terminating at the curve:
      • r(t) = f(t)i + g(t)j + h(t)k
      r(t) = < f(t), g(t), h(t) >

  • Since these functions are part of the vector components of r, they are known as component vectors. They are also classified as scalar functions or real-valued functions since their domain of t is all real numbers. Functions with domains that are vectors or regions in space like r itself are vector functions or vector-valued functions
    Example 1:
      Describe the graph made by the vector function c(t) = cos(t)i + sin(t)j + tk.

    1. The graph made by this curve is an upwards helix and is bound by the circle (cylinder) {x2 + y2 = 1}. In order to satisfy the curve properly, we can then assume that:
        2. x = cos(t); y = sin(t), z = t

    2. These equations happen to be the parametrised equations for c for t ∈ (-∞, ∞). Every time t increases by 2π, the curve makes one iteration upwards
    Limits & Continuity
  • Limits of vector functions are taken much like the limits of scalar functions
      • Given r(t) = {f(t)i + g(t)j + h(t)k} and the convergent limit L:
    • limt ➝ t0(r(t)) = L
    • limt ➝ t0(r(t)) = (limt ➝ t0(f(t)))i + (limt ➝ t0(g(t)))j + (limt ➝ t0(h(t)))k
  • A vector function is continuous if limt ➝ a(r(t)) = r(a) for a ∈ (-∞, ∞)
    Example 2:
      Solve limt ➝ π/4(r(t)) where r(t) = cos(t)i + sin(t)j + tk

    1. The same limit is applied to each component function:
        2. < limt ➝ π/4(cos(t)), limt ➝ π/4(sin(t)), limt ➝ π/4(t) >
        < 1/sqrt(2), 1/sqrt(2), π/4 >
    Derivatives & Motion
  • Assume vector function c(t) = f(t)i + g(t)j + h(t)k is the position vector of a particle moving along a curve in space and each component function is differentiable, the change between 2 points on the curve is as shown:
      • Δc = dc = c(t + Δt) - r(t)
      Δc = (c(t + Δt)) - (f(t)i + g(t)j + h(t)k)
      Δc = dc = dc = (f(t + Δt) - f(t))i + (g(t + Δt) - g(t))j + (h(t + Δt) - h(t))k

  • Dividing both sides by Δt and taking the limit as the this value approaches 0:
      • limdt ➝ 0((dc)/(dt)) = limdt ➝ 0((dc)/(dt)) = ((df)/(dt))i + ((dg)/(dt))j + ((dh)/(dt))k
    = (f(t + Δt)i + g(t + Δt)j + h(t + Δt)k) - (r(t))
  • As long as the curve is smooth (no breaks, discontinuity or sharps points), dc/dt can never equal zero
  • The vector c'(t) = dc/dt is the tangent line of the curve in space c(t) made at the position t. Regardless of the sign of dc, if dt is (+), the vector points forwards; if dt is (-), the vector points backwards

  • Assuming t = time & r = the position vector of a smooth curve in space:
      • Velocity v is the differential of the position in terms of time:
    • v(t) = dr/dt
      Speed s is the magnitude of the velocity:
    • s = ||v||
      Acceleration a is the differential of the velocity in terms of time:
    • a = dv/dt = d2r/dt2
      The direction of motion D (unit vector) is the direction of the velocity:
    • D = v / ||v||
    Example 4:
  • Determine the a) velocity; b) speed; c) acceleration; d) vector velocity at t = 7π/4; of a particle whose motion in space is given by the position vector r(t) = 2cos(t)i + 2sin(t)j + 5cos2(t)k.

      • a)
    • v(t) = dr/dt
    • v(t) = (d(2cos(t))/dt)i + (d(2sin(t))/dt)j + (d(5cos2(t))/dt)k
    • v(t) = -2sin(t)i + 2cos(t)j - 10cos(t)sin(t)k
    • v = -2sin(t)i + 2cos(t)j - 5sin(2t)k

      b)
    • s = ||v||
    • s = ||< -2sin(t), 2cos(t), -5sin(2t) >||
    • s = sqrt(4sin2(t) + 4cos2(t) + 25sin2(2t))
    • s = sqrt(4 + 25sin2(2t))

      c)
    • a(t) = dv/dt
    • a(t) = -2cos(t)i - 2sin(t) - 10cos(2t)

      d)
    • v(7π/4) = -2sin(7π/4)i + 2cos(7π/4)j - 5sin(2(7π/4))k
    • v(7π/4) = sqrt(2)i + sqrt(2)j + 5k
    • s(7π/4) = ||v(7π/4)|| = sqrt(29)
    Differentiation Rules for Vector Functions
  • let u & v = any differentiable vector function in terms of time t, C = any constant vector, c = any scalar multiple, & f = any scalar function in terms of time t:
      • d(C)/dt = 0 (a vector in any direction with a magnitude of zero)

      d(cu(t))/dt = cu'(t)

      d(f(t)u)/dt = f'(t)u(t) + f(t)u'(t)

      d(u(t) ± v(t))/dt = u'(t) ± v'(t)

      d(u(f(t)))/dt = f'(t)u'(f(t))

      d(u(t) ⋅ v(t))/dt = (u'(t) ⋅ v(t)) + (u(t) ⋅ v'(t))

      d(u(t) ⨯ v(t))/dt = (u'(t) ⨯ v(t)) + (u(t) ⨯ v'(t)) (order is significant)
    Vector Functions of Constant Length
  • Given a vector function r of constant length/magnitude (like from the centre of a sphere), its tangent vector r' is always perpendicular (Their dot product is equal to zero). This is expressed as:
      • d(r(t) ⋅ r(t)) = d(||r(t)||2)/dt
      d(r(t) ⋅ r(t)) = d(c2)/dt
      d(r(t) ⋅ r(t)) = 0
      r(t) ⋅ d(r)/dt = 0

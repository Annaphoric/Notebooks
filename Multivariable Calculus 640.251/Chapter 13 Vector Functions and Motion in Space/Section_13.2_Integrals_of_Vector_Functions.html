Section 13.2: Integrals of Vector Functions

    Indefinite Integrals
  • A differentiable vector function R(t) is an antiderivative of r(t) if r(t) = dr/dt within an interval I. The set of all antiderivatives including their constant vector C of r is the indefinite integral of r within I. This is represented as:
      • ∫(r(t))dt = R(t) + C

  • To integrate a vector function, each one of its components must be individually integrated and their sum is taken. Since each component integration results in a constant for each corresponding standard unit vector, a separate constant vector can be specified along with the derived sum instead of writing the 3 scalar constants
    Example 1:
      Solve ∫(< cos(t), 1, -2t >)dt.

    1. The integration is demonstrated as follows:
        2. ∫(< cos(t), 1, -2t >)dt = ∫(cos(t)dt)i + ∫(dt)j - 2∫(t dt)k
        ∫(< cos(t), 1, -2t >)dt = sin(t)i + tj - t2k + C
    Definite Integrals
  • Assume r = f(t)i + g(t)j + h(t)k and R is an antiderivative of r, the definite integral of r(t) over the interval [a, b] is written as follows:
      • ∫ab(r(t)dt) = ∫ab(f(t)dt)i + ∫ab(g(t)dt)j + ∫ab((h(t)dt)k
      ∫ab(r(t)dt) = R(b) - R(a)
    Example 2:
      Solve ∫0π((cos(t)i + j - 2tk)dt).

    1. The antiderivative from the previous example can be used here and the specified interval can be applied:
        2. ∫0π((cos(t)i + j - 2tk)dt) = ([sin(t)]0π)i + ([t]0π)j - ([t2]0π)k
        ∫0π((cos(t)i + j - 2tk)dt) = (0 - 0)i + (π - 0)j + (π2 - 0)k
        ∫0π((cos(t)i + j - 2tk)dt) = πj + π2k
    Example 3:
      Suppose the acceleration vector function of a hang glider is a(t) = < -3cos(t), -3sin(t), 2 >. The glider initially (at time t = 0) departed from point P = (4, 0, 0) with an initial velocity vector of v(0) = < 0, 3, 0 >. Find the glider's position vector r(t).

    1. The velocity vector v(t) is solved by taking the indefinite integral of a(t):
        2. v(t) = ∫(a(t)dt)
        v(t) = ∫(< -3cos(t), -3sin(t), 2 > dt)
        v(t) = -3sin(t)i + 3cos(t)j + 2tk + C

    2. Since v(0) is given, the constant vector C for the velocity vector function can be solved:
        3. C = v(t) - (-3sin(t)i + 3cos(t)j + 2tk)
        C = v(0) + (3sin(0)i - 3cos(0)j - 2(0)k)
        C = < 0, 3, 0 > + < 0, -3, 0 >
        C = < 0, 0, 0, >
        C = 0

    3. The true velocity vector function is written as:
        4. v(t) = -3sin(t)i + 3cos(t)j + 2tk + 0
        v(t) = -3sin(t)i + 3cos(t)j + 2tk

    4. The position vector function r(t) is solved by taking the indefinite integral of v(t):
        5. r(t) = ∫(v(t)dt)
        r(t) = ∫(< -3sin(t), 3cos(t), 2t > dt)
        r(t) = 3cos(t)i + 3sin(t)j + t2k + D

    5. The point P = (4, 0, 0) can be represented as the initial position vector:
        6. r(0) = < 4, 0, 0 >

    6. The constant vector D for the position vector function can now be solved using r(t)
        7. D = r(t) - (3cos(t)dti + 3sin(t)j + t2k)
        D = r(0) + < -3cos(0), -3sin(0), (0)2 >
        D = < 4, 0, 0 > + < -3, 0, 0 >
        D = < 1, 0, 0 >
        D = i

    7. The true position vector function is written as:
        8. r(t) = 3cos(t)i + 3sin(t)j + t2k + (i)
        r(t) = (3cos(t) + 1)i + 3sin(t)j + t2k
    Application to Projectile Motion
  • The following assumptions can be made for any object thrown on earth as long as the only force acting on the object is gravity:
      • The initial position r(0) is always at the origin:
    • r(0) = 0
      The initial velocity v(0) is given by the equation:
    • v(0) = (||v(0)||cosθ)i + (||v(0)sinθ||)j
    • ||v(0)|| = initial speed
      The acceleration a is constant for any moment in time:
    • a(t) = d2r/dt2 = -gj
    • g = force due to gravity (9.86m/s2)
      Integrating the acceleration vector function gives the equation for the velocity at any point in time:
    • v(t) = dr/dt = -gtj + v(0)
      A second integration leads to the equation for the position at any point in time:
    • r(t) = -(gt2/2)j + tv(0) + r(0)
    • r(t) = (t||v(0)||cosθ)i + (t||v(0)sinθ|| - (gt2/2))j
      The formula for the max height of the projectile is given as:
    • ymax = (||v(0)||sinθ)2 / (2g)
      The formula for the flight time (total amount of time that projectile is in air):
    • tflight = 2||v(0)||sinθ / g
      The formula for the range (distance between the origin & location of where the projectile hit the ground):
    • xrange = ||v(0)||2sin(2θ) / g
    Example 4:
      A projectile is fired from the origin over the horizontal ground at an initial speed of 500m/s and a launch angle of 60°. Where will the projectile be 10s later?

    1. The variables can be set as shown:
        2. ||v(0)|| = 500m/s; θ = 60°; t = 10s; g = 9.86m/s2

    2. Using the equation for the position of projectiles in motion:
        3. r(10) = ((10)(500)cos(π/3))i + ((10)(500)sin(π/3) - (9.86)(10)2)j
        r(10) ≈ 2500i + 3840j

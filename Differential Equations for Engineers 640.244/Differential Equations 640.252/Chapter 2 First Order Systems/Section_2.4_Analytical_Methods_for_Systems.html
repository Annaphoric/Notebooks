Section 2.4: Analytical Methods for Special Systems

    Checking Solutions
  • Solutions to a system of DEs can be checked by simple derivation & substitution

  • let SoE {dx/dy = f(x, y); dy/dt = g(x, y)}, solution vector Y(t) = (x(t), y(t)), vector field F(x, y) = (f(x, y), g(x, y)):
      • A solution Y0 = (x0, y0) to the DE would be one that satisfies both differentials of the system:
    • (f(x0, y0) == x0') && (g(x0, y0) == y0') must be true
    • This solution satisfies an initial condition Y(t0) = (a, b) only if Y(t0) = (x(t0), y(t0)) == (a, b) is true
    Decoupled Systems
  • A system is said to be decoupled if the rate of change for a dependent variable depends only on the dependent variable itself
  • A system is completely decoupled if every equation within the system depends only on its respective variable
  • A system is partially decoupled if at least one but not all of the equations depends only on its respective variable

  • The solution to any completely decoupled system {dx/dt = f(x), dy/dt = g(y)} is Y(t) = (x(t), y(t)) = (c ef'(x) t, d eg'(y) t), where c & d are constants that satisfy a given IC
  • The solution to a partially decoupled system is based on the Extended Linearity Principle which says that the general solution to a nonhomogenous (coupled) equation is the sum of its associated homogenous equation (decoupled) & one specific solution to the nonhomogenous equation
      • To solve for these, the solution to the decoupled equation will first have to found and substituted into the coupled equation
      a-substitution is then used to solve for one solution to the coupled equation
    Example 1:
      Given SoE {dx/dt = 2x - y; dy/dt = x - 2y}, determine if solution vector Y(t) = (e-t, 3e-t) satisfies IC Y(0) = (1, 3).
    1. Showing that the solution vector satisfies the system:
        2. dY/dt = (-e-t, -3e-t)

        dx/dt (e-t, 3e-t) = 2(e-t) - (3e-t) == -et
        -e-t = -e-t (True)

        dy/dt (e-t, 3e-t) = (e-t) - 2(3e-t) == -3e-t
        -5e-t == -3e-t (False)
        Since this equality leads to a false statement, the IC cannot even be considered. The given solution is not a solution to the DE

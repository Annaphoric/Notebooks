    Euler's Method
  • Euler's Method, as described previously, is a method for approximating the graph of a solution using line segments whose slopes are equal to a value of the DEs

  • let first-order SoE {dx/dt = f(x, y); dy/dt = g(x, y)}, solution vector Y(t) = (x(t), y(t)), vector function F(x, y) = (f(x, y), g(x, y))
  • Subsequent points (xk + 1, yk + 1) are calculated as follows:
      • (xk + 1, yk + 1) = (xk, yk) +^ F(xk, yk) Î”t, Note: "+^" is showing vector addition
      (xk + 1, yk + 1) = (xk + f(xk, yk), yk + g(xk, yk))
    Van der Pol Equation
  • The Van der Pol equation is a second-order DE given by:
      • d2/dt2 - (1 - x2) dx/dt + x = 0
  • This equation can be coverted to a first-order SoE where y = dx/dt:
      • {dx/dt = y; dy/dt = (1 - x2) - x}
  • Euler's Method can be applied to this equation as described earlier

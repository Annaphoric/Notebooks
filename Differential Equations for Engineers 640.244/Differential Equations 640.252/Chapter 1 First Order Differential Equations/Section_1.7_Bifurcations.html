Section 1.7: Bifurcations

    Parameters
  • Parameters are quantities that do not that do not depend on the independent variable (time)
  • Usually, they are expressed as a constant of proportionality which is specific to the type of independent variable rather than the quantity
  • Solutions to a DE may change as the parameter varies
      • The degree to how much the DE changes depends on the type of parameter; sometimes a small change in the parameter can lead to a drastic change in the long-term behaviour of solutions
      This change is known as a bifurcation

  • let autonomous DE dy/dt = {g(y) + μ}, where μ = parameter
  • The notation for a DE dependent on the parameter is written as:
      • fμ(y) = y2 - 2y + μ, where f represents the DE

  • A one-parameter family is the general DE written for any value of μ

  • When changing the value of a parameter within a specific interval, the relative position & type of equillibrium point does not change, rather the intensity of the long-term behaviour
  • A drastic change (bifurcation) will occur if the parameter reaches beyond the interval into a new interval
  • The value of the parameter which divides the 2 intervals is the bifurcation value
      • The bifurcation value is almost always a node (solutions increase both above && below the equillbrium point)

  • A bifurcation diagram is a graph plotting y over μ
    Bifurcations
  • Generally, bifurcations do not occur
  • In other words, a small change in the parameter leads only to a small change in the behaviour of the solution

  • A bifurcation only occurs for values of μ where the partial derivative of the DE in terms of y is equal to zero
  • let dy/dt = fμ(y) = g(y) + μ;
      • f'μ(y) = g'(y)
      let y0 be a value which satisfies f'μ(y0) = 0
      A bifurcation value exists where fμ(y0) == 0
    Example 1:
      Given DE fμ(y) = {y(1 - y)2 + μ}, determine the bifurcation values μ for the DE
    1. Solving for the derivative of the DE and its roots:
        2. f'μ(y) = (1 - y)2 - 2y(1 - y)
        f'μ(y) = (1 - y)(1 - 3y)

        0 = (1 - y)(1 - 3y)
        y = {1, 1/3}
    2. Solving for bifurcation values at fμ(y) = 0:
        3. fμ(1) = (1)(1 - 1)2 + μ
        μ = 0

        fμ(1/3) = (1/3)(1 - 1/3)2 + μ
        μ = -4/27

        μ = {0, -4/27}

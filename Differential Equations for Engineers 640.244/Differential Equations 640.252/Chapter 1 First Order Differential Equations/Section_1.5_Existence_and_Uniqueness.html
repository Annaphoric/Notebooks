Section 1.5: Existence & Uniqueness Theorems

    Existence
  • Even before solving for solutions to a DE, we first must prove that they exist
  • If a model for a physical system cannot locate any solutions, then the model is considered invalid

  • Given 5th degree polynomial {f(x) = 2x5 - 10x + 5}, to determine if there are any solutions (roots of x; where f(x) = 0), test values for x can be chosen:
      • f(-1) = 13
      f(1) = -3
  • Since this polynomial is continuous, it can be confirmed that there exists at least one value of zero for some value of x in the polynomial, where f(x0) = 0
      • x0 would have to be between -1 & 1 to achieve a value of zero for the polynomial, even if it may not be unique

  • The same reasoning can be applied to a first-order DE:
      • let dy/dt = f(t, y), a continuous function where t & y lie within a bounded interval, then there exists a t0 that lies within the interval, giving a value y(t0) = y0
      This theorem is true as long as the function is continuous and does not change drastically
    Uniqueness
  • A solution is considered unique if there is only one value of t for some y
  • More specifically:
      • let dy/dt = f(t, y), a continuous function where t & y lie within a bounded interval, and y1(t) & y2(t) are 2 functions that solve for the DE at a point (t0, y0); if y1(t) == y2(t) is true, then the solution is considered unique
      This theorem is true as long as the function & its partial derivative with respect to y are continuous

Section 1.4: Euler's Method

    Euler's Method
  • Euler's Method is a numerical method for approximating solutions to a differential equation in an initial value problem
  • A solution curve to the DE is always tangent to vectors within the slope field (graph of the DE within ty-plane)

  • let DE dy/dt = f(t, y), initial value point = (t0, y0)
  • Within slope field f(t, y), starting at (t0, y0), if tiny steps of step size Δt were to being taken along a tangent line segment towards increasing t, the overall slope of the solution curve can be determined
  • Say, there exists another point at (t0 + Δt), which is point (t1, y1)
  • This continues on for as many points as possible starting from point (tk, yk) depending on step size Δt
      • Generally, the shorter the step size, the more accurate the approximation
  • Since tk + 1 == (tk + Δt) is true, yk + 1 can be solved using the slope line formula
      • The slope at made from points (tk, yk) & (tk + 1, yk + 1) is equal to the value of the DE, which is f(t, y)
      So:
    • f(t, y) = (yk + 1 - yk) / (tk + 1 - tk)
    • f(t, y) = (yk + 1 - yk) / Δt
    • yk + 1 = f(t, y) Δt + yk
  • In other words, this method can be written as:
      • Point (tk + 1, yk + 1) = Point (tk + Δt, f(t, y) Δt + yk)

  • Since the slope field of the DE always lies below the solution curve, the approximation based on the tangent vectors will always lie below the solution curve as well

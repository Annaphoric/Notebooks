Section 3.3: Phase Portraits for Linear Systems

    Saddles
  • A linear system containing a positive & negative eigenvalue forms a saddle
      • The geometric representation of this form contains an equillibrium point (usually the origin) where one solution points towards it (negative eigenvalue) and another solution points away from it (positive eigenvalue)
      Equillibrium points of this form are considered unstable
      In a saddle, the solution containing the negative eigenvalue is known as a separatrix

    Sinks
  • A linear system containing 2 distinct negative eigenvalues forms a sink
      • The geometric representation of this form contains an equillibrium point (usually the origin) where both solutions points towards it
      Equillibrium points of this form are considered stable

    Sources
  • A linear system containing 2 distinct positive eigenvalues forms a source
      • The geometric representation of this form contains an equillibrium point (usually the origin) where both solutions points away from it
      Equillibrium points of this form are considered unstable, like saddles

Section 2.1: Real Roots for Second-Order Differential Equations

    Second-Order Differential Equations
  • A second-order DE is a DE where the highest derivative is the highest derivative. The general equation for a second-order DE has the form:
      • y'' = f(t, y, y')

  • Generally, there are no explicit methods to solving a second-order DE, unlike first-order DEs. Linear DEs are easier to solve, having the general form:
      • A(t) y'' + B(t) y' C(t) y = F(t)
  • In standard form:
      • y'' + p(t) y' + q(t) y = g(t)

  • A second-order DE is considered homogenous if g(t) == 0 is true. Also, it is considered constant-coefficient if p(t) && q(t) are constants

  • Constant-coefficient, homogenous equations are generally easiest to solve, whilst constant-coefficient, non-homogenous equations are not as easy. If a second-order DE is neither of these, it cannot be solved

  • As mentioned in the previous chapter, a first-order linear, constant-coefficient, homogenous DE has the form:
      • dy/dt + a y = 0
      y(t) = C / ea t (This is the general solution to this type of DE)
  • Since the solution contains an exponential (e), the exponential will persist with repeated differentiations

  • Solutions to second-order DEs can be found using a similar process. It would contain er t; the resulting equation for r is the referred to as the characteristic equation of the DE. {y(t) = er t; dy/dt = r er t; d2y/dt2 = r2 er t} & its derivatives will replace y & its derivatives within the DE (The r-exponontial method)

  • Second-order DEs apply extensively to the physics of motion, that is, the laws the govern the laws of motion. Newton's 2nd Law of Motion says that a force is caused by acceleration on an object:
      • force = mass * acceleration
      F = m a
  • The force depends on the position of an object; the acceleration is the 2nd-order derivative of the position

  • In relation to Newton's 2nd Law, Hooke's Law explains how the restoring force of a spring is dependent on a spring constant multiplied by the position:
      • restoring_force = - spring_constant * position
      F = -k x

  • Drag force is the natural force of a fluid onto an object
      • drag_force = - drag_coefficient * velocity
      F = -Î³ v

  • Another application of 2nd-order DEs is seen in Kirchhoff's Law. This law explains how the total voltage drop around a closed circuit must equal zero. The voltage may vary its variables depending on what lies within the circuit. For instance, in series, voltage is dependent on the current & resistance when passing through a resistor, charge over capacitance when passing through a capacitor, and inductance & rate of current when passing through an inductor:
      • Vresistor = current * resistance; Vcapacitor = charge / capacitance; Vinductor = inductance * current_rate
      Vresistor = I R; Vcapacitor = Q / C; Vinductor = L dI/dt
  • The current is the first-order of charge & current rate is the second-order of charge in terms of time

  • Solutions to second-order DEs can also be solved given initial conditions. let y'' = f(t, y); In order to solve this differential equation, y'(0) && y(0) must be explicitly given as ICs. There will always be 2 constants: C & D within a general solution of y(t) for y''. As long as the DE is second-order, linear, && homogenous, any 2 solutions: y1 & y2 can have any other solution that matches C y1 D y2 (linear combinations of solutions):
      • solution y(t) = C y1 + D y2
      y'(t) = C y1' + D y2'

  • let (C y1(0) + D y2(0)) = a && (C y1'(0) + D y2'(0)) = b; To solve for constants C & D, both a is multiplied with y1'(0) and b is multiplied with y1(0). Then, the difference of a y1'(0) & b y1'(0) are taken. The constant D can now be written as:
      • D = (a y1'(0) - b y1(0)) / (y1'(0) y2(0) - y1(0) y2'(0)), as long as denominator does not equal zero (This would be the general solution to the DE)

  • Abel's Theorem explains that if w(t) = {y1(t) y2'(t) - y1'(t) y2(t)} exists, the value of w(t) is always zero or never zero for all t. Testing any point will reveal the nature of this function. The function w(t) is known as the Wranskian of solutions. The solutions y1 && y2 are linearly independent only if the value of w(t) is never zero. They are linearly dependent if otherwise
  • Incorporating the r-exponential method (for real roots), another way to determine w(t) is by substituting for y & its derivative so that w(t) = {(r1 - r2) e(r1 + r2) t}, as long as r1 is not equal to r2

  • Existence & Uniqueness Theorems also apply for second-order DEs. As long as the coefficients contained within the DE are continuous on open interval I containing t0, then there exists exactly one solution y(t) for the DE throughout I. Determining the solution requires the 2 ICs as mentioned previously (y(t0) = y0 && y'(t0) = y0'). All solutions to a DE requires that there exists linear combinations of 2 y(t) solutions for all constants & all values of t

  • A fundamental set or basis is a set of linear combinations for all solutions to an equation. set S = {y1, y2} is a basis for DE {y'' + p(t) y' + q(t) y = 0} if {c1 y1 + c2 y2} applies for all solutions to the DE

  • Linear DEs are generally easier to solve. For DE {y'' + p(t) y' + q(t) y = 0}, let L(y) = y'' + p(t) y' + q(t) y; L(y) is referred to as a differential operator which takes the input of y(t) and substitutes its derivatives accordingly

  • The Theorum of Superposition says that if y1 & y2 are solutions of differential-operator L(y) = 0, then the linear combination {c1 y1 + c2 y2} is also a solution to L(y). The following equations are true:
      • L(c1 y1 + c2 y2) == (c1 y1 + c2 y2)'' + p(t) (c1 y1 + c2 y2)' + q(t) (c1 y1 + c2 y2)
      L(c1 y1 + c2 y2) == c1 (y1'' p(t) y1' + q(t) y1) + c2 (y2'' + p(t) y2' + q(t) y2)
      L(c1 y1 + c2 y2) == c1 L(y1) + c2 L(y2)
    Example 1:
      For what values of r is er t a solution to {y'' - 4y' - 5y = 0}?
    1. Replacing y & its derivatives for {y(t) = er t...}:
        2. y'' - 4y' - 5y = 0
        r2 er t - 4r er t - 5 er t = 0
        er t(r2 - 4r - 5) = 0
        (r - 5)(r + 1) = 0
        r = {-1, 5}
        This means the solutions to the given DE are y(t) = {e-t, e5t}
    Example 2:
      Find the solution to DE: y'' = t2; given ICs y(0) = 1 & y'(0) = 2.
    1. Solving for y'(t) & y(t) generally via integration:
        2. y''(t) = t2
        y'(t) = t3/3 + C
        y(t) = t4/12 + C t + D

    2. Solving for the constants individually using ICs:
        3. y(0) = (0)4/12 + C (0) + D == 1
        D = 1

      y(t) =
        t4/12 + C t + 1

        y'(0) = (0)3/3 + C == 2
        C = 2

        y(t) = t4/12 + 2t + 1
        y'(t) = t3/3 + 2
    Example 3:
      Find the general solution to DE {y'' + 2' - 3y = 0} and verify that it is, in fact, a general solution.
    1. Solutions for this DE can be determined using the r-exponential method:
        2. y'' + 2y' - 3y = 0
        er t (r2 + 2r - 3) = 0
        (r + 3)(r - 1) = 0
        r = {-3, 1}
        solutions = {1/e3t, et}
        This means the general solution is {y(t) = C / e3t + D et}

    2. To verify that this is truly the general solution, Abel's Theorem is applied:
        3. let y1(t) = 1/e3t; y2(t) = et

        w(t) = y1(t) y2'(t) - y1'(t) y2(t)
        w(t) = et / e3t + 3 et / e3t
        w(t) = 1 / e2t + 3 / e2t
        w(t) = 4 / e2t
        The value for w(t) is never zero regardless of t. Therefore, the general solution is legit

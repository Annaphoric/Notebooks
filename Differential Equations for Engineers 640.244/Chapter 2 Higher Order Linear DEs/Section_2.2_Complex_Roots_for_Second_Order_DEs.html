Section 2.2: Complex Roots for Second-Order Differential Equations

    Solutions with Complex Roots
  • In the case where solutions contain complex roots within the charaacteristic equation, we use solutions y(t) = {ea t ± b} having roots r = {a ± b i}, where b = {cos(b t) + i sin(b t)}. The general solution is written as y(t) = {c1 ea t (cos(b t) + i sin(b t)) + c2 ea t (cos(b t) - i sin(b t))}. At any time t, the value for y will always be a complex value. let DE y'' + p(t) y' + q(t) y = 0; where p(t) && q(t) are continuous functions. In order to alter a complex-valued solution y(t) so that it can be applied to the real world, the solution is split into 2 terms containing the real & imaginary portions: y(y) = u(t) + i v(t), where u(t) && v(t) are both real-valued functions. There functions are now considered solutions themselves to the DE
  • To test this out, homogenous L(u(t) + i v(t)) operation is performed:
      • L(u + i v) = (u + i v)'' + p(t) (u + i v)' + q(t) (u + i v) == 0
      (u'' + p(t) u' + q(t) u) + i (v'' + p(t) v' + q(t) v) = 0
      L(u) && L(v) == 0 must be true
    Example 1:
      Assume a characteristic equation has roots r = {a ± b i}. What is the general solution to the DE?
    1. let f(t) = ea t (cos(b t) ± i sin(b t)); The general solution can be written as:
        2. y(t) = c1 ea t cos(b t) + c2 ea t sin(b t)
        The legitimacy of the solution can be checked using the Wranskian w(t) = y1 y2' - y1' y2, where y1 = ea t cos(b t) & y2 = ea t sin(b t). w(t) is never zero as long as b is not zero
    Example 2:
      Solve the initial value problem: {y'' + 2y' - 15y = 0}, y(0) = 1, y'(0) = -2.
    1. Using the r-exponential method:
        2. y'' + 2y' - 15y = 0
        r2 + 2r - 15 = 0
        (r + 5)(r - 3) = 0
        r = {-5, 3}

        Therefore, the general solution is {y(t) = c1 / e5t + c2 e3t}. This solution is real-valued

    2. Solving for the constants using given ICs:
        3. y'(t) = 3c2 e3t - 5c1 / e5t

        y(0) = c1 / e5(0) + c2 e3(0) == 1
        c1 + c2 = 1

        y'(0) = 3c2 e3(0) - 5c1 / e5(0) == 2
        5c1 - 3c2 = -2

        3c1 + 5c1 + 3c2 - 3c2 = 3 - 2 // Via elimination addition
        8c1 = 1
        c1 = 1/8; c2 = 7/8

        y(t) = 1 / (8 e5t) + 7 e3t/8 // This is the general solution
    Example 3:
      Solve the initial value problem: {y'' + 2y' + 5y = 0}, y(0) = 2, y'(0) = 1
    1. Using the r-exponential method:
        2. y'' + 2y' + 5y = 0
        r2 + 2r + 5 = 0 // This cannot be factored
        r = (-2 ± sqrt(4 - 20)) / 2 // Using the quadratic formula; the equation has complex roots
        r = (-2 ± 4i) / 2
        r = -1 ± 2i
        let a = -1; b = 2;

        y(t) = c1 cos(2t) / et + c2 sin(2t) / et // This is the general solution based on the standard form to solutions for complex roots: y(t) = c1 ea t cos(b t) + c2 ea t sin(b t)

    2. Solving for the constants:
        3. y'(t) = -c1(2sin(2t) / et + cos(2t) / et) + c2(2cos(2t) / et - sin(2t) / et)

        y(0) = c1 cos(2(0)) / e(0) + c2 sin(2(0)) / e(0) == 2
        c1 = 2

        y'(0) = -c1(2sin(2(0)) / e(0) + cos(2(0)) / e(0)) + c2(2cos(2(0)) / e(0) - sin(2(0)) / e(0)) == 1
        2c2 - c1 = 1

        2c2 - (2) = 1
        c2 = 3/2

        y(t) = 2cos(2t) / et + 3sin(2t) / (2et) // This is the explicit solution
    Example 4:
      Find the general solution to DEs {y'' - 8y' + 16y = 0}, {y'' + 2y' - y = 0}, {y'' - 6y' + 13y}.
    1. Using the r-exponential method:
        2. y'' - 8y' + 16y = 0
        r2 - 8r + 16 = 0
        (r - 4)2 = 0 // Double root
        r = 4

        y(t) = e4t (c1 + t c2)
        t is added as constant-coefficient for double roots

    2. Using the r-exponential method:
        3. y'' + 2y' - y = 0
        r2 + 2r - 1 = 0 // Cannot be factored normally
        r = (-2 ± sqrt(4 + 4)) / 2
        r = -1 ± sqrt(2)

        y(t) = c1 e-1 + sqrt(2) + c2 e(-1 - sqrt(2))

    3. Using the r-exponential method:
        4. y'' - 6y' + 13y = 0
        r2 - 6r + 13 = 0 // Cannot be factored normally
        r = (6 ± sqrt(36 - 52)) / 2
        r = 3 ± 2i

        y(t) = e3t (c1 cos(2t) + c2 sin(2t))

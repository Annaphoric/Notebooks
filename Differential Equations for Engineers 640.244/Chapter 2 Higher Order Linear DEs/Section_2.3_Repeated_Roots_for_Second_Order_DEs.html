Section 2.3: Repeated Roots for Second-Order

    Repeated Roots
  • Repeated roots arise whenever there is a multiple root when factoring

  • For the general characteristic equation {y(t) = c1 y1 + c2 y2}, if y1 == y2 is true, the solution contains a double root (single-repeating root) and t is added as a constant coefficient to one of the coefficients:
      • y(t) = C y1 + D t y1
      y(t) = y1 (C + D t)

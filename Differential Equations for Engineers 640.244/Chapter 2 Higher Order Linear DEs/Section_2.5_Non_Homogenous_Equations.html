Section 2.5: Non-Homogenous Equations

    Variation of Parameters
  • To solve for the general solution of a non-homogenous, linear DE {y'' + p(t) y' - q(t) y = g(t)} using variation of parameters method, the homogenous portion of the DE is first isolated {y'' + p(t) y' - q(t) = 0}. let [y1, y2] = constant-coefficients for the general solution of the homogenous DE; The solution to the non-homogenous portion is as follows:
      • y(t) = u(t) y1(t) + v(t) y2(t), where du/dt = -g(t) y2(t) / y1 (y1 y2' - y1' y2) & dv/dt = g(t) y1(t) / (y1 y2' - y1' y2)
      The denominator is the Wranskian w(t)
    Undetermined Coefficients
  • To solve for the general solution of a non-homogenous, linear DE {y'' + p(t) y' + q(t) = g(t)} using undetermined coefficients, the homogenous portion of the DE is first isolated {y'' + p(t) y' + q(t) = 0}. The solution to the non-homogenous portion is then guessed by using the terms within g(t) as coefficients in a new equation {yu(t) = A f(t) + B h(t) + C, where f & h are terms within g, and [A, B, C] are the undetermined coefficients}. Note: The terms chosen within g(t) cannot be the same as the terms within the homogenous solution; if they are, multiply t to the term until it is unique. The first & second derivatives of this solution are found and substituted into the given DE. The undetermined coefficients are evaluated to make the equation true. These coefficients now form part of g(t) and are added to the final general solution
    Example 1:
      Find the general solution to DE {y'' + 3y' - 4y = 3 e2t} using variation of parameters method.
    1. Solving for the general solution of the homogenous portion:
        2. y'' + 3y' - 4y = 0
        r2 + 3r - 4 = 0 // r-exponential method
        (r + 4)(r - 1) = 0
        r = {-4, 1}

        yh(t) = C / e4t + D et // This is the general solution to the homogenous portion

    2. Solving for the solution to the non-homogenous portion:
        3. w(t) = y1 y2' - y1' y2 // This is the Wranskian
        w(t) = et / e4t + 4 et / e4t
        w(t) = 5 / e3t

        du/dt = -g(t) y2(t) / w(t)
        du/dt = -3 e2t et / (5 / e3t)
        du/dt = -3 e6t / 5
        u(t) = -e6t / 10 + C

        dv/dt = g(t) y1(t) / w(t)
        dv/dt = (3 e2t / e4t) / (5 / e3t)
        dv/dt = 3 et / 5
        v(t) = 3 et / 5 + D

        y(t) = u(t) y1(t) + v(t) y2(t)
        y(t) = (-e6t / 10 + C) / e4t + (3 et / 5 + D) et
        y(t) = C / e4t - e2t / 10 + 3 e2t / 5 + D et
        y(t) = C / e4t + D et + e2t / 2 // This is the general solution to the non-homogenous DE
    Example 2:
      Find the general solution, if possible, to DE {y'' - 2y' + 5y = e3t + t} using underdetermined coefficients method
    1. Finding the general solution to the homogenous portion of the DE:
        2. y'' - 2y' + 5y = 0
        r2 - 2r + 5 = 0
        r = (2 ± sqrt(4 - 20)) / 2
        r = 1 ± 2i
        yh(t) = D cos(2t) et + E sin(2t) et

    2. Guessing the undetermined coefficients in a solution equation, then substituting:
        3. yu(t) = A e3t + B t + C

        yu'(t) = 3A e3t + B

        yu''(t) = 9A e3t

        (9A e3t) - 2(3A e3t + B) + 5(A e3t + B t + C) = e3t + t
        9A e3t - 6A e3t - 2B + 5A e3t + 5B t + 5C = e3t + t
        8A e3t + 5B t - 2B + 5C = e3t + t
        let A = 1/8, B = 1/5, C = 2/25; // These values make the equation true

        y(t) = D cos(2t) et + E sin(2t) et + e3t / 8 + t/5 + 2/25 // This is the final form of the general solution to the given non-homogenous DE
    Example 3:
      Find the general solution to the DE {y'' + 2y' + y = 1 / et} using undetermined coefficients method.
    1. Solving for the homogenous solution:
        2. y'' + 2y' + y = 0
        r2 + 2r + 1 = 0
        (r + 1)2 = 0
        r = -1

        yh(t) = (C + D t) / et

    2. Solving for the non-homogenous solution:
        3. yu(t) = A / et // This is an invalid term because it matches a term within the homogenous solution
        yu(t) = A t2 / et // The term is multiplied by t twice to become unique

        yu'(t) = 2A t / et - A t2 / et
        yu'(t) = A t(2 - t) / et

        yu''(t) = 2A / et - 2A t / et - 2A t / et + A t2 / et
        yu''(t) = A t2 / et - 4A t / et + 2A / et
        yu''(t) = A(t2 - 4t + 2) / et

        y(t) = A((t2 - 4t + 2) + 2(2t - t2) + t2) / et = 1 / et
        y(t) = 2A / et = 1 / et
        let A = 1/2;

        y(t) = (C + D t) / et + t2 / (2et)
    Example 4:
      Find the general solution to DE {y'' + 4y' + 4y = 5 cos(2t)} using undetermined coefficients method.
    1. Solving for the homogenous solution:
        2. y'' + 4y' + 4y = 0
        r2 + 4r + 4 = 0
        r = -2

        yh(t) = (C + D t) / e2t

    2. Solving for the non-homogenous solution:
        3. yu(t) = A cos(2t) // Guess #1

        yu'(t) = -2A sin(2t)

        yu''(t) = -4A cos(2t)

        (-4A cos(2t)) + 4(-2A sin(2t)) + 4(A cos(2t)) = 5 cos(2t)
        -4A cos(2t) - 8A sin(2t) + 4A cos(2t) = 5 cos(2t)
        -8A sin(2t) = 5 cos(2t)
        There is no A to satisfy the equation, another undetermined term must be chosen alongside the equation

        yu(t) = A cos(2t) + B sin(2t) // Guess #2

        yu'(t) = 2(-A sin(2t) + B cos(2t))

        yu''(t) = -4(A cos(2t) + B sin(2t))

        -4(B sin(2t) + A cos(2t)) + 8(B cos(2t) - A sin(2t)) + 4(B sin(2t) + A cos(2t)) = 5 cos(2t)
        8 B cos(2t) - 8 A sin(2t) = 5 cos(2t)
        let A = 0, B = 5/8;

        y(t) = (C + D t) / e2t + 5 sin(2t) / 8 // This is the non-homogenous solution
    Example 5:
      Find the solution to DE {y'' - 2y' - 8y = 2 e4t} given ICs {y(0) = 1} & {y'(0) = 3}
    1. Solving for the homogenous solution:
        2. y'' - 2y' - 8y = 0
        r2 - 2r - 8 = 0
        r = {-2, 4}

        yh(t) = C / e2t + D e4t

    2. Solving for the non-homogenous solution:
        3. yu(t) = A e4t // Guess #1

        yu'(t) = 4A e4t

        yu''(t) = 16A e4t

        (16A e4t) - 2(4A e4t) - 8(16A e4t) = 2 e4t
        0 == 2 e4t // Unable to use A so the guess is invalid

        yu(t) = A t e4t // Guess #2

        yu'(t) = A e4t + 4A t e4t
        yu'(t) = A e4t(4t + 1)

        yu''(t) = 4A e4t + 4A(e4t + 4t e4t)
        yu''(t) = 8A e4t + 16A t e4t
        yu''(t) = 8A e4t(2t + 1)

        (8A e4t(2t + 1)) - 2(A e4t(4t + 1)) - 8(A t e4t) = 2 e4t
        6A e4t = 2 e4t
        A = 1/3

        y(t) = C / e2t + D e4t + t e4t / 3 // This is the general solution to the given DE
    3. Solving for the explicit solution to the DE using ICs:
        4. y(0) = C / e2(0) + D e4(0) + (0) e4(0) / 3 == 1
        C + D = 1
        y'(t) = -2C / e2t + 4D e4t + (e4t + 4t e4t) / 3

        y'(0) = -2C / e2(0) + 4D e4(0) + (e4(0) + 4(0) e4(0)) / 3
        4D - 2C + 1/3 = 3
        4D - 2(1 - D) = 8/9
        6D = 26/9
        D = 26/54
        D = 13/27

        C = 1 - (13/27)
        C = 14/27

        y(t) = 14 / (27 e2t) + (13 e4t) + t e4t / 3 // This is the explicit solution

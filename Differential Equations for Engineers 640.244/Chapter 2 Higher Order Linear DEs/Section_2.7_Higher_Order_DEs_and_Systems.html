Section 2.7: Higher-order DEs and Systems

    Systems of Equations
  • In the real world, there are multiple factors that are accounted for when solving an equation. The system can be updated by changing a model of a system. As far as we know, there is only one method (Euler's method) for solving for a DE numerically (regardless if it is homogenous or not; linear or not) which only works for first-order DEs. However, Euler's Method also works for first-order systems. A higher-order DE can be solved using Euler's Method by converting it into a first-order system
  • An n-th order equation will become an n-component system using a n-amount of arbitrary variables. This leads to a linear equation with matrices that would solve the equation. let second-order DE {y'' + a(t) y' + b(t) y = g(t)}; This second-order DE will have a matrix derivative of 2 components: [x1 x2], where x1 = y & x2 = y'. In this case:
      • [x1 x2] = [y y']
      [x1 x2]' = [y' y'']
      [x1 x2]' = [y' g(t) - (a(t) y' + b(t) y)]
      [x1 x2]' = [x2 g(t) - (a(t) x2 + b(t) x1)]
      [x1 x2]' = [0 -b(t)] x1 + [1 -a(t)] x2 + [0 g(t)]
      [x1 x2]' = [[0 -b(t)] [1 -a(t)]] [x1 x2] + [0 g(t)]
      x' = [[0 -b(t)] [1 -a(t)]] x' + [0 g(t))]
    Importance of Systems of Equations
  • When encountered with a multi-variable problem, linear/systems of equations are convenient in organising and reaching a solution more efficiently. Systems can be used to solve complex systems such as where multiple variables are changing
  • Systems can also be used to predict a final value over time and achieve the best-fit parameters based on the data
    Example 1:
      Convert DE {y'' + t2 y' - et y = sin(t)} and ICs {y(0) = 3; y'(0) = 1} into a first-order system.
    1. let x1 = y == 3, x2 = y' == 1; In matrix form, this equation is written as:
        2. [x1 x2]' = [y' (sin(t) - t2 y' + et y)]
        [x1 x2]' = [x2 (sin(t) - t2 x2 + et x1)]
        [x1 x2]' = [[0 et] [1 -t2]] [x1 x2] + [0 sin(t)]

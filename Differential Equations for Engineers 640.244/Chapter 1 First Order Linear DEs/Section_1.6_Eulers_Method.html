Section 1.6: Euler's Method

    Euler's Method
  • Euler's Method provides a way of numerically approximating the solution to a DE without graphical methods. Existence & uniqueness must be present

  • The tangent line (derivative) of a function provides the best linear approximation. Tangent lines at different points of a function at h-distance away from each other are iterated to approximate how the function will behave, starting with an initial point. Given y' = f(t, y) (the slope), initial point (t0, y0), step-size h, points are chosen as followed:
      • Point 0 = (t0, y0)
      Point 1 = (t1, y1) == (t0 + h, y0 + f(t0, y0) h)
      Point 2 = (t2, y2) == (t1 + h, y1 + f(t1, y1) h)
      Point k = (tk, yk) == (tk - 1 + h, yk - 1 + f(tk - 1, yk - 1) h)

  • Decreasing the step size (or increasing number of intervals) would increase the accuracy of Euler's Method Approximations

Section 1.1 - 1.3: Separable Equations

    Classifying Solutions
  • There are 2 classifications of solutions:
      • Explicit solutions
    • A solution whereby a dependent variable can be written explicity in terms of an independent variable such as y = f(x) = x2
    • These types of solutions can usually be solved by simple integration
      Implicit solutions
    • A solution whereby a dependent variable cannot be written explicitly in terms of an independent variable such as f(x, y) = (y2 + x2 + y + x). In this case, it is not possible to isolate the dependent variable y onto one side of the equation

  • A DE is said to be separable if the differential and its like-terms can be isolated to one side of the equation such as dy/dt = f(t, y) -> g(y) dy = h(t) dt
    Example 1:
      Find the general solution of the DE: dy/dx = -x/y if y(0) = 3.
    1. Attempting to solve for y in terms of x (separating the equation):
        2. dy/dx = -x/y
        y dy = -x dx (multiply both differentials on opposite sides)
        y2/2 = -x2/2 + C (integrated both sides, on the left with respect to y and on the right in terms of x)
        y2 + x2 = C (multiply by 2 on both sides, let C absorb the 2)
        This is an implicit equation since y cannot be written as an explicit function of x (it would have Â±)

    2. Solving for C given the initial condition y(0) = 3:
        3. y2 + x2 = C
        (3)2 + (0)2 = C (substituting)
        9 = C

    3. Solving for y explicitly:
        4. y2 + x2 = 9
        y = sqrt(9 - x2)
        This function of x is an implicit solution because y is written in term of x without any ambiguity
    Example 2:
      Find the general solution given DE: {dx/dt = (x2 - 1) t} & initial condition: x(0) = 0.
    1. Solving for x(t):
        2. dx/dt = (x2 - 1) t
        dx / (x2 - 1) = t dt (Separating the DE)
        dx / ((x + 1)(x - 1)) = t dt

        ((A / (x + 1)) + (B / (x - 1))) = 1 / ((x + 1)(x - 1)) (Converting to partial fractions)
        ((A(x - 1) + B(x + 1)) / ((x + 1)(x - 1))) = 1 / ((x + 1)(x - 1))
        A(x - 1) + B(x + 1) = 1
        For x = 0: B - A = 1 -> B = A + 1
        For x = 1: 2B = 1 -> B = 1/2
        Therefore, A = -1/2

        (1 / 2(x - 1) - 1 / 2(x + 1)) dx = t dt
        ln|x - 1|/2 - ln|x + 1|/2 = t2/2 + C (Integrating both sides; on the left using u-substitution)
        ln((1 - x) / (x + 1)) = t2 + C (Since the absolute value will always make the expression within the log positive, the numerator can be written)
        (1 - x) / (x + 1) = et2 + C
        1 - x = D et2 (x + 1) (let D absorb C where D = eC)
        1 - x = Dx et2 + D et2
        Dx et2 + x = 1 - D et2
        x (D et2 + 1) = 1 - D et2
        x(t) = (1 + D et2) / (1 - D et2)

    2. Solving for D using initial condition x(0) = 0:
        3. 0 = (1 + D) / (1 - D)
        D = -1

    3. Simplifying x(t):
        4. x(t) = (1 - et2) / (1 + et2)
    Example 3:
      Find the solution given DE: dy/dx = {(3x2 + cos(x)) / (y4 + y)} & initial condition: y(0) = -3.
    1. Attempting to solve for y in terms of x:
        2. dy/dx = (3x2 + cos(x)) / (y4 + y)
        (y4 + y) dy = (3x2 + cos(x)) dx (multiplying differentials on both sides)
        y5/5 + y2/2 = x3 + sin(x) + C (integrate both sides)
        This is the implicit form of the equation

    2. Solving for C given the initial condition y(0) = -3:
        3. (-3)5/5 + (-3)2/2 = (0)3 + sin(0) + C
        C = -243/5 + 9/2

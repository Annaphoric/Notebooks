Section 1.4: First Order DEs

    First Order DEs
  • A first order differential equation is an equation containing a max of a first derivative. The first derivative can easily be isolated

  • A differential is the tangent line made at a point of a function in terms of an independent variable such as dy/dt based on y(t). Depending on what the function y = f(t) is, different points on the curve will have tangent lines with different slopes. A graphical representation of the slopes of tangent lines along a curve is referred to as a slope field. The slopes are drawn out as vectors based on how the function behaves for a given value of independent variable (t). These vectors have a magnitude based on the differential (tangent line slope) of a given point and a direction based on nearby differentials for a given point
      • For instance, if a point y(0) = 3 (or (3, 0)) of function y(t) has slope vectors that point upwards towards y = 5, the solution is said to approach 5 given initial condition y(0) = 3

  • Also written as y(t) = f( t, y(t), y'(t)), yn - 1(t)) for t in I, where t = independent variable & y = dependent variable

  • First order DEs can be either linear or non-linear
      • Linear 1st order DEs have the standard form: {a(t) y' + b(t) y = c(t)} or {dy/dt + p(t) y = g(t)}. The equation g(t) is said to be..
    • a constant coefficient only if p(t) is a constant
    • homogenous only if it is equal to zero
    • non-homogenous if it is equal to a non-zero number

      Any linear first-order DE can be multiplied on both sides by an integrating factor μ such that the reverse product rule can be performed to explicitly solve for the dependent variable. The process using a linear 1° DE in standard form is as shown:
    • μ(t) (y' + p(t) y) = μ(t) g(t) (the integrating factor μ is multiplied on both sides)
    • μ dy/dt + μ p(t) y) = μ g(t) (factor out μ)

    • μ' = μ p(t) (based on the product rule for derivatives)
    • dμ/dt = μ p(t) (rewriting derivative as differential form)
    • dμ / μ = p(t) dt (separating the DE)
    • ln(μ) = P(t) + C, where P'(t) = p(t) (integrating both sidess)
    • μ(t) = eP(t) (this is the integrating factor; the constant is removed because the integrating factor is multiplied on both sides)

    • μ(t) (y' + p(t) y) = (μ(t) y)' (Rewriting left side of the equation)
    • d(μ(t) y)/dt = μ(t) g(t) (Rewriting as differential form)
    • d(μ(t) y) = μ(t) g(t) dt (Separating DE)
    • μ(t) y = μ(t) G(t) / p(t) + C, where G'(t) = g(t) (integrating both sides)
    • y(t) = G(t) / p(t) + C / μ(t) (This is the general solution; dividing both sides by μ(t))
    • y(t) = G(t) / p(t) + C / eP(t) (This is another way to write the general solution)
    Example 1:
      Solve for y(t) given the differential equation: t2 dy/dt + 2ty = et.
    1. Since this DE is first-order & linear, it can be separated:
        2. t2 y' + 2ty = et
        (t2 y)' = et (the reverse product rule of differentiation was used here. (t2 y)' can also be written as d(t2 y)/dt)
        t2 y = et + C (both sides are integrated in terms of t)
        y(t) = (et / t2) + C / t2 (Solving for y)
    Example 2:
      Solve for y(t) explicitly given the DE: y' + 4y = 1; & initial condition: y(0) = 1;.
    1. Solving for y(t) generally:
        2. dy/dt + 4y = 1
        e4t (dy/dt + 4y) = e4t (p(t) = 4; P(t) = ∫(p(t) dt) = 4t; μ(t) = e4t) (Multiply both sides by integrating factor)
        d(e4t y) = e4t dt (Rewriting left-side)
        e4t y = e4t/4 + C (Integrating both sides)
        y(t) = 1/4 + C / e4t (This is the general solution for y(t); divide both sides by integrating factor)

    2. Solving for y(t) explicitly using initial condition:
        3. y(0) = 1/4 + C / e4(0) == 1 (Substituting)
        1/4 + C = 1
        C = 3/4

        y(t) = 1/4 + 3 / (4e4t)
    Example 3:
      Explicitly solve for y(t) given DE: y' - y = 4t e3t; initial condition: y(0) = 2.
    1. Finding the general solution of y(t):
        2. dy/dt - y = 4t e3t
        e-t(dy/dt - y) = 4t e2t (p(t) = -1; μ(t) = e-3t)
        d(y / et) = 4t e2t dt (Separating equation)
        y / et = 2t e2t - 2∫(e2t dt) + C (Integration via integration by parts (IBP): u = 4t; dv = e2t dt; du = 4 dt; v = e2t/2)
        y / et = 2t e2t - e2t + C (Completing integration)
        y(t) = (2t - 1) e3t + C et (This is the general solution)

    2. Solving for y(t) explicitly using initial condition:
        3. y(0) = C - 1 = 2
        C = 3

        y(t) = (2t - 1) e3t - et (This is the explicit solution)
    Example 4:
      Solve for y(t) explicitly given DE: ty' - 4y = t2 + 1/t; initial condition: y(1) = 2.
    1. Solving for y(t) generally:
        2. t dy/dt - 4y = t2 + 1/t
        dy/dt - 4y/t = t + 1/t2 // Dividing both sides by t to put the DE in standard form, isolating the differential
        e-4 ln(t)(dy/dt - 4y/t) = e-4 ln(t)(t + 1/t2) // Multiplying by integrating factor; p(t) = -4y/t; μ(t) = e-4 ln(t);
        d(t-4 y) = t-4 (t + 1/t2) dt // e-4 ln(t) becomes t-4 according to power log rule
        d(y / t4) = (1/t3 + 1/t6) dt
        y / t4 = C - (1/2t2 + 1/5t5) // Integrating both sides
        y(t) = C t4 - (t2/2 + 1/5t) // This is the general solution

    2. Solving for y(t) explicitly using initial condition:
        3. y(1) = C - (1/2 + 1/5) == 2
        C - 7/10 = 2
        C = 27/10

        y(t) = 27t4/10 - (t2/2 + 1/5t) // This is the explicit solution
    Example 5:
      Solve for y(t) explicitly given DE: 2y' - ty = 1; IC: y(1) = 4;
    1. Solving for y(t) generally:
        2. dy/dt - ty/2 = 1/2 // Converting to differential, standard form by dividing by 2
        e-t2/4(dy/dt - ty/2) = e-t2/4/2 // Multiplying by integrating factor μ(t) = e-t2/4; p(t) = -t/2;
        d(y / et2/4) = dt / 2et2/4
        y / et2/4 = 1/2 ∫(dt / et2/4) + C // Integrating the left-side. The right-side cannot be evaluated, converting it into a definite integral would be the simplest form to write out the integral
        y / et2/4 = 1/2 ∫1t(ds / es2/4) + C // Arbitrary variable s was used to signify the change in form. 1 is used as the lower bound to match the IC
        y(t) = et2/4 (1/2 ∫1t(ds / es2/4) + C) // This is the general solution

    2. Solving for y(t) explicitly using the IC:
        3. y(1) = e1/4 C == 4
        C = 4/e1/4

        y(t) = et2/4 (1/2 ∫1t(ds / es2/4) + 4e1/4)
